# 第3章 许老头的粥

今天早上的粥是许老头煮的。又放多了水。

两个人各自喝完，谁都没先开口。灶台上那把菜刀横在砧板上。刀口朝墙。但今天刀口朝外。

许若存早看见了。

老头你今天这摆的是个什么阵仗。

窗外传来老王那只八哥模仿的咳嗽声。不是一声，是三声，连着的，然后才是真人嗓门：行了知道你起来了，别咳了。老王的。许若存笑了一下，但笑意没到眼睛里。他的视线一直落在那把刀上。

三声，说明今早平安。这是老王和许老头十几年前的约定，许若存小时候听过一次解释：八哥叫一声是人醒了，叫三声是外面没事。至于叫几声算有事，老王没说过。

巷口陈婶的嗓门准时响起。许老头，今天肉馅新鲜，给你留了俩。

陈婶天天给你留包子，你又不给钱，她图啥。许若存问。

图我不在她摊前晕倒。许老头把空碗放进水槽，我去看看老王的鸟。

许若存想说"你又不养鸟你看个锤子"，忍住了。老头出门的步子比平时急了一截。藏着事呢。

许若存伸手去拿碗，余光瞥见窗纸上有道人影一闪而过。不是老王。那影子又高又瘦，从窗根底下一掠向西，没有脚步声。他猛地转头，只看见窗纸上晃动的树影。

"老头。"

"看见了。"许老头站在门口，背对着他，声音从门口传进来，"别追。追了他就知道我们在怕他。"

许若存的手停在碗沿上。碗里的粥已经凉了。

吃完饭许老头从床底下拖出一个破木箱。在许若存面前打开。

里面是一堆旧物件。青铜镜、瓷瓶、旧书、断了柄的剑。

今天教你点新东西。他拿起一块怀表递过来。看货。

许若存接过来扫了一眼就放下。假的。表壳是真旧，名字是后来刻上去的，刻痕里没有氧化层。

这叫基础。许老头面无表情，递过来一只瓷瓶。

也是假的。釉色太新。

这个呢？一面旧铜镜，花纹磨得几乎看不清。

许若存翻过来看了看背面，又对着光眯了一眼。真的。但品相太差，不值钱。

错了。许老头的语气变了。不是在教，是在说一件他憋了很久的事。这镜子是前朝一位小姐的。她每天照镜子都嫌自己胖，后来嫁了个厨子，真胖了，又怪镜子没提醒她。她叹了半辈子气，这黑斑就是气叹出来的。

许若存愣了一下。编的？

不是编的。她嫁的那个厨子后来开了个小饭馆，我年轻时常去吃。那老板喝多了，拉着我说了大半宿。

许老头看着他。旧货市场最怕的不是收到假货。假货不会害人。真货背后都有故事，故事多了就惹事。看货不是看真假，是看它背后有没有麻烦。有麻烦的真货，不如没麻烦的假货。

许若存没笑。他听出了老头话里有话。

那玉佩背后有什么麻烦。

许老头又从箱子底下翻出一只小木鱼。半个巴掌大，身上有一道从顶部直裂到底的裂纹。底部刻着两个字，笔画磨得几乎看不清。

许若存接过来眯着眼看了半天。真言？

许老头的手明显顿了一下。你能认出这两个字？

第一个字像是"真"，第二个剩半边，猜的。他把木鱼翻了个个儿，语气随意。你总共也没教我几个字。

这个不卖。货是拿来卖的，这个是拿来记的。

他把木鱼放回箱底，动作比平时慢了整整一拍。许若存注意到了。那道裂纹和玉佩上的纹路走向一致。他的手不自觉地摸了一下怀里的玉佩。许老头看见了，没说话。

许老头又从侧袋里摸出一块碎瓷片。画着半朵没画完的花。

真的。但是刚碎的。

他把瓷片塞进灶台边的破陶罐。让它在灰里待着。灰能压住一些东西。

什么东西？

说出来就不灵了。他拍了拍手上的灰。记住，看货看到最后，看的是谁碰过它、为什么不要它了。这两点比真假重要一百倍。

许若存把玉佩从怀里拿出来。那这玉佩是谁碰过，为什么不要了。

许老头的手停在了箱子边缘。不认得。

那你为什么多给我一枚铜钱。

沉默。许老头的右手不自觉地伸进袖子里，摩挲那块碎布。

有的钱认主。许老头的声音低了一截，没有刚才教训人的气势了。有些东西不是你不想拿就能不拿的。它自己会找你。

那它找了我了。然后呢。

许老头没回答。他站起来，衔上空烟杆，背起竹筐走到门口。站了一会儿。若存。如果以后有人问你那玉佩哪来的，就说是旧货摊上随手买的。

为什么。

因为这种解释最安全。

安全个屁。许若存蹲在门槛上说。语气不重，但字字清楚。你今天刀口都朝外摆了。你怕有人今晚进这屋。你跟谁结了梁子你自己心里清楚。别拿"安全"两个字糊弄我。我八岁跟你混，今年十六了。糊弄不了。

许老头在门口站了很久。像被人钉在了门槛上。他没回头，最后只说了一句。

你长大了。

就走了。

许若存手指在门槛上抠了一下。

许若存没有留在屋里。他去了旧货市场，蹲在张秀才书摊前翻一本缺了下册的旧书。余光能看到自家摊位全貌。许老头正把货一件一件往外摆，把昨天那只木盒放在最不起眼的角落，用破蓝印花布盖住一半。

然后他看到那个人。白天看清楚了。是个穿蓝衣服的中年男人，站在茶馆二楼，靠着窗喝茶。在这个鸟不拉屎的小城已经好几天了。每天早上来喝一碗咸豆浆，然后上二楼。不跟任何人说话，只是看。像一条蹲在墙角的狗。不动。但谁都能感觉到它在。

许若存蹲到自家摊位旁。压低声音。那个人又来了。

许老头正在修一只旧怀表，头也不抬，把发条往回退了一点，重新拧。别看。越看他越来劲。他不是承序府的人，是雇来的寻踪者。前天看的瞎老太太，昨天看的张秀才，今天看的我们。他在排顺序。等他排完就会下来。

你怎么知道他是雇来的？

那靴子我认识。承序府的人穿不起那种底。牛皮夹层底，走长路的，不是官靴。承序府那帮坐衙门的，哪穿得起这个。

谁雇的？

许老头没答话。他把怀表翻了个面，用绒布擦了擦表盖内侧，好一会儿才开口。万流坊。你记住这个名字就行，别多问。

许若存没想到许老头连这都看得出来。刚想再问，后脑勺忽然一紧。一股视线从茶馆二楼落下来。很轻。但很准。

他蹲在原地没动。低头解一截打了十几个死结的麻绳。手指稳稳的。

蓝衣人的目光在许老头手上停了片刻。许老头正在数零钱。每一枚都在指节上转一圈才放下，慢得出奇。

蓝衣人忽然放下茶杯，从茶馆侧门走了。方向是旧货市场后面的一片废屋。

许若存压低声音。他去废屋了。

别管。许老头把怀表收进木格。声音平淡。你现在最该做的，是站起来去张秀才那里蹭书看，然后回家煮粥。

你就不好奇他去废屋干什么？那破地方住了好几天的乞丐。

好奇能当饭吃？

许若存站起来，看见老王拎着扫帚从茶馆底下那条巷子穿过去，像是去倒垃圾。蓝衣人走进废屋的阴影里之后，老王在巷口停了一下，把扫帚靠在墙根，蹲下来系鞋带。系了很久。废屋门口那道影子被他的背影挡住了一半。

许若存站起来拍了拍膝盖上的灰。行。我去当我的饭桶了。

他走到张秀才书摊前蹲下，翻开书。眼睛没在书上。他在想：许老头今天所有的话，都是在把他往外推。教他看货、告诉他木鱼的事、提玉佩的事。全是交代后事的节奏。这老东西在准备什么。

许若存把书翻了一页。喉结动了一下。

心里骂了一句：你个老不死的别想把我甩了。

傍晚许老头收摊回来。步伐比平时慢。他把竹筐放在灶台边，先坐在门槛上喘了一会儿。

许若存在灶台前煮粥。今天水放得正好，米粒开花但不烂。他想起那年生病时许老头把米油刮进他碗里的触感。老头自己从来不喝那层油。每次都留给他。

许老头喝了一口。点头的幅度比平时大了半分。有进步。

那是。学了八年了。再没进步我对不起那口锅。

老头嘴角动了一下。幅度很小。但许若存看见了。

两个人对着灶台喝粥。屋里只有筷子碰碗的声音。偶尔掺一声许老头喝粥的嘶哈。

吃完饭许老头坐在门槛上。衔着空烟杆，盯着院子里的月光。许若存在灶台前洗完碗，走出来蹲在旁边。许老头今晚抽烟杆连烟都没放，就干衔着。

许老头。

嗯。

你后悔过吗。

后悔什么。

留下我。

许老头伸手在门槛的木头上摸了一下。那是他每天坐的位置。木头被磨得光滑发亮，跟包了浆似的。后悔过。后悔没早点告诉你一些事情。总想着等哪天你长大了再说。结果等着等着，有些事就变得不好说了。

现在说也不晚。

现在说。你会怕。

我连蓝衣人蹲在茶馆数你数钱都见过了。还有什么好怕的。

许老头转过头看了他一眼。那个眼神里有点意外。还有点儿别的。然后他转回去了，吸了一口空烟杆。没装烟叶，干吸。

你和你爹。他收了声。

许若存等了片刻。等来的只有沉默。

说一半藏一半也是你们许家的传统技能吗。

你不姓许。你姓许是因为我姓许。

那你姓许是因为什么。

你爹把我名字借给我用的。这条命是他给的。名字也是他给的。

两个人沉默了一会儿。院子里只有风吹石榴树叶子的沙沙声。

许若存说。石榴树今年没结果。

可能明年结。

去年你也是这么说的。

树的事。急不来。

许若存站起来走进院子。石榴树落了大半叶子，剩下的在夜风里轻轻晃着。这棵树跟许老头挺像——什么都扛着，什么都不说。

许老头跟出来。抬头看天。

那颗星叫什么。

衡门。

我问的是星名。

星名也是人名。给你取名字的那个人。他就叫这个。

许若存没接话。他仰头盯着那颗最暗的星看了很久。久到脖子发酸。他想起很多事。但每件事都不完整，像缺了页的书。

他还说了什么。

说了很多。许老头衔着空烟杆，咬得比平时用力，声音有一点点哑。走之前说了四个字：绵绵若存。你的名字就是从这四个字里来的。别的也没说完。雨太大。后半句被雨吞了。

许若存的手指在膝盖上攥紧了。绵绵若存。他用这个名字活了十六年。今天才知道这四个字不是随便翻书翻出来的。

你当时怎么不留他多待一会儿。

许老头没回答。他低头磕了两下烟杆。把一个人的名字在舌尖上搁了一下。然后转身往屋里走。走了两步。顿了一下。没回头。

天冷。进去吧。

许若存站在石榴树旁没动。

他把玉佩从怀里摸出来。天已全黑。但背面那道影子在某个角度下浮现了——门前站着一个驼背的人，右手前伸，掌心贴向门面。那个人影的驼背和许老头有点像。又不全是。

他把玉佩收回去。掏出那枚有符文的铜钱。它在掌心里微微发热。方向指向城东。

他想起疯乞丐的话。没拿回来的东西会在某一滩水里浮起来。城东。去看看。水面上有你认识的影子。

城东只有一个湖。心湖。

许老头说那里蚊子多。那是他小时候用来骗他的借口。

明天。不管许老头让不让，他都要去。不是因为倔。今晚他知道了给自己取名字的那个人叫什么。这个答案落进心里，就再也吐不出来了。

许老头不让他去。他就偷着去。

十六了。腿长在自己身上。

他转身往屋里走，刚踏上门槛，身后老槐树的枝叶忽然哗啦啦响了一阵。没有风。那声响从树梢一直传到墙头，又沿着瓦脊滚过去，像有什么东西刚从树上跳到了屋顶上，正贴着瓦片往院子这头走。

许若存没回头。他推门进屋，反手把门闩插紧。

门闩落下去的声音，在夜里很沉。
