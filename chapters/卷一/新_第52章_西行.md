# 第52章 西行

沈见微没有直接往西。

他出了涤心院的后巷，贴着墙根往南走。洛京的南郊在卯时的天光里泛着一层灰青色，像一块还没醒透的旧布。

城西有人在等。那是叶知秋说过的话。

但他要先去一个地方。

那个孩子在南郊。涤心院用来威胁纪无咎的孩子。

沈见微记得那处院子的位置。上个月，他被贺姓女修带去"交代"过一次。院门朝西，门前有一棵歪脖子枣树。

他说了谎，只说自己没见过纪无咎半夜出过门。

那孩子大约五六岁，穿一件洗得发白的蓝布衫，蹲在门槛上玩石子。孩子的眼睛很亮，看人的时候不带怕。

涤心院选人质，一向很会选。

沈见微赶到时，天已经大亮。

歪脖子枣树还在，叶子落尽了，枝桠像几只枯手伸向天空。院门关着，门上的漆剥落得更多，露出底下灰白的木头。

他敲了三下门。

门里没有动静。他又敲了三下，指节叩在木头上，声音闷而轻。

门开了一条缝。一张布满皱纹的脸露出来，是个老妇人。她眯着眼睛打量沈见微，目光在他脸上停了许久。

"你找谁。"

"住在这儿的孩子。"沈见微说。

老妇人没有立刻答话。她把门缝又拉开一些，露出身后空荡荡的院子。灶台上的锅是冷的，屋檐下挂着几件旧衣裳，没有孩子的。

"三天前有人来接他，"老妇人说，"说是他舅舅。"

沈见微的手指在袖中收紧。

"来人穿什么衣裳。"

"深灰的袍子，领口绣着一道黑边。"老妇人顿了顿，"腰间挂着块牌子，我没看清字。"

涤心院的人不会称自己是舅舅。他们只会说"院里的先生"。

沈见微没有再问那个人的长相。他问另一件事："有没有留下东西。"

老妇人看了他一眼。那眼神里有审视，也有一种久病成医的疲惫。

"你不是第一个来问的。"她说。

"还有谁。"

"前日晚上，有个穿黑衣的男人。"老妇人低声说，"他给了我一点碎银，让我把这院子看牢。他说，总会有人来问孩子的事。"

沈见微的心跳慢了一拍。纪无咎在被带走之前，已经安排过这一步。

"东西呢。"

老妇人转身进屋。过了片刻，她捧着一块木牌走出来。木牌不大，边缘焦黑，像是从什么更大的东西上断裂下来的。

她用袖子擦了擦，递到沈见微面前。

"那人让我藏在灶底下。说若是来一个问'有没有留下东西'的人，就把这个给他。"

沈见微接过木牌。

牌子是檀木质地，被烟火熏过，表面有一道很深的裂纹。正面刻着半个纹路，是承序府镇魔司的令纹。背面刻着一个字。

纪。

沈见微把木牌握在掌心。木头的裂纹硌着手指，像一道尚未愈合的伤口。

纪无咎折断了自己的令牌。他用其中一块碎片，把这个孩子换了出来。

"他什么时候给你的。"沈见微问。

"五天前。"老妇人说，"他说，三日内会有人来接孩子。让我不要声张。"

五天前。那时纪无咎还在承序府，尚未被钟离燕带走。他已经算到了这一步。

沈见微把木牌收进怀里。贴着心口的位置，木头的凉意很快被体温覆盖。

"那黑衣人还说什么。"

老妇人想了想，摇头："没了。他只是看了孩子很久，临走时在孩子头上摸了一下。"

沈见微没有再问。

他摸出身上所有的铜钱，大约七八枚，塞进老妇人手里。老妇人怔了一下，想要推辞，他已经转身走出院门。

身后传来门关上的轻响。沈见微没有回头。

他沿着来时的路往北走，走到城门前才停下来。洛京的西门在辰时开，眼下还早，城门口只有三两个挑着担子的菜农在等。

沈见微在城墙根下站了片刻。

怀里那块木牌一直硌着他。他把它取出来，又看了一遍。令牌的裂纹从"纪"字中间穿过，像是把那个字劈成了两半。

纪无咎把一半自己留给了承序府，把另一半留给了这个孩子。

或者说，留给了他。

城门开了。

沈见微随着出城的人流走出去。他没有走官道正中，而是贴着路边。背上的布包不重，但里面的册子和画让他每一步都走得很小心。

册子是叶知秋从废档楼取出来的。画上是两个没有脸的女人，左肩都画着一圈螺纹。

他母亲的名字被洗掉了。许若存母亲的名字还在。

现在他手里又多了一样东西。纪无咎的令牌碎片。

这些东西能拼成什么，他还不知道。但他知道，把它们带在身上，比留在洛京任何一处都安全。

出城不到五里，官道分成两条。

一条往西北，是通往青阳镇的大路。路面平整，车马往来，尘土飞扬。另一条往西南，杂草丛生，路碑倒了一半，是废弃多年的驿道。

叶知秋说过，铜钱指向西南。

沈见微在岔口停下来。他从袖中取出那枚"西"字铜钱，摊在掌心。铜钱冰凉的表面刻着一个小小的"西"字，笔画很浅，但每一笔都像是用指甲刻进去的。

铜钱没有动。它只是躺在那里，像一个沉默的指引。

岔口边有一间茶棚。棚子很旧，四根木柱有三根已经歪斜，顶上铺着一层发黑的稻草。棚下摆了三张桌子，两条长凳，只有一个客人。

那人穿一身深蓝短打，袖口挽到小臂，露出一段白皙的手腕。他面前放着一碗没动的热茶，茶水上漂着几根枯茶叶。

谢渊。

沈见微没有立刻走过去。他先在路边站了片刻，观察茶棚周围。棚后有一棵枯死的老槐树，树干上刻满了过往的行人名姓。

棚前的土路上有几道新鲜的车辙，但往西南方向去的只有脚印，没有车痕。

那人没有抬头，却开口了："茶是冷的，碗是热的。你要坐，就坐。"

沈见微走过去，在长凳另一端坐下。

谢渊抬起头。他的眉眼和青石集那次相比没有变化，依旧带着一种不太属于这个地方的干净。只是眼下多了一点青黑，像是很久没有睡好。

"你也往西。"沈见微说。

"不往西。"谢渊说，"我在找那道缝。"

沈见微的手指顿了一下。

"我也在找那道缝。"他说。

谢渊看着他，眼神里没有意外。他像是早就知道沈见微会来，也早就知道他会说什么。

"你从哪里出来。"谢渊问。

"涤心院。"

"我知道。"谢渊端起茶碗，又放下，"我问的是，你心里那个院子。"

沈见微没有回答。

他从怀里取出那块木牌，放在桌上。木牌裂开的边缘朝着谢渊，背面的"纪"字朝上。

谢渊低头看了一眼。他的目光在木牌上停了很短的一瞬，随即移开。

"纪无咎的人情，"他说，"不周山庄记下了。"

沈见微仍然看着他："你为什么要找那道缝。"

"师尊让我来洛京看一样东西。"谢渊说，"他说，门边上那道缝，是十六年前有人故意留下的。那人想给后来的人留一条不必推门的路。"

"所以你站这边。"

"我不站任何一边。"谢渊说，"我只看缺口。完整的秩序是假象，缺口才是入口。"

沈见微沉默片刻，把木牌收回怀里。

"许若存已经往西走了。"谢渊说，"他手里有四枚铜钱。你手里有一枚'西'字铜钱，和纪无咎的半块令牌。"

"我们三个加起来，凑得出一把钥匙。"

"还差多少。"

"还差一个愿意推门的人。"谢渊站起身，把一枚铜钱放在桌上。铜钱正面朝上，刻着一个"西"字，和沈见微手中那枚一模一样。

"但我师尊说，"谢渊看着沈见微，"这次不要推门。推门的人，十六年前就已经选好了。"

沈见微低头看那枚铜钱。铜钱的边缘有一圈细密的纹路，像是被什么人反复摩挲过很多年。

"你师尊还说什么。"

"他说，"谢渊转身往西南驿道走去，声音从风里传回来，"跟着缺口走。别跟着门走。"

沈见微把桌上的铜钱拾起来。两枚"西"字铜钱叠在一起，发出一声极轻的碰撞。

他站起身，朝谢渊的背影追去。

西南驿道在晨雾里延伸，像一条被遗弃了很多年的带子。路边的杂草已经枯了，但仍能遮住半个脚踝。远处有乌鸦从一棵枯树上飞起，扑棱棱的声响在空旷里传出很远。

沈见微跟在谢渊身后三步远的地方。

他摸了摸怀里的木牌。"纪"字的裂纹还在硌着他的掌心。

纪无咎现在在哪里，他不知道。钟离燕会不会真的放他走，他也不知道。

但他知道，那个孩子已经安全了。

这足够了。

他想起孩子的眼睛。那双眼睛里没有涤心院教出来的顺从，也没有被当作人质后的怨恨。那孩子只是蹲在门槛上，把石子排成一排，等一个永远不会回来的人。

沈见微加快脚步，与谢渊并肩。

"你不问我为什么逃出涤心院。"

"不问。"谢渊说，"你逃出来了，这就是答案。"

"你也不问我手里那幅画是什么。"

"不问。"谢渊停下脚步，转头看他，"我只问你一件事。"

"说。"

"找到那道缝以后，"谢渊说，"你是想进去，还是想把它打开。"

沈见微看着他。晨风吹过谢渊的额发，露出他光洁的额头。他的眼睛很静，像两口不会起波澜的井。

"我只想弄清楚，"沈见微说，"我母亲当年为什么选择'入'。"

谢渊点了点头。

"那就够了。"他说，"你想清楚的人，比想进去的人更难对付。"

两人继续往前走。驿道在前方拐了一个弯，拐过那道弯以后，洛京的城墙就再也看不见了。

沈见微没有回头。

他知道，身后那座城市里，贺姓女修也许已经醒了，裴先生也许正在整理净心堂的架子，钟离燕也许正把纪无咎的断令牌收进抽屉。

那些都是别人的棋局。

他的棋局在这条往西的路上。

怀里，册子、画、木牌、铜钱，一样挨着一样。它们彼此不说话，却在同一个温度里慢慢靠近。

沈见微握了握拳。

西面有风来，带着枯草的涩味和远处土地庙的香火气。

他知道许若存就在前面某个地方。也许正在一口废井前犹豫，也许正在听灵犀讲那些关于缝隙的道理。

袖中的铜钱微微发烫。不是震动，是一种很轻的温度，像有人在远处点了一盏灯。

他会赶上他们的。

不是为了推门。

是为了把母亲的名字，从"被洗心者"那栏里，重新刻回人间。

母亲当年选择了"入"。他选择了"出"。

同一个门，两个方向。他不知道哪一个更接近真相。
