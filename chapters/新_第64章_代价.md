# 第64章 代价

灵犀的手从光里抽了回去。

那只很小的手缩回裂缝时，没有发出声音。缝里的光暗了一瞬，像有人在门后吹熄了一盏灯。许若存抱着灵犀跌坐在地上，她的身子轻得像一捆干草。

纪无咎没有过去看。

他知道灵犀还活着。她的胸口还在起伏，只是那起伏很浅，像是被刚才那一下抽走了大半魂魄。许若存低着头，手指按在她腕上，指节发白。

村子中央的圆还在转。

暗青色的纹路从纪无咎断指处爬出去，在地上绕成一个完整的圈。四枚铜钱立在圆心，方孔朝着裂缝。圈外的人进不来，圈内的人也出不去。

但圆不能一直转。

纪无咎能感觉到脚下的纹路在变轻。他的血、他的封印、他十六年前被斫下的那根小指，都在这一圈里被慢慢抽干。

贺姓女修动了。

她从灰袍队伍里走出来，脚步很轻，像踩在别人的呼吸上。她手里还捏着那只白瓷瓶，瓶塞已经拔开了一半，苦味的药气飘出来，混着缝里的腥甜。

她走到圆边，没有跨过那条暗青色的线。

“让开。”她说。

纪无咎站在圆的另一侧，背对着许若存和灵犀。他没有应声，只是把左手抬到身前，断指处的纹路亮了一下。

那圈线认得他。它也认得贺姓女修。

他们都是被门吐出来的人。

被吐出来的人有一个共同点：身体比记忆更诚实。贺姓女修腕上的红痕，纪无咎断指处的封印，都是门留下的牙印。他们互相认得，比认人更快。

“你要把净尽倒进去。”纪无咎说。

“是。”

“门后有多少人。”

“不知道。”贺姓女修说，“但每一个被洗过心的人，都会被它叫醒。”

“包括我。”

“包括你。”

纪无咎看着她。她的脸比七日前更瘦，眼眶下面的青黑像被人用指节碾过。但她的眼睛很亮，亮得近乎发狠。

那种亮他见过。在涤心院的洗心堂里，在那些被洗掉名字的人脸上。他们脸上刻着的，是痛苦催着人想起。

“你也是被洗过的人。”纪无咎说。

贺姓女修的眼皮抖了一下。

“你知道那是什么滋味。”

她没有回答。握着瓷瓶的手指收紧了，指节发出轻微的响。

“所以我才不能让她们出来。”她说。

声音比刚才低，像是从喉咙深处挤出来的。

“出来了，只会更痛苦。”

“那不是你能替她们选的。”

“她们自己也不会选。”贺姓女修说，“你以为她们想醒来？你以为她们愿意记得自己被丢掉、被洗掉、被关在门后面十几年？”

她把瓷瓶举高了些。瓶口对准裂缝，瓶塞已经完全松脱，苦味更重了。

“我来替她们不醒。”

纪无咎往前踏了一步。

圆里的暗青色纹路被他踩得往下一沉，四枚铜钱同时发出一声很细的响，像琴弦被人拨了一下。

“你站住。”贺姓女修说。

纪无咎没有站住。

他又踏了一步。这一步踩在圆的边界上，暗青色的光从他靴底漫上来，缠住他的脚踝。

那光像一圈活过来的绳子，先缠踝，再往上攀了一寸。纪无咎没有低头。脚下的圆在认他，也在警告他。

“你也是被洗过的人。”他说。

“你自己知道醒来是什么滋味。”

贺姓女修的手抖了一下。瓶口的光映在她脸上，把那层灰白照得透明。

“那滋味不好受。”纪无咎说。

“但那是我的滋味。不是她们的。”

贺姓女修看着他。她的嘴唇抿成一条线，嘴角往下坠。

“你不明白。”她说。

“我明白。”

“你不明白！”

她的声音拔高，像一根绷到极限的弦。

“你以为醒来是解脱？你以为想起自己是谁，就能重新活一遍？”

她往前跨了半步，踩在圆的边界上。暗青色的纹路立刻缠上她的靴面，但她没有退。

“想起来的每一天，我都想死。”她说。

“可我死不了。涤心院把我洗了三十年，我的身体已经不记得怎么死了。”

“所以你要让所有人都和你一样。”

“不是一样。”贺姓女修摇头，“是让她们不要像我一样。”

她举起瓷瓶。

纪无咎看见了她的手腕。那只手腕从灰袍袖口里露出来，腕上有一圈很淡的红痕。那红痕和断指处的封印不同源，但一样旧，一样深。

她也是从门里走出来的。比他晚，或者比他早。时间在那边没有意义。

门里出来的人，时间观念是乱的。有人早一天，有人晚十年，落地时却可以是同一个年纪。贺姓女修也许比他进去得更早，只是醒得比他晚。

“你倒下去，她们会醒。”纪无咎说。

“会醒。”

“醒来，门就会乱。”

“会乱。”

“但你也醒了。”

贺姓女修愣了一下。

“你不想让院长再把你洗回去。”纪无咎说。

“你不想让任何人再替你决定，记起还是忘记。”

她的手指僵在半空。瓷瓶还举在那里，但没有倾。

“所以你才要倒。”纪无咎说，“不是替她们选。你是替自己选。”

贺姓女修的脸色变了。

那层发狠的亮裂开了一条缝，下面露出更深的颜色。是恐惧，也是羞耻。她不愿意承认，但纪无咎替她说了出来。

“闭嘴。”她说。

“你恨的不是她们醒来。”纪无咎说，“你恨的是只有你自己醒着。”

贺姓女修的眼眶红了。

但她没有哭。涤心院的人不被允许哭，这个规矩刻在她骨头里，比净尽还难洗掉。

“你以为你懂我。”她说。

“我不懂你。”纪无咎说，“我懂门。”

他抬起左手，断指处的暗青色纹路和圆里的光连在一起。

“门把我们都吐出来的时候，没问过我们愿不愿意。现在我们站在门外，也不该替门里的人选。”

贺姓女修看着他，看了很久。

村子其他方向传来脚步声。承序府的黑甲在靠近，万流坊的人也动了。他们都在等圆里的两个人先分出结果。

谁先倒，缝就是谁的。

“你说得对。”贺姓女修说。

纪无咎没有应声。

“我是不该替她们选。”她说。

她的手垂了下来。

纪无咎刚要松一口气，贺姓女修的手腕却猛地一翻。瓷瓶重新举高，瓶口对准裂缝，这一次没有犹豫。

“但我更不能替她们醒。”

纪无咎扑了上去。

他的左手先伸出去，断指处的暗青色纹路像活物一样缠向瓷瓶。贺姓女修侧身避开，右手从袖中滑出一柄短刺，刺向他的肋下。

那柄刺很短，锋口却极薄。纪无咎认得出那是涤心院的洗心刺，专门用来挑断记忆的根。被它刺中的人不会死，只会慢慢忘记自己为什么要挡在门前。

刺尖还带着一股凉意。不是杀意，是涤心院特有的药气。离肋下还有三寸时，纪无咎已经闻到了那股味道。

他没有躲。

他想躲，但身后就是许若存和灵犀。他退了，那瓶净尽就会倒进缝里。

短刺没入肋间，发出一声很轻的闷响。

血没有立刻涌出来。洗心刺入肉时，伤口会暂时封住，像被药糊了一层。但纪无咎知道，里面的筋已经被挑断了一根。

纪无咎感觉自己的肺被什么东西顶了一下。皮肉没有痛，只剩一股被挤空的凉。那凉意顺着肋骨往上爬，爬到他十六年前的断指处，停住了。

旧伤被引燃。

他左手的小指断口处猛地一跳，暗青色的纹路骤然发亮，像是有火在骨头缝里烧。痛的不在皮肉，在封印里。它记得这道刺的气味，记得被洗心时挑断的每一根线。

纪无咎咬紧牙。

他没有退。右手握住贺姓女修的手腕，往下压。

贺姓女修的眼神变了。她没想到他不躲。

“你疯了。”她说。

“我欠的债。”纪无咎说。

“还没还完。”

他用力一拧。贺姓女修的手腕发出一声脆响，骨头没断，只是腕骨错了位。她的手指一松，瓷瓶往下滑。

贺姓女修抬膝顶向他腹间。纪无咎没有避，硬生生受了一下。那股力道撞在伤口边缘，眼前黑了一瞬。

他右手没松，借着那一顶的力量，把她的手往更高处推。

贺姓女修用左手去接。纪无咎比她快半步，右手掌根撞在瓶底。

瓷瓶飞了出去。

它在空中转了一圈，瓶口的塞子完全脱落。透明的药液从瓶口洒出来，被缝里的光照成淡黄色，像一蓬被风吹散的雨。

瓶子落在地上，碎成三片。

净尽渗入泥土。地面上的暗青色纹路被药液一浸，发出嘶的一声，冒出一缕很细的白烟。

圆外的脚步声停了一瞬。承序府的黑甲举起了小旗，却没有挥下。万流坊的人把指骨铜钱捏在指节间，发出很轻的响。

他们都在等，看这一局还有没有翻盘。

贺姓女修跪了下去。

她的双膝砸在地上，发出沉闷的响。她没有看纪无咎，眼睛盯着那滩正在渗入土里的液体。

药液越来越少。泥土把它吸进去，颜色变深了一小块，像一滴墨落在干纸上。

“你赢了。”她说。

纪无咎低头看着她。他的右手还悬在半空，掌心沾了一点药液，皮肤开始发烫。那热不像火，像记忆正在苏醒。

他压了下去。

贺姓女修没有抬头。

“但院长还会派别人来。”她说。

“我知道。”

“下一次，不会是我。”

“我知道。”

贺姓女修笑了一下。那笑里没有喜悦，只有一种被掏空后的轻。

“你守不住的。”她说。

“门后面的人想出来。外面的人想进去。你站在中间，只会被两边一起踩碎。”

纪无咎没有回答。

他转过身，朝许若存走去。

东面灰袍里有人往前挪了半步，被身旁的人拉住。西面桥头的黑旗停在那里，像一根被钉住的针。北面苇荡里没有铜钱声，万流坊的人在重新作价。

他们都没有进圆。圆还在转，只是比之前慢了很多。

每一步都踩得很重。肋下的伤口在流血，旧伤在左手断指处一跳一跳地疼。他能感觉到净尽的药性正在皮肤底下爬，想把他十六年前被洗掉的东西翻出来。

他没有让它翻。

现在不是想起的时候。

许若存还抱着灵犀，抬头看着他。那孩子的眼神很静，但静下面压着很多东西。

纪无咎在他身前停下。

他想蹲下去，但腿一软，单膝跪在了地上。这一跪无关示弱，是身体终于不支。

“纪大人。”许若存说。

纪无咎摆手，示意他不要说话。

他需要把力气省下来。把这句话说完。

“别让你娘出来。”他说。

声音很轻，但每个字都咬得很清楚。

许若存看着他。

“她不是想出来。”纪无咎说。

他的呼吸开始发沉，每吸一口气，肋下的伤口就抽一下。

“她是想让你进去。”

许若存的肩膀动了一下。

纪无咎用尽最后的力气，把话说完。

“门后……有人在等你。”

最后一个字落下，他向前倒去。许若存伸手扶住他的肩，手指触到一片湿热的血。

缝里的光又跳了一下。

那光比先前更亮，像有人在门后面应了一声。

贺姓女修还跪在原地，看着那滩渗入土里的净尽。她的手腕垂在身侧，已经抬不起来。

圆里的暗青色纹路慢慢淡了下去。

但裂缝还在。
