# 第81章 余烬

岁蚀的第一波浪潮过去了。

城墙没有完全倒——只倒了一段，大约十来丈。承序府和涤心院都撤了，他们需要时间来重新布防，也需要时间来弄明白，为什么墙会从里面塌。

许若存没有走进那道豁口。

他在豁口前面站了很久——久到风从城外吹回来，吹过他的脸和衣领，把一些不属于这里的味道带到他的鼻腔里。那是土、干草、和一点极淡的水腥气。和十六年前无名小城的旧货市场清晨，空气中的味道一样。

他收回那只已经跨出去半步的脚，后退了两步。

不是因为害怕——是因为他看见豁口里走出来的人。

不是军队。不是修行者。

是一群穿着旧衣裳的人。衣裳的样式不是洛京的，不是这个年代的——料子洗到发白，边角磨出毛边，像放在箱底压了很多年忽然被翻出来穿上。

他们没有名字。没有面容。没有一个人能认出自己的同伴，也没有一个人记得自己是从哪里来的。

但他们从那道坍塌的墙里走出来，一步一步，踩在碎砖和灰浆上，走到洛京的阳光里。

街边一个蹲在地上哭的老太太抬起头，和其中一个走过来的人对上了目光。两个人都没有动。过了很久，老太太开口说："你认识我吗。"那人没有回答，但他在她面前蹲下来，用手背擦了一下她脸上的泪。动作很轻，像是怕碰碎她。他不认识她，也不认识自己。但那个动作——天底下只有一种人会用这种力道去擦别人的眼泪。

纪无咎站在许若存身边，看着那群人从豁口里出来。

他忽然笑了一声。很短，像喉咙里挤出来的一口气。

"你爹开的那道缝——不是用来进去的。"

许若存转头看他。

"是用来让他们出来的。"

许若存没有说话。他想起许衡信里的那句话——"门不能关，但你可以在门边上开一道缝。"

他用了十六年来理解这句话。

他以为那道缝是留给他的。留给他的选择，留给他的退路。是给所有被留在门后的人——那些被人忘记在家里每个角落里的呼吸与印记——开的一道门。

那群人里走出一个老太太。她走得很慢，每走一步都要停一下，像是要确认脚下的砖能承住她的重量。

她径直走到许若存面前，停住了。

她看着他。看了一会儿。她的视力应该已经很差了，但她凑得很近，近到能看清他眉尾那颗很小的痣。

她用一种很哑、很低的声音说了一句话。

"你长得像你娘。"

许若存站在那里。

没有动。没有回答。但他左肩的封印跳了一下，很轻，像有人在很远很远的地方，隔着墙，隔着十六年，隔着那扇没有完全关上的门，用手指轻轻敲了一下。
