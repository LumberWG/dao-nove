# 《岁蚀》写作规范

## 写作前 · 必读文档

在开始写或修改任何一章**之前**，必须先读取以下设定文档（按优先级）：

1. `docs/核心设定.md` — 世界观规则、三条主线、写作原则
2. `docs/人物档案.md` — 所有角色设定（拆分明细：柳三娘=客店，秦望舒=纸坊）
3. `docs/人物出场记录.md` — 各角色在各卷的出场安排
4. `docs/伏笔与线索追踪.md` — 已埋的伏笔状态（未回收/已揭示/已使用）
5. `docs/世界观设定.md` — 隙径、岁蚀、修行境界、物价等具体设定
6. `docs/系统与情节对照表.md` — 各卷关键NPC和系统展开时间线

如果是修改已有章节，还要额外读取该章节原文和前后各一章。

## 写作后 · 自动对照检查

写完或改完一章后，必须逐项检查：

### ☐ 角色一致性
- 新出现的角色名是否已在人物档案/出场记录中注册？
- 已有角色的性格、身份、台词风格是否与设定一致？
- 柳三娘是客店老板娘，秦望舒是纸坊老板娘——身份不能混用

### ☐ 情节连续性
- 本章事件是否与前后章节衔接？（可查 系统与情节对照表 中的时间线）
- 是否有伏笔应该回收但遗漏了？（查 伏笔与线索追踪 中预期回收章节为本章的条目）
- 如果有新埋的伏笔，是否记录了？（追加到 伏笔与线索追踪 末尾）
- **前后章交叉检查**：必须读本章的前一章和后一章原文，逐条对照——角色在这一章知道的事、去过的地方、见过的人，不能在后一章被写成"不知道/没去过"。反之亦然。
- **角色知识边界**：角色只能知道到这章为止发生过的事。不能把他后面才经历的事提前拿来用，也不能忽略他前面已经做过的事。

### ☐ 世界观一致性
- 隙径的使用规则是否一致（入口、晕眩、记忆渗漏、散掉等）
- **隙径量子规则**：叠加态（走隙径时同时存在于多处）→ 观测坍缩（"不要回头"）→ 退相干（记忆/感觉泄漏）→ 量子互换（捡到别人的感觉）。前后章需保持同一套逻辑
- 修行境界是否对应正确的卷（感气卷一、通法卷二等）
- 物价/货币是否与世界观设定一致

### ☐ 笔法风格
- 感情线走含蓄路线，不说破（参考 笔法核心规则.md）
- 对话符合角色性格，不出现与设定矛盾的口头禅或知识面

## 一致性检查脚本

