## 二、群像场景规则（基于已写章节的实践经验）

### 多人物场景的写作顺序

当同时有4个以上角色在场时，写作遵循以下顺序：

1. **谁先动**——第一个做出动作/说话的角色
2. **谁在反应**——对第一个人做出反应的角色
3. **谁在旁观**——注意到但没有干预的角色
4. **谁在暗处**——在场但未被注意到的角色

**示例·第72章纸坊议事**：
```
纪无咎先开口→叶知秋想接话被拦住→沈见微接着说→谢渊从窗口转身
→所有人看向许若存→许若存沉默后说"我去找一个人"
```

### 遮住名字能认出是谁——各角色标志性说话方式

| 人物 | 标志性说话方式 | 示例 |
|------|--------------|------|
| **许若存** | 嘴贫带刺，用玩笑藏真心 | "你这个人跟我很像"/"不像，你嘴比我碎" |
| **沈见微** | 话少，用词准确，偶尔冷幽默 | "冷。但比净心堂暖一点。" |
| **纪无咎** | 短，稳，不用问句表态 | "你们逃。我年纪大了，不想再逃了。" |
| **灵犀** | 轻，干冷，不解释 | "前面有水。" |
| **谢渊** | 认真到好笑，用陈述句表观点 | "完整的秩序是假象，缺口才是入口。" |
| **苏缠** | 轻，压着说，每个字都像在违规 | "我叫苏缠。你们可以忘了。" |
| **叶知秋** | 犹豫，低，用词谨慎 | "怕。但我这辈子就这一回。" |

### "每人一个专属时刻"规则

在场4人以上的场景，以下三种角色最容易变成"背景板"，需要专门分配专属动作：

| 高危角色 | 原因 | 解决方案 |
|---------|------|---------|
| **灵犀** | 话少，容易变成"跟在后面的影子" | 给她感官观察的机会（她是最敏锐的感知者） |
| **叶知秋** | 非战斗人员，在战斗场景中无定位 | 让他负责"注意到别人没注意到的细节" |
| **谢渊** | 独立性强，在群体中容易游离 | 给他一个"从外部看群体"的视角瞬间 |

### 群像场景长度控制

| 在场人数 | 建议字数 | 每人专属时刻数 |
|:-------:|:-------:|:-------------:|
| 2-3人 | 2000-3000字 | 每人2-3个时刻 |
| 4-5人 | 3000-4000字 | 每人1-2个时刻 |
| 6人以上 | 不超过4000字 | 每人至少1个时刻 |

```
           许若存

    灵犀  沈见微  纪无咎
      |     |       |

    妹妹  谢渊   叶知秋
```

### 关键关系

| 关系对 | 关系类型 | 核心互动模式 |
|--------|---------|-------------|
| 许若存↔沈见微 | **镜像** | 我有的你没有，你有的我没有——互为缺失的那一半 |
| 许若存↔纪无咎 | **父辈传承** | 从追捕者到"替你爹还人情"——纪无咎通过许若存完成与许衡的未竟之路 |
| 许若存↔灵犀 | **过命的同行** | 不解释的信任——她相信他会在隙径里等她，他相信她不会丢下他走 |

| 沈见微↔纪无咎 | **体制内外** | 同一个体制的一体两面——一个从内部打破，一个从外部拆墙 |
| 沈见微↔灵犀 | **感知者之间** | 两人都有"被洗掉/被忘记"的共性——一个在记忆层面，一个在隙径层面 |

## 四、人物加入/离开原则

### 加入原则

1. **必须有功能缺口**——队伍缺某个能力时，新角色加入才自然
2. **必须有情感阻力**——不是"来了就加入"，要有信任门槛
3. **必须有代价**——加入行为本身应让角色付出某种代价（纪无咎断令牌、苏缠断月丝）

### 离开原则

1. **功能完成即可退场**——何伯安（传完话就退）、曲姓守塔人（传完钥匙信息就退）
2. **退场不是死亡**——老陈在归寂院是"散掉"而非死亡，保留了回忆线
3. **退场角色可回收**——柳三娘、红姑、何伯安在后续卷中仍可临时出现
