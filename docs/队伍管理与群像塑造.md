# 《岁蚀》队伍管理与群像塑造

## 二、性格互补矩阵

### 每一对都有独特的化学反应

| 对子 | 互动模式 | 代表性场景 |
|------|---------|-----------|
| **许若存 ↔ 灵犀** | 话多 vs 话少。他贫嘴，她只回半句——那半句比他说的一整段都有意思 | 隙径中：许若存紧张就话多，灵犀不回应但在他快散掉时握他手腕 |
| **许若存 ↔ 谢渊** | 嘴贫 vs 板正。他开玩笑，谢渊认真接——接的方式让玩笑变成哲学 | 第16章：许若存"我就路过"，谢渊"你跟我走了四家摊位" |
| **许若存 ↔ 苏缠** | 靠近 vs 灼伤。想靠近但不能——不是不想，是身体不允许。量子纠缠式 | 第31章：靠近就痛，远离就牵挂 |
| **灵犀 ↔ 谢渊** | 沉默型互相理解。两个都不爱说话的人，话最少但默契最深 | 待写：两人同守夜，一炷香没说一句话，但都知道对方醒着 |
| **霍斩 ↔ 季真** | 豪爽 vs 沉稳。一个出手快，一个收手稳。打架风格完全不同 | 待写：霍斩劈开门，季真按住他"里面是空的，你那一刀会把自己带进去" |
| **谢渊 ↔ 苏缠** | 哲学 vs 感知。谢渊用理论理解世界，苏缠用身体感应世界 | 待写：谢渊问"你的红绳感应的是什么"，苏缠"你自己碰一下就知道了"——他碰了，被灼了一下 |

### 三人以上互动规则

- **争执场景**：必须有两种以上不同意见。不是主角一锤定音——有时候主角的意见反而是错的
- **日常场景**：每8-10章至少有一场"非推进情节的日常"——吃饭、赶路、等隙径激活。日常场景是性格展示窗口
- **决策场景**：重要决策前，每个在场的人必须表态。表态反映性格——不是谁对谁错

## 三、每个人的独立叙事线

> 每个人物有一条**独立于许若存的个人问题线**。这条线与主线交叉，但不依附于主线。

| 人物 | 个人问题线 | 推进节奏 | 关键节点 |
|------|-----------|---------|---------|
| **灵犀** | "我没有八岁前的记忆。最早记忆是从隙径里走出来。"——她从哪里来？隙径为什么对她不产生退相干？ | 每2-3次出场推进一点 | 卷二：隙径崩塌时镜子反转看到自己→卷三：与天道代理人连接→卷六：身份揭示 |
| **谢渊** | 不周山庄说"完整是假象"——他要打一把陨铁剑，给谁打的？师尊问"想好了？"意味着这趟下山有代价 | 卷二初展开→卷三深入→卷五揭示 | 打剑→剑成→送剑的那个人出现→代价揭晓 |
| **苏缠** | "我分不清我想靠近你是因为你的频率还是因为我自己"——红绳是能力还是牢笼？解开之后她还是苏缠吗？ | 每5-6章出场推进一次 | 卷三：第一次主动解开红绳→卷四：去废界开布料铺→卷七：布上纹路"没什么意思，好看" |
| **霍斩** | 废界武修，刀是师父留下的。师父说"这把刀没杀过人"——他不信。他在查刀的来历 | 卷三展开 | 刀的故事→师父的真相→挂刀的选择 |
| **季真** | "收剑和出剑一样重要"——她以前收剑太慢了，代价是什么？ | 卷四展开 | 过去揭示→面对→收剑 |

### 推进原则

1. **不急于揭示**。个人问题线是长线，每卷只推进1-2个节点。
2. **不自说自话**。人物不会主动讲自己的过去——是在事件中被迫暴露出来的。
3. **交叉触发**。一个人的个人线节点，常常由另一个人的行为触发。灵犀的身世线索可能藏在谢渊办的一件事里。

## 四、人物加入原则

### 准入标准

新人物加入队伍前，必须同时满足：

| 条件 | 说明 |
|------|------|
| **有自己的事** | 不是来找许若存的——是来做自己的事，恰好同路 |
| **有独立性格** | 不能和已有成员性格重叠。加入前检查：TA的说话方式、决策风格、幽默感是否与已有6人区分开 |
| **有独立问题** | 有一条不依附于主线、也不依附于许若存的个人问题线 |
| **离开的理由** | 加入时就想好：TA什么时候离开？为什么？离开之后去哪里？ |

### 反模式（避免）

| 反模式 | 为什么不行 |
|--------|-----------|
| "久仰大名特来追随"型 | 人物没了独立人格，变成主角的崇拜者 |
| "我就是来帮忙的"型 | 没有自己的事，变成工具人 |
| "我跟你一样惨"型 | 和已有角色情感光谱重叠 |
| "谜语人"型 | 每次出场只说半句话，永远不推进个人线 |

### 离开的设计

> **人物的退场和出场一样重要。** 不是所有人都陪到最后。有的人在某个节点自然离开——不是因为死了，是因为他的事做完了。

| 离开类型 | 例子 |
|---------|------|
| **事做完了** | 霍斩查到刀的真相→挂刀回废界帮村里修屋顶。不是退隐——是换了一种方式做自己 |
| **选择了另一条路** | 季真去无名坟前拔草——她的代价在那里，她选择去面对 |
| **主动放手** | 苏缠解开红绳，叠好放木盒，"去废界开布料铺" |
| **牺牲** | 沈见微站上石门化为云纹——不是被打败，是看清一切后的清醒选择 |
| **再也不见** | 许衡在门后——物理上还在，但就是见不到 |

## 五、多人场景写作规则

### POV分配

| 场景类型 | POV规则 |
|---------|--------|
| **日常** | 许若存POV为主，偶尔切到当天"主场"的那个人 |
| **战斗** | 谁的动作最关键切谁的POV——不一定是许若存 |
| **决策** | 必须给在场每个人至少一句表态。不表态的人给动作 |
| **个人线高潮** | 以那个人为POV——许若存是旁观者 |
| **势力内部** | 独立POV——涤心院、风闻司、承序府（许若存不在场） |

### 对话密度

- **3人场景**：每个人必须有话。不能一个人说话另一个捧哏。
- **4人+场景**：可以有沉默者，但沉默者要给动作/表情/一个细节。读者知道他在想什么，哪怕他没说。
- **全队场景**（6人+）：每章最多一次。全队场景必须有至少两个子对话同时进行（角落里的、走在前面的），不是所有人围着一个人听。

### 避免"排队说话"

> 最差的群像写法：A说一句→B说一句→C说一句→D说一句，像排队领饭。

**解法**：
- 打断。B没等A说完就接。
- 沉默。有人选择不说话，但用动作表态。
- 离题。有人走神想自己的事，突然插一句看似不相关的。
- 动作压过语言。有人直接动手。

## 八、检查清单（每章写完后自检）

### 群像章

- ☐ 本章在场的人，每个人都至少有一个"只有TA会这么做"的瞬间？
- ☐ 有没有人整章只说了"嗯""好""对"？（有的话要么加内容，要么给一个动作说明为什么沉默）
- ☐ 新角色的说话方式和已有角色区分开了吗？（把台词遮住名字读一遍——能认出是谁说的吗？）
- ☐ 至少有一个非许若存的角色推动了情节？（不是"许若存决定往东→大家往东"，是有人主动做了什么）

### 个人线章

- ☐ 这个人的个人问题往前推进了吗？（哪怕只推进1mm）
- ☐ TA的过去是"被迫暴露"的，不是"主动讲述"的？
- ☐ 许若存在这一章里对TA的态度有变化吗？（不需要说破——一个停顿、多看一眼、少开一句玩笑就够了）
