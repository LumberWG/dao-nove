# 《岁蚀》域外设定与过渡设计

> 说明“域外”是什么、它如何运作、以及如何从第1-20章的铺垫自然过渡到域外场景。
> 不影响第1-20章已发表内容。

---

## 一、域外是什么

### 1.1 一句话定义

域外不是另一个世界，是天道被切掉、被遗忘、被压扁的那一部分。

它不是“外面”，是“被删除的里面”。

### 1.2 与常见设定的区别

| 常见玄幻设定 | 岁蚀中的域外 |
|-------------|------------|
| 仙界/魔界/妖界等独立位面 | 不是独立位面，是天道的伤口 |
| 可以通过修炼飞升到达 | 不能修炼到达，只能通过“门”或“隙径深渊”短暂进入 |
| 有完整生态和文明 | 没有稳定形态，没有名字，没有连续的时间 |
| 强者可以长期生存 | 进入者会快速“散掉”，除非有锚定物 |

### 1.3 域外的本质

天道原本是完整的。完整不是“完美”，而是“所有可能性都还在流动”。

后来有人试图把天道“固定”下来——修门、筑塔、写律、设阵。天道没有反抗，只是裂开了一道缝。被切掉的那部分，就是域外。

所以域外里保存的，是**所有被放弃的可能性**：
- 没有选择的那条路
- 被遗忘的人
- 被删除的记忆
- 没有成为现实的现实

## 二、域外的基本规则

### 2.1 没有“名字”

在域外，没有名字的东西才能存在，有名字的东西会立刻被稀释。

这与凡间相反。凡间是“有名字才能被记住”，域外是“有名字就会被固定，而固定意味着死亡”。

所以进入域外的人，第一件事是忘记自己的名字。不是真的忘记，是暂时不把自己当成“谁”。

### 2.2 时间不是线性的

域外没有“过去、现在、未来”，只有“可能性的叠加”。

一个人在域外可能同时看见自己五岁、十五岁、二十五岁的状态。他们不是倒影，都是“可能的自己”。

### 2.3 空间不是连续的

域外没有“从这里到那里”，只有“你记得就存在，不记得就不存在”。

走进去的人不能靠走，要靠“想”。但“想”太多会让自己散掉。

### 2.4 情绪比物质更真实

在域外，情绪是唯一的稳定坐标。
- 强烈的牵挂可以当锚
- 强烈的恐惧会吸引东西
- 强烈的执念会形成暂时的“陆地”

### 2.5 域外的“居民”

域外没有生命，但有“存在过的痕迹”。

| 类型 | 说明 | 在第1-20章的体现 |
|------|------|----------------|
| **镜中人** | 走隙径时散掉的人留下的残影 | 第8章破庙里搭许若存肩膀的女人声音 |
| **讨债者** | 门环断了之后跑出来的“旧秩序看守者” | 第17章青石集阴影里浮着的人 |
| **未择之身** | 被放弃的可能性凝聚成的实体 | 许若存前世记忆里的那个自己 |
| **母亲** | 用身体按住门十六年，已经半域外化 | 第6章心湖影像、第7章门上的手 |
| **许衡** | 与门共生，部分本源在域外 | 尚未直接出场 |

## 三、域外与门、岁蚀、隙径的关系

### 3.1 门是伤口，不是入口

门是连接凡间与域外的伤口。门关得越紧，凡间越稳定；门开得越大，域外漏进来越多。

但门不可能永远关死，因为天道本来就是要流动的。强行关门只是在积累压力。

### 3.2 岁蚀是域外的渗透

岁蚀不是自然灾害，是域外向凡间“回忆”自己。

- 无声区域：域外正在“覆盖”凡间，让这片区域回到“未被选择”的状态。
- 人互不认识：域外让社会关系回到“未被命名”的状态。
- 植物灰化：域外让物质回到“未被确认存在”的状态。

### 3.3 隙径是域外的毛细血管

隙径不是物理通道，是域外在凡间的细小裂口。

- 走隙径等于在域外的边缘蹭过去。
- 走得太深会进入域外。
- 不要回头，是因为回头等于“确认自己还在原来的位置”，会在叠加态中坍缩到不可控状态。

## 四、域外中的关键存在

### 4.1 母亲

母亲不是被困在门里，她是门的一部分。

十六年来，她用右手按门，左手推开许若存。她把自己变成了门和凡间之间的缓冲带。

在域外视角里，母亲已经不是一个完整的人，而是一股“不想让孩子被吸进去”的执念。她的形象在域外中反复出现：
- 有时是按门的女人
- 有时是推婴儿的手
- 有时是叫“若存”的声音

### 4.2 许衡

许衡在域外，但他不是“活着”。

他与门共生，把自己一部分本源撕下来塞给儿子后，剩下的部分和门长在一起。他在域外中以一种破碎的状态存在：
- 不能完整说话
- 不能离开门太远
- 但能感觉到儿子越来越近

他在域外中给许若存留的路标，不是因为他能算准每一步，而是因为他把自己嵌进了门的可能性里。

### 4.3 前世的许若存

前世的许若存不是转世，是“未被选择的那条路”。

 Xu Heng 把本源塞进婴儿许若存体内时，也塞进了一些属于“另一个可能”的记忆和性格。这些记忆在域外中被放大，形成了一个越来越清晰的“另一个许若存”。

这个前世的自己，会试图覆盖现在的许若存。因为他认为：如果当年没有被推开，他才是“真正的许若存”。

### 4.4 域外深处的“东西”

第7章中，门后黑暗深处有东西在缓慢移动。那不是某个具体生物，是**所有被放弃可能性的集合意志**。

它没有善恶，它只想被记住。被记住越多，它越稳定；被遗忘越多，它越躁动。

## 五、如何自然过渡到域外

过渡不能突然，必须层层递进。以下是五层过渡设计。

### 5.1 第一层：通过记忆渗透（已实现）

第8章中，许若存走隙径后丢了对马婶的难过，捡了别人对一个馒头的愧疚。

这是域外第一次接触读者：不是空间穿越，而是记忆/情绪的交换。

**后续用法**：许若存会越来越多地“想起”不属于自己的记忆，这些记忆来自域外。

### 5.2 第二层：通过岁蚀区域（已实现+可扩展）

第14章中，许若存进入金泉镇外那片无声灰白区域。

这是域外在凡间的“滩头阵地”。那片区域里，凡间的规则开始失效，域外的规则开始生效。

**后续用法**：
- 岁蚀区域会越来越大
- 许若存需要学会在岁蚀区域中行动
- 最终他会发现，岁蚀区域不是“被污染”，是“被还原”

### 5.3 第三层：通过门缝窥视（已实现）

第7章中，许老头帮许若存内视封印，他第一次“看见”门和门后的黑暗。

这不是真正进入域外，是透过门缝看了一眼。

**后续用法**：
- 许若存会越来越频繁地“看到”门后的景象
- 这些景象从模糊到清晰
- 他会在这些景象中听到母亲和许衡的声音

### 5.4 第四层：通过深度隙径（卷三/卷四）

普通隙径是绕过域外边缘。深度隙径是穿过域外。

**触发条件**：
- 铜钱的指引偏离常规路线
- 灵犀带许若存走一条“她也不知道出口在哪”的隙径
- 追兵把许若存逼入死角，他只能选择最深的隙径

**域外表现**：
- 许若存会同时看见多个自己
- 他需要选择“哪一个自己”走出来
- 选择错误会被前世的自己覆盖

### 5.5 第五层：真正推门（卷五/卷六）

最终，许若存会真正进入门后。

这不是为了打败什么，是为了：
- 把母亲从门里带出来
- 让许衡完成最后的告别
- 让前世的自己有一个答案
- 让门不再需要被推开或被按住

## 六、域外在各卷中的作用

### 卷一 · 旧雨不来

域外只作为背景存在：
- 左肩封印的脉动
- 心湖母亲的影像
- 第7章门后的惊鸿一瞥
- 第8章隙径中的记忆交换
- 第14章岁蚀无声区域

**作用**：让读者隐约感到“有什么东西在另一边”，但不用解释清楚。

### 卷二 · 序隙之裂

域外开始影响洛京：
- 序塔底部有“呼吸声”
- 洛京地下通道连接域外边缘
- 沈见微的梦境越来越频繁地出现域外景象
- 纪无咎展示收缴司收藏的门遗物

**作用**：把域外从“远方的恐怖”变成“身边的异常”。

### 卷三 · 万流暗涌

深度隙径暴露域外的一部分：
- 灵犀带许若存走一条穿过域外的隙径
- 许若存在域外边缘看见前世的片段
- 万流坊的“旧物司”交易域外带出的物品

**作用**：让读者第一次真正“进入”域外，但只是边缘。

### 卷四 · 记忆吞噬

域外与记忆主题结合：
- 照影台映照许若存，让他看见前世的自己
- 沈见微在涤心院深层净化中，被推向域外边缘
- 岁蚀大规模降临，多个城市出现无声区域

**作用**：域外成为“被遗忘之物”的具象化空间。

### 卷五 · 虚空不空

许若存与许衡在域外重逢：
- 许衡已经半域外化，不能完整表达
- 许若存通过父亲的残留记忆拼凑真相
- 母亲的状态揭示：推门十六年不是被困，是自愿选择

**作用**：域外从“恐怖空间”变成“有情感重量的空间”。

### 卷六 · 前世来生

前世的许若存成为实体威胁：
- 前世的自己试图通过门完全进入凡间
- 许若存必须决定：消灭他，还是接纳他
- 照影台、不周山庄、涤心院三方势力对“如何处理前世”有不同主张

**作用**：把域外的冲突转化为许若存的内在冲突。

### 卷七 · 后来不来

最终解决：
- 许若存不推门，也不挡门
- 他选择“记住一切”，让门不再需要存在
- 母亲可以放手，许衡可以安息，前世的自己可以被承认
- 域外不再是被遗忘的角落，而是被重新纳入天道的可能性

**作用**：域外不是被征服，是被接纳。

## 七、域外场景写作原则

### 7.1 不要解释，要感受

错误：
> “域外是天道被切掉的部分，这里没有时间和空间……”

正确：
> “他往前走了一步，脚下的地面还在，但他不确定这是刚才那块地面。他回头，看见自己还站在原地。两个他同时存在，都在往前走。”

### 7.2 用身体反应代替环境描写

域外的恐怖不在景象，在身体感受：
- 名字在舌尖融化
- 心跳声从胸腔传到背后
- 影子比自己慢一拍
- 想哭但想不起为什么

### 7.3 每次进入域外都必须有“锚”

许若存不能空手进域外，必须有一样东西把他拉回：
- 铜钱
- 玉佩
- 对母亲的牵挂
- 柳三娘那碗面的味道
- 沈见微掌纹里的分叉

没有锚，他就会散掉。

### 7.4 域外中的对话要简短、重复、像回声

域外中的存在不直接说话，它们重复人心里已有的东西。

例如：
> “你丢的东西在这里。”
> “别人丢的东西，你为什么要捡？”

这些话不是问句，是域外在映照许若存自己的念头。

### 7.5 不要让域外变成“另一个地图”

域外不是副本，不是练级场。每次进入都必须是剧情必需，必须有代价，必须有变化。

## 八、关键场景示例

### 示例1：深度隙径中的域外边缘

许若存跟着灵犀走进一条比平时更黑的隙径。

走着走着，灵犀停下脚步。

“再往前，我就不能带了。”她说。

“为什么？”

“前面不是隙径。前面是域外。”

许若存往前看，黑暗中有什么东西在动。不是走，不是爬，是像一整片黑暗在翻转自己的身体。

他低头看自己的手。手的轮廓在变淡。

“记住一样东西。”灵犀说，“什么都行。想不起来，你就不是你了。”

许若存闭上眼睛。他想起一碗面。清汤，手擀面，中间卧着一个荷包蛋。

他握紧那碗面的记忆，走了进去。

---

### 示例2：门后的重逢

许若存穿过门，来到一片灰色的虚空。

远处有一个人影，背对着他，右手前伸，掌心贴着一扇看不见的门。

“爹。”

那人影没有回头。他的声音从四面八方传来，像很多人同时说话：

“你来了。我等了很久。也等不了多久了。”

许若存想走近，但每一步都让他变小。不是身体变小，是他身上“被记住的东西”在减少。

他停下脚步，从怀里掏出铜钱。

铜钱在域外里发出暗淡的光。借着那光，他看见父亲的轮廓稳定了一些。

“我来不是接你回去。”许若存说，“我来问你一个问题。”

“问。”

“门后面是什么？”

父亲的笑声很轻，像风穿过门缝：

“门后面不是你想的那样。门后面是所有你没选的路，都在这里。”

## 九、常见误区提醒

1. **不要把域外写成地狱或魔界** —— 它没有善恶，只有被遗忘。
2. **不要让主角在域外自由行动** —— 每次进入都应有锚和代价。
3. **不要用域外解释一切** —— 它是背景，不是万能答案。
4. **不要让域外有完整社会** —— 它没有稳定文明，只有碎片。
5. **不要忘记域外的情感核心** —— 它是关于“被放弃的可能性”的悲伤空间。

## 十、与现有设定的衔接清单

| 已有设定 | 如何衔接域外 |
|---------|------------|
| 门是伤口 | 门是域外与凡间的伤口 |
| 岁蚀是遗忘泄露 | 岁蚀是域外向凡间渗透 |
| 隙径是量子效应 | 隙径绕过或穿过域外边缘 |
| 未择之身 | 许若存能看见域外中的可能性 |
| 铜钱指路 | 铜钱指向的不是地点，是域外中最需要被记住的坐标 |
| 母亲推门十六年 | 母亲在域外边缘缓冲凡间与域外 |
| 许衡与门共生 | 许衡部分存在于域外，能留下路标但不能离开 |
| 沈见微被洗记忆 | 被洗掉的记忆流入域外，成为域外的一部分 |
| 红姑靠“名”修行 | 她的名字修行与域外中“名字的力量”相反相成 |
| 不周山庄“缺口是入口” | 他们承认域外是天道的一部分，不试图关门 |
