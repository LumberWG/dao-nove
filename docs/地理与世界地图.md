## 二、主要地点详录

### 无名小城

| 属性 | 内容 |
|------|------|
| **类型** | 凡人居住的边境小城 |
| **特征** | 旧货市场、城东心湖、城西废屋 |
| **保护网** | 许老头、柳三娘(曾在此开店)、瞎老太太、老孙头、老王、陈婶等 |
| **在卷一中的作用** | 起点。许若存离开后不可回归 |
| **与洛京距离** | 约5-7天脚程（许若存通过隙径缩短了时间） |

### 落霞镇

| 属性 | 内容 |
|------|------|
| **类型** | 官道旁的中转镇 |
| **重要地点** | 三娘客店（柳三娘）、福来客栈、镇北老槐树 |
| **相关事件** | 第10章柳三娘登场、第11章追兵水塘战、第12章沈见微首次正面接触 |
| **与青石集距离** | 约1天脚程 |

### 青石集

| 属性 | 内容 |
|------|------|
| **类型** | 各路势力汇聚的集镇 |
| **重要地点** | 老槐树下说书场（红姑）、各色茶棚 |
| **相关事件** | 第16-19章：何伯安认铜钱、红姑说老陈的故事、纪无咎正式登场、韩指挥使写"不动" |
| **势力分布** | 万流坊有节点、承序府有眼线、风闻司在监视 |

### 洛京

| 属性 | 内容 |
|------|------|
| **类型** | 昭朝都城，全书核心城市 |
| **人口** | 密集，凡人+修行者混居 |
| **重要地点** | 序塔、涤心院（城西）、承序府（城东）、纸坊（秦望舒）、入世堂、城西土地庙 |
| **城墙** | 第70-80章从内裂开。岁蚀最先从城墙开始 |
| **特殊状态** | 卷一末城墙坍塌一段，被门留下的人走出 |

## 三、通行时间参考

| 路线 | 方式 | 时间 | 出现章 |
|------|------|:----:|:------:|
| 无名小城→落霞镇 | 隙径 | 很短（第8章） | 8 |
| 无名小城→落霞镇 | 步行 | 数日 | — |
| 落霞镇→青石集 | 步行 | 约1天 | 16 |
| 青石集→洛京 | 隙径 | 很短（第21章） | 21 |
| 青石集→洛京 | 官道 | 约3-4天 | — |
| 洛京出城→废驿站 | 步行 | 约1天 | 43 |
| 废驿站→青石镇 | 步行 | 约2天 | 54 |
| 青石镇→河边村 | 步行 | 约1天 | 57 |
| 河边村→洛京（回程） | 步行 | 约2-3天 | 66-67 |
| 洛京→倒流河源头 | 步行 | 约5-7天 | 🔮 卷二 |

**通行速度参考**：
- 普通步行：约40-50里/天
- 急行/逃亡：约60-80里/天
- 隙径穿越：瞬间（但有记忆泄漏风险）
- 骑马：约80-100里/天
