# 前10章保护网暗线设计

> 核心设定：许若存十六年里不是被"放养"在旧货市场，而是被一张由许衡、许老头、万流坊早年布置的保护网围着。这张网平时不出手，只在玉佩认主、各方势力开始动的节点上，一点一点把许若存往外推。

---

## 一、保护网的构成

### 第一层：旧货市场（日常遮蔽）

**功能**：让许若存看起来像普通人，同时在关键时刻替他挡眼、传信、制造盲区。

| 人物 | 表面身份 | 实际角色 | 保护行为 |
|------|---------|---------|---------|
| 许老头 | 旧货摊主 | 抚养人、节点守护者 | 收养、教本事、最后断后 |
| 瞎老太太 | 隔壁摊主 | 外围眼线 | 第一个发现木盒异常，用吵架打掩护 |
| 老孙头 | 豆浆摊主 | 消息缓冲 | 帮许若存确认"有没有外乡人" |
| 小六子 | 跑腿少年 | 移动哨 | 不是偶然看见灰袍人，是被安排盯西城 |
| 陈婶 | 包子摊主 | 补给点 | 每天给许老头留包子，是"人活着"的信号 |

**关键原则**：这些人平时是真的在讨生活，不是职业护卫。他们被许老头或更早的人欠过人情、拿过钱、或者被托付过一件事。平时不联络，出事时按约定动作。

### 第二层：信息人（关键节点传递）

**功能**：在许若存拿到玉佩后，把必须知道的信息一点一点塞给他，但又不能让他一下子知道太多。

| 人物 | 传递信息 | 保护方式 |
|------|---------|---------|
| 红姑 | 老陈的故事、铜钱来历 | 用说书当掩护，让信息看起来像故事 |
| 疯乞丐 | "两条命"、"城东水面" | 装疯，说真话也不会被当真 |
| 孟婆婆 | 心湖禁忌、"水认识你" | 守湖人，替他挡第一道水险 |
| 钓鱼老人 | "这湖得有人看着" | 三十年守湖，不让普通人接近核心 |

### 第三层：暗线人物（最后出手）

| 人物 | 角色 |
|------|------|
| 灵犀 | 万流坊接应人，带他走隙径 |
| 老陈 | 守门人，等他来落子 |
| 柳三娘 | 逃亡路上的补给站，替他挡镇魔司 |

## 二、需要在前10章里埋的暗线

### 第1章

1. **瞎老太太不是普通邻居**：她知道那个木盒今早才出现，许老头往摊上放的时候她看见了。她故意跟许若存吵架，是为了让周围人觉得"这只是两个旧货摊主在斗嘴"，掩护许若存研究盒子。
2. **许老头给玉佩时的不自然**：他嘴上"不要钱"，但那个"嘴角往左偏"的习惯性动作，说明他在演戏。他不是真的舍不得，他是演给可能存在的眼睛看。
3. **外乡人的对话**：那两个外乡人（背药箱、扛扁担）不是随便聊天。其中一个人说完"涤心院洗心"之后，往许若存这个方向看了一眼。他们是万流坊或类似组织故意放出来的"消息源"，让许若存提前知道世界有多大。

### 第2章

1. **老孙头知道许若存会出事**：许若存能力觉醒、脸色变了的时候，老孙头没有多问，只是默默把辣油放下。他受过交代：这孩子如果有一天不对劲，别拦，让他走。
2. **小六子不是偶然看见灰袍人**：他能在西城"跑腿"时正好听见灰袍人打听许老头，是因为他本来就在那条线上活动。他往后跑的时候，不是跑回家，是跑向旧货市场方向——他在按约定传信。
3. **疯乞丐的"两条命"**：这句话不是疯话，是有人教他的。他说完就走了，没有纠缠，说明他完成了任务。

### 第3章

1. **陈婶的包子**：她每天留包子，不只是照顾许老头。如果有一天她不留了，就是"出事了"的信号。这一天她照常留，说明外围还稳。
2. **老王那只八哥**：八哥学咳嗽的声音，可以改成每天早上固定叫几声。这是旧货市场几个老人约定的"平安信号"。如果有一天不学了，就是预警。
3. **窗外的影子**：那个影子不是第一次来，但许老头知道他不会在白天动手。他只是在测许若存的反应。

### 第4-6章

1. **红姑的青衣人**：青衣人可以是另一股势力，也可以是保护网的"反向测试"——看他会不会被错误信息拐走。
2. **孟婆婆的心湖**：她不是普通老太太。她喝了十六年湖里的水，湖认她，她也认湖。她说"水认识你父亲"，是因为她见过许衡。
3. **心湖影像**：母亲影像出现时，北岸洗衣妇人和南岸小孩都看见了。这不是意外，是母亲用最后一点力气把影像推到水面，让许若存看见，同时让普通人也看见——这样承序府就不能随便说"这是幻觉"抹掉。

### 第7-10章

1. **许老头的送别**：他不是临时决定断后，是十六年前就练好了这一天的所有动作。地窖暗道、灰褐色衣服、井口铜钱，都是保护网最后一环。
2. **柳三娘**：她不是偶然开店的。她认识那件灰褐色衣服，认识镇魔司的人，说明她也是网中人。

## 三、修改原则

1. **不挑明**：前10章里不能让许若存或读者明确知道"这些人是一伙的"。只能让细心的读者事后回想时觉得"原来如此"。
2. **每个人有真实生活**：保护者首先是卖豆浆的、卖包子的、说书的、摆摊的。保护任务是副业。
3. **保护网的漏洞**：不是所有人都能挡住。灰衣人、镇魔司、蓝衣寻踪者还是会突破到许若存面前。这样才有张力。
4. **许若存的感受**：他应该会隐隐觉得"这些人对我太好了"，但说不出为什么。这种"不对劲"本身也是悬念。

---

## 四、具体要改的章节文件

按优先级：

1. **第1章**：瞎老太太、许老头给玉佩、外乡人对话
2. **第2章**：老孙头、小六子、疯乞丐
3. **第3章**：陈婶包子、窗外影子、许老头教看货
4. **第5章**：红姑与阿芷的配合
5. **第6章**：孟婆婆和心湖
6. **第10章**：柳三娘认出灰褐色衣服
