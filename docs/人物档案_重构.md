# 《岁蚀》人物档案（重构版）

> 本档案覆盖第1-20章已出场人物及后续卷中预设登场的重要角色。重构目标：去重、分层、补全人物与势力/部门的对应关系，为后续写作提供统一口径。
> 不影响第1-20章已发表内容。

---

## 一、核心主角

### 许若存

| 项目 | 内容 |
|------|------|
| 年龄 | 开篇16岁，全书结束约25岁 |
| 身份 | “未择之身”，旧货摊养大的少年 |
| 性格 | 开朗、嘴贫、有脾气、护短、嘴硬心软 |
| 核心矛盾 | 嘴上什么都不在乎，实际上什么都往心里放；越在乎的人和事，越用玩笑掩饰 |
| 能力 | 未择之身：可看见短期选择的两种未来；代价为流鼻血、消耗父亲残余本源、松动左肩封印 |
| 关键道具 | 碎玉（门之残片）、三枚铜钱（路标） |
| 一句话 | 欠的一定还，但还的方式经常让人哭笑不得 |

**人物弧光**：从“被追杀的逃亡者”到“主动选择的人”。最终他不再逃避选择的代价，即使知道天道归于无序，仍然选择记住一切。

## 二、同行者与潜在盟友

### 沈见微

| 项目 | 内容 |
|------|------|
| 年龄 | 约20岁 |
| 身份 | 镇魔司头号人物，涤心院出身，五岁起洗心三十次 |
| 性格 | 表面冷淡，内心空洞，重诺，正在找回自己 |
| 核心矛盾 | 每个选择都不是选择，是身体先于大脑行动 |
| 关键意象 | 手抖、掌纹分叉、不会做梦→第一个梦、雨 |

**与许若存关系**：追捕者→镜像→同行者。他左肩没有封印，但身体里被洗掉的东西对许若存的封印产生共振。两人在“被塑造的自我”与“真实的自己”之间互为对照。

**后续发展**：
- **卷二**：十日期限到，回院接受洗心，但在洗心中**第一次主动抵抗**，成功保留了一段记忆。这是她从"被动承受"到"主动选择"的第一转折点。发现自己母亲和许若存母亲认识。
- **卷三**：独立追查老廖改过的档案，发现母亲被涂改的名字，与许若存并行推进。
- **卷四**：全面觉醒，从涤心院叛出。梦中母亲说的四个字是卷六关键反转线索。

### 灵犀

| 项目 | 内容 |
|------|------|
| 年龄 | 约16-17岁 |
| 身份 | 隙径向导，万流坊暗线，花姐的线人 |
| 性格 | 话少不闷，干冷幽默，看透不说透 |
| 核心设定 | 八岁之前记忆一片空白；天生部分状态与隙径重叠，走隙径不会退相干 |
| 核心矛盾 | 她是隙径的“本地人”，但不知道自己从哪来 |

**与许若存关系**：向导→同伴。她帮许若存走隙径，最终也需要许若存帮她找回八岁前丢失的记忆。

**后续发展**：
- **卷二末**：通过花姐传信，告知洛京隙径网络即将关闭。向许若存求助——走隙径时听到有声音从深处叫她，她想知道那个声音是谁。
- **卷三**：揭示其身世与隙径网络的”原生居民”有关。

---

### 谢渊

| 项目 | 内容 |
|------|------|
| 年龄 | 约20岁出头 |
| 身份 | 不周山庄弟子，持“问字令”下山 |
| 性格 | 清冷板正，认真到好笑，内在坚定 |
| 核心理念 | 完整是假象，缺口才是入口 |
| 下山目的 | 买陨铁打剑→问一个人→看世界的秩序长什么样 |

**与许若存关系**：观察者→同行者。他不是来帮许若存的，是来验证自己的道。在关键时刻，他会因为相信“碎”而站在许若存这边。

**后续发展**：
- **卷二**：在洛京序塔地下水道与许若存重逢（第28章）。带他破阵、看名字墙、认棋局。他在本卷中的角色定位是”观察者逐渐变成同行者”——嘴上说”我只是来看的”，做的事已经是在站队了。
- **卷三**：带去不周山庄对”门”的更深理解。

## 三、父辈与前代守护者

### 许衡

| 项目 | 内容 |
|------|------|
| 身份 | 许若存之父，守门人，纪无咎旧日同门 |
| 性格 | 沉默、决绝、爱里夹杂了债 |
| 核心行为 | 制造门→掰断门环→与门共生→撕下本源塞给儿子 |
| 核心矛盾 | 他为儿子算好了一切，却没算到自己能否活到最后 |

**人物定位**：全书最大的“缺席者”。他不直接出场，但所有路标都指向他。他的选择是：把未来留给儿子，而不是替儿子走。

## 十、弧2新增角色

### 曲姓人（守塔人）

| 项目 | 内容 |
|------|------|
| 身份 | 许衡旧部，序塔底下的守塔人 |
| 年龄 | 约50余岁，瞎了一只眼 |
| 性格 | 话少，不解释。"我只负责传话，不负责帮你理解" |
| 能力 | 无特殊修为——他靠的是许衡留的铜镜（能显示持铜钱者靠近序塔时的面容）来判断来人。他照着镜子画下了许若存的画像（第21章街头画像的来源） |

| 与许若存关系 | 传信者——替许衡传最后一句话："钥匙在你身上。别找人借。" |
| 后续发展 | 卷五许若存再进塔时重逢。届时揭示：他瞎掉的那只眼，是在许衡进去之后他自己弄瞎的——因为"我答应过你爹，不在塔底下看不该看的东西" |

## 十二、写作使用说明

1. **新增人物登场前**，必须在此档案中备案，确认其势力归属、功能定位、与许若存的信息边界。
2. **同一人物在不同章节出场**，必须保持性格关键词、习惯动作、核心矛盾一致。
3. **人物台词检验**：遮住名字读对话，能认出是谁在说话，才算立住。
4. **后续新增势力/部门人物**，优先从已有势力中生长，不轻易创造全新独立势力。
