# 《岁蚀》人物档案

> 核心人物卡片 + 性格冲突设计。全文约300万字，7卷，主角年龄跨度16-25岁。

> 每个人物必须有自己的矛盾——没有矛盾就没有戏。

---

## 许若存

- **年龄**：开篇16岁，全书结束约25岁

- **身份**：被遗弃在无名小城的"未择之身"，旧货摊主养大

- **性格关键词**：嘴贫记仇·有嘴有脾气·但对自己人绝对护短

- **核心矛盾**：他嘴上什么都不在乎，实际上什么都往心里放。越在乎的人和事，他越用开玩笑来掩饰。这种"嘴硬心软"的性格导致他经常在该说的时候不说，该留的时候不留，该认错的时候不认错——制造大量冲突。

- **和谁冲突**：和许老头（想保护他但被推开）、和沈见微（立场对立但逐渐理解）、和苏缠（想靠近但体质排斥）、和自己（能力越用代价越大）

- **一句话**：欠的一定还，但还的方式经常让人哭笑不得。

## 许老头

- **年龄**：约60余岁

- **身份**：无名小城旧货摊主，驼背

- **性格关键词**：沉默·嘴硬心软·说谎时会磕空烟杆

- **核心矛盾**：他这辈子最大的恐惧不是死，是许若存叫他"爷爷"。叫了就是生分了。所以他宁愿什么都瞒着，让许若存一直叫他"许老头"。这种"为了保护你而不告诉你"的选择，恰恰是许若存最恨的方式。

- **和谁冲突**：和许若存（一个想保护，一个想知道真相；一个话到嘴边咽回去，一个追到门口又停住）、和老余头（年轻时的旧账）

- **关键台词**："后悔过。后悔没早点告诉你一些事情。总想着等哪天你长大了再说。结果等着等着，有些事就变得不好说了。"

## 柳三娘

- **年龄**：约26岁（比许若存大约6岁）

- **身份**：官道"三娘客店"老板娘

- **性格关键词**：话少心热·温柔·善良·不问来路不问归期

- **核心矛盾**：她是唯一一个不需要许若存解释"我是谁"的人。她从来不问你来做什么、你从哪里来、你什么时候走。他来，她下一碗面；他走，她继续擦灶台。她的善良不是软弱的善良——她知道许若存不会留下，但她不因此改变自己对待他的方式。

- **和谁冲突**：和许若存（不是冲突——是一段不需要确认的关系。正因为不需要确认，所以谁都没有开口）

- **关键台词**："你太瘦了。瘦的人跑不远。"

- **关键动作**：他第一次来的时候，她嘴上说没钱住什么店，转身却给他下了一碗面，卧了一个荷包蛋。后来他每次来，她都腾一间屋子、放一盆热水、剥一筐蒜、煮一锅粥。他走的时候，她给他在布袋里装了一袋剥好的蒜。

- **出场章节**：第10章（初登场·一碗面荷包蛋）→第48章（收留十一天）→第57章（瓜子壳算术题）→第61章（走马灯·一碗面五片牛肉）

## 秦望舒

- **年龄**：约26岁（比许若存大约6岁）

- **身份**：洛京郊外纸坊老板娘，手工造纸、修复旧卷

- **性格关键词**：话少心热·手上的植物染料洗不掉·等人但不问

- **核心矛盾**：她是许若存"透气口"的另一端。他不需要解释自己为什么来，她也不需要解释自己为什么等。她的纸坊是一处不必说话就可以待着的地方。她等他来，但从不表现出来——只在窗台上放一壶茶，冬天垫一块蓝棉布。那块蓝布底下的宣纸上写着一个"等"字，但她从来没问过他看见了没有。

- **和谁冲突**：和许若存（一壶茶的距离——他来她不问，他走她不送，但窗台上的蓝布一直叠在那里）

- **关键台词**："冬天凉。"

- **关键动作**：她的窗台上永远叠着一块蓝棉布，底下压着一叠宣纸。她不说那是给他的，但他每次来的时候那个位置都空着。后来纸坊门锁了，窗台上那块蓝布收了，只剩一张叠成四方的纸，上面两个字——"收了。"

- **出场章节**：卷二起穿插"路过"线——第24章（姜糖水）→第31章（洗碗）→第38章（巷口看晒纸）→第44章（冬天蓝棉布）→第61章（走马灯·蓝布"等"字）→卷七收束

## 沈见微

- **年龄**：比许若存大一岁

- **身份**：涤心院弟子，被洗去十一年记忆

- **性格关键词**：表面冷淡·内心空洞·重诺·正在找回自己

- **核心矛盾**：他从小被洗心，已经不知道"自己想要什么"了。他追捕许若存不是因为恨，是任务。他放走许若存不是因为感动，是洗心后遗症让他愣了半秒。他倒戈不是因为正义，是他记起了母亲的脸。他做的每一个选择都不是"选择"，是身体先于大脑行动了。他到死都不知道自己到底是谁——但他知道他想成为谁。

- **和谁冲突**：和许若存（追捕与反追捕）、和裴先生（导师与囚笼）、和自己（"我记得我忘了什么，但我想不起来了"）

- **牺牲方式**：不是被杀死，是站上石门之后身体变成门上的云纹浮雕。每年下雨时云纹会微微发光。

## 纪无咎

- **年龄**：约45岁

- **身份**：承序府收缴司司长，许衡旧日同门

- **性格关键词**：冷峻·执着·信"秩序"胜过一切

- **核心矛盾**：他不是坏人，他是"选错了方法的好人"。他认为天道碎裂的原因是"被问得太多了"——未择之气是仍在提问的力量，只要这股力量还存在天道就永远无法修复。他要抓许若存不是为了杀他，是为了把未择之气抽出来"还给天道"。但他的手段越来越极端，他自己也知道。

- **和谁冲突**：和许衡（被斩断小指）、和许若存（理念对立）、和叶知秋（体制内的灰色地带）

- **他的悲剧**：他最在乎的秩序，最终吞噬了他自己。

## 红姑

- **年龄**：约25岁

- **身份**：说书人，靠"名"修行的散修

- **性格关键词**：台上亮台下冷·怕被遗忘·用故事藏真话

- **核心矛盾**：她怕的不是死，是死了之后没人记得她。所以她拼命说书，拼命让人记住——但她也知道，让人记住的故事都不是真话。真话她藏在花生壳里、藏在门槛上的铜钱里、藏在散场后一个人的沉默里。

- **和谁冲突**：和她自己（想被人记住但不想说谎）、和许若存（她知道的比说的多）

- **关键动作**：散场后一个人坐在后台，摸出铜钱，对着它说话。

---

## 苏缠

- **年龄**：开篇约17岁

- **身份**：幻音楼观测者，红绳感应体质

- **性格关键词**：温柔的疏离者·敏感于边界·内在比外表坚韧

- **核心矛盾**：她和许若存的关系是"分而未分"——像量子纠缠中的两个粒子，无论隔着多远，改变一个的状态另一个会瞬间响应，但你永远无法同时确定两个粒子的确切位置。她靠近他时会灼伤，远离他时会感应到他在危险中。她分不清这到底是红绳的作用还是她自己的意愿——**读者也分不清**。全书不会给出明确的答案。她的感情状态就像薛定谔的猫——在被观测之前，既是爱也不是爱。

- **和谁冲突**：和许若存（接近就灼伤、远离就牵绊——一种天生注定有距离的关系）、和幻音楼（把她当工具）、和自己（不是她分不清纠缠和感情——是这个世界的规则本来就没有"纠缠和感情中间那条线"）

- **关键台词**："我分不清我想靠近你是因为你的频率吸引我，还是因为我自己想靠近你。"

- **设计思路**：她的存在不是一个"她爱不爱他"的问题——是"爱和纠缠是不是同一件事"。读者最终得到的不是答案，是选择权。

## 其他人物

### 小六子

- **性格**：黏人但不烦人·机灵但不精明·把许若存当亲哥

- **冲突**：他想跟着许若存走，但许若存不让。他最后没跟，不代表他不想。

### 阿芷

- **性格**：独立不黏人·记恩·嘴硬心软

- **冲突**：她是唯一一个不欠许若存的人——她把欠他的还清了，然后走了。许若存反而怅然若失。

### 叶知秋

- **性格**：务实·被体制消磨·关键时有担当

- **冲突**：想帮忙但怕丢饭碗。每次帮完许若存回家都要面对老婆的药钱。

### 柳无眠

- **性格**：勤奋到病态·停不下来·一停就会怀疑人生

- **冲突**：他发现家人早就死了但宗门一直骗他——他没有报复，只是退出了。

### 老陈

- **性格**：通透·不甘被遗忘·话多但没人听

- **冲突**：他画了洗心阵又后悔了。他在归寂院里跟自己下棋，等一个人来走他布的那步棋。

### 许衡

- **性格**：沉默·决绝·爱里夹杂了债

- **冲突**：他爱儿子，但他用"把天道碎片塞进儿子身体"的方式爱。他不是伟大的父亲——他是不敢面对儿子，所以在门后待了十六年。

## 卷一冲突高潮结构

> 每8-12章设一个"小高潮"，不追求打斗场面，追求**情绪和决策的峰值**。

| 高潮节点 | 核心冲突 | 关键场面 |
|---------|---------|---------|
| **第7-8章·暗道逃亡** | 许老头让许若存从暗道走，自己留下面对蓝衣人。许若存第一次独自面对未知。 | 暗道里听到追兵的脚步越来越近，他蹲在拐角捂住自己的嘴不让自己出声。 |
| **第14-15章·岁蚀初现** | 第一次看到噬名者，铜钱符号第一次消耗。老陈说"岁蚀在找你"。 | 许若存用铜钱逼退噬名者，铜钱背面的符号变淡了一半。他第一次意识到这东西不是玩具。 |
| **第20-22章·纪无咎追捕** | 纪无咎正面出场，许若存差点被抓。叶知秋救他但代价是欠一个人情。 | 纪无咎的手指被许衡切断的往事揭晓。许若存从叶知秋口中知道了一个他不想知道的真相：他父亲不是英雄。 |
| **第28-30章·隙径遇险** | 走隙径时界壁破损，许若存差点"散掉"。灵犀拉住他在最后关头冲了出来。 | 隙径中他看到镜中人——一个被困在隙径里的旅行者残影。那个人嘴里反复念叨着一个名字，是他自己的名字。 |
| **第35-36章·虚甲兽战** | 第一次集体战斗。柳无眠燃命、周小满拔剑、许若存用铜钱找弱点。每个人都付出了代价。 | 战斗结束后柳无眠坐在地上看自己手腕上新增的疤，数了数还剩多少寿元。 |
| **第44-45章·隔门对话** | 许若存第一次和父亲说话。许衡说"后悔的不是掰断门——是把你留在门外"。许若存没有说原谅。 | 门那边沉默了很久。许若存把额头抵在门板上。他什么都没说。但门那边的人知道他在。 |
| **第53-54章·岁蚀降临** | 有人当着许若存的面被岁蚀吃掉了一部分。不是死，是概念消失——那人的"脸"没了。 | 许若存看到那张空白的脸时吐了。他第一次意识到自己面对的不是一个人、一个宗门——是一种无法理解的东西。 |
| **第63-64章·推门** | 许若存站在石门前做最后的决定。不是推不推的问题——是他发现自己左肩封印彻底裂开之后，可能就回不来了。 | 他闭眼把十息走完。然后推门。门缝里的光照亮他的脸。他不是走进光里——是走进光后面的黑暗里。 |

> **写作原则**：每个高潮必须有至少一个角色做出**不可逆的选择**——选了就回不了头。没有不可逆的选择，就不算高潮。
