## 沈见微

| 属性 | 内容 |
|------|------|
| **年龄** | 约20岁 |
| **身份** | 涤心院弟子/镇魔司头号人物，五岁起洗心三十次 |
| **性格** | 表面冷淡·内心空洞·重诺·正在找回自己 |
| **核心矛盾** | 每个选择都不是选择，是身体先于大脑行动 |

### 洗心的真相

涤心院告诉弟子：洗心是净化。真相是洗心是控制——每半年一次，不是因为你"需要净化"，是因为组织需要你忘记你开始怀疑的东西。

### 岁蚀的反向作用

岁蚀侵蚀的不是他的记忆——是他脑子里那层被洗心反复涂抹上去的"干净"。岁蚀每侵蚀一寸，那个被压制的自己就往外面顶一寸。

### 关键意象

| 意象 | 含义 |
|------|------|
| **手抖** | 身体比大脑先醒来。洗心洗不掉的东西在肌肉记忆里活了下来 |
| **掌纹分叉** | 茧子底下的秘密。一条线分了两条路 |
| **不会做梦→第一个梦** | 洗心封锁了潜意识。梦是防线上的第一个洞 |
| **雨** | 母亲送他进涤心院那天下了大雨。雨声盖住了那四个字 |
