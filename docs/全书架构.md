### 弧2（第21-30章）待写

**弧中心悬念**：洛京是一座知道许若存要来的城。他还没到，每张网已经张开了。

**时间线**：与第20章沈见微的十日倒计时同步。

| 章 | 摘要 | 本章悬念 |
|----|------|---------|
| 21 | 隙径把他扔进洛京一间锁着的屋子。一个人推门进来说等你很久了。 | 谁在等他？隙径出口为什么是锁着的？ |
| 22 | 那人问他封印什么时候开始跳的，留了一本旧册子，封面写别信纪无咎。 | 册子是谁写的？纪无咎说了什么谎？ |
| 23 | 出口被封死。天亮时一只手从天窗外伸下来递了一把钥匙。 | 谁在屋顶上？为什么知道他被困在这里？ |
| 24 | 墙上贴了一张寻人启事。画像上的人眉眼和他有七分像，只写告知序塔守门人。 | 画像上是谁？贴告示的人为什么知道他会来洛京？ |
| 25 | 有人在砖窑顶上蹲了一夜。天亮前放下一块玉——他父亲那半块。 | 谁送来的玉？他父亲为什么把半块玉留在外面？ |
| 26 | 沈见微的十天到了。他做了一个决定。 |
| 27 | 许若存在洛京的地下通道里听到一个消息：纪无咎进城了。不是来抓他的——是来赴一个约。 | 纪无咎赴谁的约？ |
| 28 | 有人把许若存带到一个院子里。院子里有一张石桌，桌上刻着半局棋。落子的人十六年前没下完就走了。 | 那半局棋是谁留下的？ |
| 29 | 那人的声音从屋里传出来，隔着门："你进门之前，有件事你得知道。你爹当年不是被抓进去的。他是自己走进去的。而且他进去之前，留了一句话给你。"他说了那句话。 | 那句话是什么？ |
| 30 | 许若存站在序塔门口。他没有进去——他看见另一个人比他先到了。 | 谁先进了塔？ |

**弧末变化**：有人比许若存先进了塔。
**下弧钩子**：塔底的呼吸声变了——不是一个人。
