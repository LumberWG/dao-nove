# 笔法核心规则

> 适用对象：小说所有章节的写作与修改

---

## 一、篇章结构

### 1. 对话驱动

每一章的主要情节靠对话推进，不是靠描写或叙述。

对话里包含信息、冲突、人物关系。能通过两个人说话交代的事，不用旁白去说。

**不要**：许若存知道许老头今天不对劲，桌上那把菜刀的朝向变了，他心想这老东西肯定有事瞒着他。

**要**：

```

你今天刀口都朝外摆了。

你怕有人今晚进这屋。

你跟谁结了梁子你自己心里清楚。

```

### 2. 无大段背景描写

背景信息揉碎在剧情里一点一点喂给读者，不集中倒。

**不要**：人物的来龙去脉、世界观的设定、前史——这些全挤在一段里说完。

**要**：通过对话、动作、情节推进自然带出。一章最多给三五句背景，不能再多。

### 3. 每章末尾必须有钩子

章尾必须让读者想翻页。不能停在情绪消散的地方。

钩子可以是：

- 一个突然出现的人

- 一句没说完的话

- 一个反常的细节

- 一个悬而未决的决定

**不要**：许若存望着窗外的月光，心里踏实了许多。

**要**：窗外有什么东西轻轻响了一下。他没睁眼。手指攥着那块玉，在黑暗里慢慢收紧了。

## 二、句式与段落

### 4. 一段不超过三句话

写满三句必须换行。一句能说完的事，绝不用两句。

情绪戏可以放宽到四句，节奏戏必须卡死在一到两句。

**要**：

```

许老头伸手在门槛的木头上摸了一下。

那是他每天坐的位置。

木头被磨得光滑发亮，跟包了浆似的。

```

### 5. 对话该有的引号必须要

对话用双引号标注，位置按《用法须知》执行。引号里的话优先保留口语节奏，不加不必要的修饰。

谁在说话靠动作和上下文带出来，不在引号外加"他说""她问"这类标签。

**要**：

```

许若存没动。门夹着那只手。谁都没退。

"你问你的，我听着。手先拿出去。"

门外的人笑了一声，收回去了。

```

### 6. 少用破折号，每章不超过3处

破折号会打断阅读节奏。能用逗号或句号解决的问题，不用破折号。

**数量限制**：每章原则上不使用破折号，特殊需要时不得超过 **3 处**。

**最多使用的错误模式**：「不是X——是Y」纠正性解释。绝大多数破折号都在干同一件事——作者跳出来纠正读者的想象。直接去掉破折号，用逗号或句号断开即可。

**不要**（纠正模式）：他想起很多事——但每件事都不完整——像缺了页的书。

**要**：他想起很多事。但每件事都不完整。像缺了页的书。

**可保留的 3 处额度**优先给：
- 对话中断（"你——"）
- 情绪性停顿
- 节奏性渲染

**判断标准**：写完数一遍。——号个数。超了就去掉最不必要的那几个。

### 7. 少用句号，多用逗号

文中长句不多，不用刻意断句。除非非要句号不可——意思说完了、句子结构完整了——否则优先用逗号连下去。

动作连续的地方用逗号一气呵成，情绪转折的地方也用逗号带过去，不靠句号制造停顿感。

**要**：

他蹲在原地没动，低头解一截打了十几个死结的麻绳，手指稳稳的，心里在算距离。

**不要**：

他蹲在原地没动。低头解一截麻绳。手指很稳。他在算距离。

判断标准：读一遍。有没有不必要的断气感。有，就把句号改成逗号。

## 三、语言风格

### 8. 不写心理描写，写动作

禁止直接说"他心里很难过""他很紧张""他感到恐惧"。情绪全部通过身体动作来传达。

| 不要写 | 要写 |
|--------|------|
| 他感到紧张 | 他把手伸进袖口，摩挲那块碎布 |
| 他很难过 | 他低头磕了两下烟杆，没说话 |
| 他很震惊 | 他的筷子停在半空，顿了两息 |
| 他很害怕 | 他后背的寒毛炸了一下 |
| 他在犹豫 | 他捻佛珠的速度慢了下来 |
| 他很愤怒 | 他嘴角抽了一下，又敲了他一下，这回轻了点 |

### 9. 不直接解释动作和现象

不要替读者"翻译"角色的动作或眼前的现象。展示动作本身，让读者自己领悟含义。

**不要**：他的手指在醒木上按了一下——那是她稳住自己的习惯动作。

**要**：他的手指在醒木上按了一下。

**常见的"解释"模式——全部要删除：**

| 模式 | 示例（不要） | 改为 |
|------|-------------|------|
| **不是X，是Y** | 他不是在歇腿，是在缓气 | 他在缓气 |
| **那是...的动作/习惯** | 那是他紧张时的微表情 | 他嘴角往左偏了一下 |
| **像是在/仿佛在** | 像是心里一块石头落地了 | 他的肩膀松了下来 |
| **他懂了/他知道了** | 他懂了，铜钱在帮他认位置 | 铜钱在他掌心里转了个方向 |
| **意思是/这说明** | 意思是今天场子里有外人 | 门口倒扣着一只空碗 |
| **叙事者总结** | 这老太太嘴硬了一辈子 | 老太太又装生气了，耳朵动了一下 |

**判断标准**：读一遍。如果有一句话是作者跳出来告诉读者"这个动作意味着什么"，删掉它。读者不傻，让读者自己品。

### 10. 主角内心可用现代口语

每章限2-3处。用在情绪最外露的时候——被气到、被逗到、忍不住吐槽的时候。

**要**：

心里骂了一句：你个老不死的别想把我甩了。

**要**：

这鸟比老王还会来事儿。

**要**：

靠。许若存笑了一声，把镜子放下了。

不能整章都是口语腔。它是调味料，不是主菜。每章2-3处，多了就油。

## 四、人物

### 11. 许若存的人物设定

性格核心：**开朗、嘴贫、有脾气**。不是沉默观察者。

- 他遇到事情的第一反应是**说话**，不是沉默

- 他怼许老头，怼红姑，怼青衣人——谁他都敢顶

- 他有脾气，但来得快去得也快，不记仇

- 他的贫嘴是保护色，也是他爱人的方式

**对话示例**：

```

你说看一眼就一眼，我说不给看就不给看。

现在是听你的还是听我的？

```

```

行。我去当我的饭桶了。

```

```

你话题转得跟说书一样。

```

### 12. 感情处理：不说破

感情线（秦望舒、苏缠等）走含蓄路线，不让角色表白或内心承认。

秦望舒——懂他不点破。苏缠——分而未分，让读者自己判断。

判断标准：如果一句话说破了感情，删掉它。

### 13. 人物要有血有肉

每个出场人物——哪怕只出现一场——都必须有一个让人记住的细节。不让任何角色变成"推进剧情的工具人"。

**具体做法**：

- **一个习惯动作**：许老头说谎前磕空烟杆、红姑散场后摸铜钱、小六子用袖子擦鼻涕。动作就是性格。
- **一个矛盾点**：最硬气的人有最软的一块（许若存嘴硬心软），最沉默的人有最想说的一句话（许老头"后悔没早点告诉你"），最冷淡的人有最放不下的一件事（沈见微被洗掉的记忆）。
- **一个属于自己的说话方式**：许若存嘴贫带刺，许老头说半句留半句，红姑用故事藏真话，小六子黏人但不烦人。每个人的台词换给对方说都不像。

**判断标准**：遮住人物名字，读一遍台词。能认出是谁在说话——就对了。认不出来——角色还没立起来。

**不要**：所有人都用同一种腔调说话，所有情绪都用同一种方式表达。

## 五、节奏与结构

### 14. 战斗节奏

- **小高潮**：每8-12章安排一场。可以是一场单挑、一次突围、一个反转。

- **重要战斗**：跨多章写，不在一章内打完。中间穿插其他视角和线索，保持悬念。

不要为打而打。每一场战斗都必须推动剧情或改变人物关系。

### 15. 冲突推动剧情，要有紧迫感

剧情靠矛盾和冲突往前推，不靠叙述和交代。每一章必须有至少一处"必须现在解决，不能等"的冲突。

**冲突的类型**：

- **人际冲突**：许若存怼许老头（价值观碰撞——一个想瞒一个想知道）、跟红姑较劲（信息不对等——一个知道很多一个在猜）、跟青衣人互不退让（立场对立——两条路只能选一条）。
- **内在冲突**：许若存嘴硬心软（嘴上什么都不在乎，实际什么都往心里放）、沈见微洗心后遗症（做选择的人不知道自己在选什么）。
- **与环境的冲突**：能力有代价（用铜钱预见未来会流鼻血）、时间在倒计时（岁蚀在扩散）、整个世界用秩序的名义在吞噬个体。

**紧迫感的要求**：

冲突不能是"等几天再处理也行"的软矛盾。每一场冲突都必须让角色付出代价——选了A就得放弃B，不动就得承受后果。

- **时间压力**：铜钱发热在催、岁蚀在蔓延、追兵在靠近。角色没有"先放着以后再说"的选项。
- **代价可见**：每次用铜钱都流鼻血、柳无眠燃命手腕上添新疤、许老头瞒了十六年最后只能说三个字"你长大了"。冲突必须有看得见的代价，不能只停留在嘴上说"很危险"。
- **选择不可逆**：推门就回不来、留下就追不上、说了就收不回。每个角色的关键选择一旦做出就不能反悔——读者必须感受到"这一步踏出去就回不了头了"。

**不要**：两个角色坐下来长篇大论地"争论"，谁也不动，谁也不付出代价。冲突必须是动作，不是辩论。

### 16. 场景之间不写过渡

上一场和下一场之间只空一行。不加"与此同时""另一边""就在这时"这类连接词。

写完一个场景的最后一句，直接空行，切下一个场景的第一句。

**要**：

```

我说撤。再不撤，咱们的人就全折在这里了。

拓跋禅的追击队伍停在五里外。

不是他不想追了。是前面那片地形不适合再追了。

```

## 附录：三要三不要

**要：**

- 要一段一景，每段只讲一件事

- 要用动词推进画面，少用形容词

- 要对话即剧情，每一句对话都推动情节或塑造人物

**不要：**

- 不要两个画面塞在同一段

- 不要连用两个形容词（"黑暗而寒冷的夜晚"→"冷。也黑。"）

- 不要让角色自己说"我明白了"——让读者自己明白

## 六、排版格式

### 17. 段间空行

段落之间留一个空行分隔。无论同场景还是跨场景，每段结束后空一行再接下一段。

**要**：

```

许若存蹲到自家摊位旁，压低声音。那个人又来了。

许老头正在修一只旧怀表，头也不抬，把发条往回退了一点，重新拧。别看。越看他越来劲。他不是承序府的人，是雇来的寻踪者。

```

**不要**：段落之间不留空行，连续换行接段。

### 18. 对话结尾用正确的标点

疑问句用问号，陈述句用句号。即使对话简短，也要按语气选对标点。

**要**：

```

"那他想要什么？"

"现在呢？"

"替谁？"

"跟你学的，你台上不也这么接。"

```

**不要**：把所有对话结尾统一用句号，忽略疑问语气。

## 七、主题深度

### 19. 剧情影射现实社会

小说的玄幻设定是外壳，内核要扎进当代社会的真实问题。每一卷至少有两条情节线能让人联想到"这不就是现在发生的事吗"。

**可影射的方向**：

- **秩序对个体的碾压**：承序府的"规矩"——表面上是为了稳定，实际上是把活人变成档案里的一行记录。被洗心的妇人连自己孩子的名字都叫不出来，但嘴角挂着笑，"整整齐齐的，像个瓷器"。
- **信息控制与记忆清洗**：涤心院的"洗心"——抽走记忆、抹掉过去、让人忘记自己是谁。一个不会做梦的人还是一个完整的人吗？
- **普通人面对体制的无力感**：老陈画了阵又被改了阵——"找了第一个人，那人说：你确定吗？不确定就别惹事。找了第二个人，那人说：根基太深了，你搬不动他。"叶知秋每次帮许若存都要回家面对老婆的药钱。
- **"为你好"式的控制**：许老头什么都瞒着——"总想着等哪天你长大了再说。结果等着等着，有些事就变得不好说了。"许衡把天道碎片塞进儿子身体——"他是爱他的，但爱里夹杂了债。"
- **被遗忘者的命运**：红姑怕的不是死，是死了之后没人记得她。老陈在归寂院里跟自己下棋等了十六年。孟婆婆喝了十六年湖里的水，走远了骨头会疼。

**怎么做**：

- 不直接议论。不借角色之口喊口号。让读者自己在故事里看出"这不就是XX吗"——而不是作者跳出来说"这是在讽刺XX"。
- 影射要藏在情节里：一个被改了图纸的阵法、一个被洗掉记忆的弟子、一个"为了你好所以瞒着你"的老人。读者自己品出来才有力量。
- 每个正面角色都有缺陷，每个反面角色都有理由。承序府的人不是天生邪恶——他们只是选择了"秩序大于人"。纪无咎不是坏人——是选错了方法的好人。这让影射更有力，因为它不是黑白分明的寓言，是灰色地带里真实的人的选择。

**判断标准**：遮住玄幻设定（灵力、阵法、封印），故事里的人面对的问题——被控制、被遗忘、被代表、被"为你好"——是不是在今天的社会里同样存在。如果答案是"没有"，说明影射没到位。

## 八、写后自查

### 20. 每章写完后必须逐项检查

写完一章不是结束。以下四项必须过一遍，缺一不可。

**一、字数**

对照字数参考表，确认本章字数在目标范围内。情节推进章 4500-5500 字，大战/多线冲突章 5500-7500 字，过渡/铺垫章 4500-5000 字。差太多说明节奏出了问题——要么信息倒得太快，要么场景没展开。

**二、与前面剧情有无矛盾**

逐条核对：人物性格有没有前后不一致（许若存忽然沉默寡言、许老头忽然滔滔不绝）、已揭示的信息有没有被推翻（昨天说铜钱只有一枚，今天又冒出一枚没交代来源）、时间线有没有错位（同一天内发生的事对不上）、人物关系有没有跳级（昨天还是陌生人，今天忽然掏心掏肺）。

**三、对后续情节铺开有何影响**

本章打开的线索，后面必须收。本章关上的门，后面不能再开（除非有意的伏笔回收）。

**开写前**：搜索 `docs/伏笔与线索追踪.md`，找出所有「预期回收章节」= 本章号的条目。写的时候脑子里装着这些待还的债。

**写完后列出**：
- 本章新抛出的线索（一条一条记下来，后面哪一章收——必须给预计章号，不留空）
- 本章回收了前面哪条线索（标出来源章号，去线索文档里把状态从「未回收」改成「已揭示」或「已使用」）
- 本章出现的新人物/新地点/新设定（确认是否与设定文档一致）
- 本章给下一章留的钩子是什么
- 原定本章回收但没回收的线索——注明原因：延后到哪章 / 废弃 / 改方案

以上内容必须同步更新到 `docs/伏笔与线索追踪.md`。每章有独立的表格分区，按"类型/内容/状态/预期回收章节"四列记录。状态分四种：未回收、部分回收、已揭示（本章）、已使用（已发挥作用）。写完不更新线索文档等于白写——下周你自己就忘了埋过什么。

**四、规则自检**

对着规则表过一遍：有没有超过三句的段、有没有心理描写该改动作没改的、有没有破折号可以换成句号的、有没有对话标点写错的、有没有场景过渡写了"与此同时"的。一条一条过，不要凭感觉。

**判断标准**：如果一章写完，说不清它推动了什么、改变了什么、为后面埋了什么——这一章就是废章，要么改，要么删。
