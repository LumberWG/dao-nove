# 笔法核心规则

> 适用对象：小说所有章节的写作与修改

---

## 一、篇章结构

### 1. 对话驱动

每一章的主要情节靠对话推进，不是靠描写或叙述。

对话里包含信息、冲突、人物关系。能通过两个人说话交代的事，不用旁白去说。

**不要**：许若存知道许老头今天不对劲，桌上那把菜刀的朝向变了，他心想这老东西肯定有事瞒着他。

**要**：

```

你今天刀口都朝外摆了。

你怕有人今晚进这屋。

你跟谁结了梁子你自己心里清楚。

```

### 2. 无大段背景描写

背景信息揉碎在剧情里一点一点喂给读者，不集中倒。

**不要**：人物的来龙去脉、世界观的设定、前史——这些全挤在一段里说完。

**要**：通过对话、动作、情节推进自然带出。一章最多给三五句背景，不能再多。

### 3. 每章末尾必须有钩子

章尾必须让读者想翻页。不能停在情绪消散的地方。

钩子可以是：

- 一个突然出现的人

- 一句没说完的话

- 一个反常的细节

- 一个悬而未决的决定

**不要**：许若存望着窗外的月光，心里踏实了许多。

**要**：窗外有什么东西轻轻响了一下。他没睁眼。手指攥着那块玉，在黑暗里慢慢收紧了。

## 二、句式与段落

### 4. 一段不超过三句话

写满三句必须换行。一句能说完的事，绝不用两句。

情绪戏可以放宽到四句，节奏戏必须卡死在一到两句。

**要**：

```

许老头伸手在门槛的木头上摸了一下。

那是他每天坐的位置。

木头被磨得光滑发亮，跟包了浆似的。

```

### 5. 对话该有的引号必须要

对话用双引号标注，位置按《用法须知》执行。引号里的话优先保留口语节奏，不加不必要的修饰。

谁在说话靠动作和上下文带出来，不在引号外加"他说""她问"这类标签。

**要**：

```

许若存没动。门夹着那只手。谁都没退。

"你问你的，我听着。手先拿出去。"

门外的人笑了一声，收回去了。

```

### 6. 少用破折号，每章不超过3处

破折号会打断阅读节奏。能用逗号或句号解决的问题，不用破折号。

**数量限制**：每章原则上不使用破折号，特殊需要时不得超过 **3 处**。

**最多使用的错误模式**：「不是X——是Y」纠正性解释。绝大多数破折号都在干同一件事——作者跳出来纠正读者的想象。直接去掉破折号，用逗号或句号断开即可。

**不要**（纠正模式）：他想起很多事——但每件事都不完整——像缺了页的书。

**要**：他想起很多事。但每件事都不完整。像缺了页的书。

**可保留的 3 处额度**优先给：
- 对话中断（"你——"）
- 情绪性停顿
- 节奏性渲染

**判断标准**：写完数一遍。——号个数。超了就去掉最不必要的那几个。

### 7. 少用句号，多用逗号

文中长句不多，不用刻意断句。除非非要句号不可——意思说完了、句子结构完整了——否则优先用逗号连下去。

动作连续的地方用逗号一气呵成，情绪转折的地方也用逗号带过去，不靠句号制造停顿感。

**要**：

他蹲在原地没动，低头解一截打了十几个死结的麻绳，手指稳稳的，心里在算距离。

**不要**：

他蹲在原地没动。低头解一截麻绳。手指很稳。他在算距离。

判断标准：读一遍。有没有不必要的断气感。有，就把句号改成逗号。

## 三、语言风格

### 8. 不写心理描写，写动作

禁止直接说"他心里很难过""他很紧张""他感到恐惧"。情绪全部通过身体动作来传达。

| 不要写 | 要写 |
|--------|------|
| 他感到紧张 | 他把手伸进袖口，摩挲那块碎布 |
| 他很难过 | 他低头磕了两下烟杆，没说话 |
| 他很震惊 | 他的筷子停在半空，顿了两息 |
| 他很害怕 | 他后背的寒毛炸了一下 |
| 他在犹豫 | 他捻佛珠的速度慢了下来 |
| 他很愤怒 | 他嘴角抽了一下，又敲了他一下，这回轻了点 |

### 9. 不直接解释动作和现象

不要替读者"翻译"角色的动作或眼前的现象。展示动作本身，让读者自己领悟含义。

**不要**：他的手指在醒木上按了一下——那是她稳住自己的习惯动作。

**要**：他的手指在醒木上按了一下。

**常见的"解释"模式——全部要删除：**

| 模式 | 示例（不要） | 改为 |
|------|-------------|------|
| **不是X，是Y** | 他不是在歇腿，是在缓气 | 他在缓气 |
| **那是...的动作/习惯** | 那是他紧张时的微表情 | 他嘴角往左偏了一下 |
| **像是在/仿佛在** | 像是心里一块石头落地了 | 他的肩膀松了下来 |
| **他懂了/他知道了** | 他懂了，铜钱在帮他认位置 | 铜钱在他掌心里转了个方向 |
| **意思是/这说明** | 意思是今天场子里有外人 | 门口倒扣着一只空碗 |
| **叙事者总结** | 这老太太嘴硬了一辈子 | 老太太又装生气了，耳朵动了一下 |

**判断标准**：读一遍。如果有一句话是作者跳出来告诉读者"这个动作意味着什么"，删掉它。读者不傻，让读者自己品。

### 9.5 禁止"不是X，是Y"纠正句式

"不是X，是Y"是全书最常见的水字数模式。每出现一次，作者就跳出来替读者理解了一遍。删掉"不是X"，直接写Y。

**不要**：不是疼。不是痒。是被拨动的感觉。
**要**：一种被拨动的感觉。

**不要**：不是用手腕甩出来的，是用气劲推进刀身。
**要**：气劲推进刀身，脱手之后仍有余力。

**不要**：不是脚步声。是一种气场的变化。
**要**：一种气场的变化，像气压在下降。

**不要**：不是普通灰尘，颜色比灰更深，像是烧过什么东西留下的残渣。
**要**：一小片灰烬，颜色比普通灰尘深得多。烧过什么东西之后留下的残渣。

**判断标准**：写完一章，搜索"不是"二字。每出现一个"不是……是……"结构，把"不是X"删掉。超过3处说明这章在替读者阅读理解。

### 9.6 禁用过渡词——"然后""一会儿""忽然""接着""之后"

这些词是流水账的骨架。每用一次，读者就被提醒一次"有人在给我讲故事"。删掉它们，让动作直接发生。

**不要**：他想了很久，然后站起来，然后走到窗边。忽然外面传来钟声。他站了一会儿，然后推开窗。

**要**：他想了很久，站起来，走到窗边。窗外传来钟声。他推开窗。

| 禁用词 | 问题 | 替代 |
|--------|------|------|
| **然后** | 流水账连接词，拉平所有动作的节奏 | 直接删掉，接下一个动作 |
| **一会儿** | 模糊时间填充词（"想了一会儿""站了一会儿"） | "想了很久" / "站了一阵" / 或直接删掉接动作 |
| **忽然/突然** | 替读者做惊讶预警（"忽然传来钟声"→钟声本身就够突然） | 直接写事件，让读者自己感受到突然 |
| **接着/随后** | 和"然后"同类，更书面化的流水账标记 | 删掉 |
| **之后** | 时间顺序标记（"走出几步之后"） | 改为动作直接承接（"走出几步，脚下一顿"） |

**例外**：对话中的口语化"然后"可以保留（角色说话习惯），叙述中的全部删掉。

**判断标准**：写完一章，搜索"然后""一会儿""忽然""接着""之后"。每超过2处说明这章在拉流水账。

### 10. 主角内心可用现代口语

每章限2-3处。用在情绪最外露的时候——被气到、被逗到、忍不住吐槽的时候。

**要**：

心里骂了一句：你个老不死的别想把我甩了。

**要**：

这鸟比老王还会来事儿。

**要**：

靠。许若存笑了一声，把镜子放下了。

不能整章都是口语腔。它是调味料，不是主菜。每章2-3处，多了就油。

### 11. 写感情要直击内心，写悲情要让人不甘

**写感情——让读者跟着角色一起起伏。**

感情戏不是靠旁白"他很感动""她很伤心"来传递的。是靠**具体的选择和动作**。

- 一个人爱不爱你，不是听他嘴上怎么说，是看他在关键时刻选了谁。柳三娘懂许若存但不点破——一个眼神、一句恰到好处的圆场比十句"我喜欢你"都重。
- 感情升温的节点必须用事件去推。一起经历过生死、一起守过一个秘密、一起在雨夜里走过一条路——这些场景本身就在替角色表白。
- 暧昧期的拉扯要够长够细。一个没碰到的指尖、一句说到一半停住的话、一个转头时来不及收回的眼神——读者比角色更急，那就对了。

**判断标准**：写完感情戏，读一遍。如果自己没有被甜到或者揪到，那读者也不会。

**写悲情——好人和坏人的死都要让读者心有不甘。**

- 好人死了，读者替他不值。"他本不该死"——这才是好的悲情。如果读者觉得"他该死"，那是角色没立住。
- 坏人死了，读者替他可惜。不是同情他的恶，是理解他走到这一步的路。一个有血有肉的反派死了，读者会想"如果当时换一条路就好了"。
- 悲情的秘密不在"死"那一瞬间，在死之前的细节。他手里攥着的东西、他最后看了一眼的方向、他咽气之前没说完的那半句话。细节留给读者，情绪让读者自己填。
- **不要**：大段渲染"他多么悲惨、多么伟大"——那是作者在替读者哭。**要**：只写事实——他倒下之前把刀柄转了方向，刀刃朝自己。

**判断标准**：写完悲情戏，问自己——如果我站在这人的对立面，我会不会也替他难受？如果不会，悲情没到位。

## 四、人物

### 12. 许若存的人物设定

性格核心：**开朗、嘴贫、有脾气**。不是沉默观察者。

- 他遇到事情的第一反应是**说话**，不是沉默

- 他怼许老头，怼红姑，怼青衣人——谁他都敢顶

- 他有脾气，但来得快去得也快，不记仇

- 他的贫嘴是保护色，也是他爱人的方式

**对话示例**：

```

你说看一眼就一眼，我说不给看就不给看。

现在是听你的还是听我的？

```

```

行。我去当我的饭桶了。

```

```

你话题转得跟说书一样。

```

### 13. 感情处理：不说破

感情线（柳三娘、秦望舒、苏缠等）走含蓄路线，不让角色表白或内心承认。

柳三娘——懂他不点破。秦望舒——知道一切但等他来懂（她不是"等他不追问"的被动者——她知道隙径、岁蚀和他的去向，但她选择不说，因为说了他也听不懂。她的沉默不是等待，是保护）。苏缠——分而未分，让读者自己判断。

判断标准：如果一句话说破了感情，删掉它。

### 14. 人物要有血有肉

每个出场人物——哪怕只出现一场——都必须有一个让人记住的细节。不让任何角色变成"推进剧情的工具人"。

**具体做法**：

- **一个习惯动作**：许老头说谎前磕空烟杆、红姑散场后摸铜钱、小六子用袖子擦鼻涕。动作就是性格。
- **一个矛盾点**：最硬气的人有最软的一块（许若存嘴硬心软），最沉默的人有最想说的一句话（许老头"后悔没早点告诉你"），最冷淡的人有最放不下的一件事（沈见微被洗掉的记忆）。
- **一个属于自己的说话方式**：许若存嘴贫带刺，许老头说半句留半句，红姑用故事藏真话，小六子黏人但不烦人。每个人的台词换给对方说都不像。

**判断标准**：遮住人物名字，读一遍台词。能认出是谁在说话——就对了。认不出来——角色还没立起来。

**不要**：所有人都用同一种腔调说话，所有情绪都用同一种方式表达。

### 15. 世界观通过角色反映，不通过角色解释

世界观不应由角色口述给读者，而应通过角色的性格、心理和故事情节自然折射。

**反面**（把设定硬扣在角色头上）：
> "你体内有两道频率——一道未择之气，一道来自门那边的异物。它们在封印里绞在一起……"

角色变成了FAQ界面，台词功能是"帮作者解释设定"，不是"帮角色表达自己"。

**正面**（让读者通过行动感受世界观）：
> 她从废布头里挑出那块沾血的布，用红绳调谐后让许若存自己看——"两种颜色的线在同一根线上绞合。一种是你自己的命脉颜色。另一种——我认不出来。"

读者通过看**她做什么**来理解设定（她读布、她调绳、她指出纹理异常），而不是听她解释设定。

**核心原则**：
- 世界观应像沉在杯底的糖——喝的时候能感觉到甜，但看不到糖粒
- 角色的困境应自然折射世界的规则——许若存嘴贫是因为他用玩笑掩盖在乎，这折射了"未择者不被允许有确定选择"的世界
- 作者的观点应藏在情节缝隙里——柳三娘不问来路不问归期地煮面，比任何一段"关于善良的论述"都更有力

**判断标准**：写完一句台词后问——"这句话是在帮角色表达自己，还是在帮作者解释设定？"如果是后者，删掉重写。

这同时也是"感情处理：不说破"规则的延伸——不仅是感情，世界观也一样不说破。

### 16. 情节驱动——人物不可预判，故事靠演不靠讲

没有情节，读者不会感兴趣。

**人物不可预判：** 角色一出场，不能让读者立刻判断他的立场、势力、是好是坏。人物的阵营和动机应通过他的**选择和行动**逐步揭晓，而不是靠出场时的标签。

- 反面：一个灰袍人上来就说"我是承序府的，来抓你"——读者瞬间知道他是谁、来干嘛。
- 正面：一个挑染料的姑娘在街上捡了块带血的废布头看了很久——读者不知道她是谁、什么立场、是敌是友。

**反转不是刻意为之：** 反转要自然，但不能让读者在三页内预判结局。读者应该猜得到一部分、猜不到全部——猜对了有成就感，猜错了有惊喜。

**一切靠情节推进：** 角色的动机、世界的规则、势力的立场——这些信息不靠台词直接说出来，而是通过角色的行动让读者"看到"。
- "我想去废界开布料铺" → 她在攒钱、挑染料、找铺面、跟幻音楼对抗。读者通过这些行动理解她的愿望，不需要她解释"为什么想开布料铺"。
- "许若存嘴硬心软" → 他说"关我什么事"但转身去帮忙了。

**把读者带入故事：** 每个场景都要有"接下来会怎样"的悬念。意料之中的发展不如不写。读者不是被动接受信息的人——他是跟着角色一起发现世界的同行者。

**判断标准**：遮住角色出场的头三页，问自己——"读者现在能判断这个人是好是坏吗？"如果答案是"能"，说明出场方式需要重写。

## 五、节奏与结构

### 17. 战斗要火爆，全书有爽文格调

战斗场景是整个故事的肾上腺素。打得不够爽，读者翻页的欲望就断了。

**战斗三原则：**

- **打得要爆**——每一场战斗必须有明确的**胜负手**，有让人屏住呼吸的瞬间。力量对撞要有破坏力（地面裂开、墙壁崩塌、空气震荡），速度和力量感要通过具体描写传达，不是"他很快""他很强"这种空话。
- **节奏要密**——战斗中的对话不超过三句，思考不超过两行。战场上没有人站着不动聊天的。瞬息之间的判断用动作带出来，不是内心独白。
- **打完要有变化**——打赢了要有奖励（修为突破/获得关键信息/人物关系推进），打输了要有代价（受伤/失去/追兵逼近）。不能让读者觉得"这架白打了"。

**全书爽文格调：**

- **情绪先于逻辑**——读者看到这里想拍桌子叫好，那逻辑上略粗糙一点也可以接受。反之，逻辑再严密，读者看了没感觉，那就是败笔。
- **不让读者憋屈超过三章**——主角被压制、被误解、被追着跑，可以，但最多三章必须给一次反击或扬眉吐气的机会。憋久了读者会弃书。
- **每一次付出都有回报**——主角的努力、准备、隐忍，必须在后续剧情中兑现成看得见的回报。不是顿悟，是前面埋的因在这一刻结果了。读者要的是"他就该赢"的畅快感。
- **不给读者喂刀片还不给糖**——可以虐，但不能一直虐。虐完要给一点甜头（友情升温、误会解开、实力突破），让读者缓一口气再继续。

**小高潮**：每8-12章安排一场。可以是一场单挑、一次突围、一个反转。

**重要战斗**：跨多章写，不在一章内打完。中间穿插其他视角和线索，保持悬念。

**不要**为打而打。每一场战斗都必须推动剧情或改变人物关系。

**招式/法宝/阵法的多样性：**

手头有的东西要用上，不要每场战斗都千篇一律。

- **阵法**不是摆设。伏击战用困阵、追击战用疾行阵、防御战用壁垒阵。不同战局用不同阵法，读者才会觉得主角是真的在动脑子，不是在等剧情安排。
- **招式**随着对手变。打皮厚的用穿透技、打敏捷的用范围压制、打群架用控场技。主角的招数库要够大，不能"一招鲜吃遍天"。
- **法宝**在不同场景有不同用法。铜钱能探路也能偏转攻击、玉佩能认主也能感应灵力波动。每一件法宝至少开发三种以上用法，读者才会觉得这东西是真的"宝"。
- **判断标准**：遮住章节名，读战斗场景。如果看不出这一场和上一场有什么区别——招式、阵法、法宝的用法都一样——那就说明重复了。

**主角不是每次都能赢：**

- 输过的对手再赢才有爽感。如果主角一路碾压，读者很快就会麻木。
- 输了要有代价。不能"虽败犹荣"——输了就是输了，断条胳膊、丢件法宝、死个帮手，读者才会替主角心疼，才会期待下一场赢回来。
- **判断标准**：这一场如果主角赢了，问自己——他赢得艰难吗？如果赢得太轻松，删掉这场战斗，或者让对手更强。

### 16. 冲突推动剧情，要有紧迫感

剧情靠矛盾和冲突往前推，不靠叙述和交代。每一章必须有至少一处"必须现在解决，不能等"的冲突。

**冲突的类型**：

- **人际冲突**：许若存怼许老头（价值观碰撞——一个想瞒一个想知道）、跟红姑较劲（信息不对等——一个知道很多一个在猜）、跟青衣人互不退让（立场对立——两条路只能选一条）。
- **内在冲突**：许若存嘴硬心软（嘴上什么都不在乎，实际什么都往心里放）、沈见微洗心后遗症（做选择的人不知道自己在选什么）。
- **与环境的冲突**：能力有代价（用铜钱预见未来会流鼻血）、时间在倒计时（岁蚀在扩散）、整个世界用秩序的名义在吞噬个体。

**紧迫感的要求**：

冲突不能是"等几天再处理也行"的软矛盾。每一场冲突都必须让角色付出代价——选了A就得放弃B，不动就得承受后果。

- **时间压力**：铜钱发热在催、岁蚀在蔓延、追兵在靠近。角色没有"先放着以后再说"的选项。
- **代价可见**：每次用铜钱都流鼻血、柳无眠燃命手腕上添新疤、许老头瞒了十六年最后只能说三个字"你长大了"。冲突必须有看得见的代价，不能只停留在嘴上说"很危险"。
- **选择不可逆**：推门就回不来、留下就追不上、说了就收不回。每个角色的关键选择一旦做出就不能反悔——读者必须感受到"这一步踏出去就回不了头了"。

**不要**：两个角色坐下来长篇大论地"争论"，谁也不动，谁也不付出代价。冲突必须是动作，不是辩论。

### 17. 场景之间不写过渡

上一场和下一场之间只空一行。不加"与此同时""另一边""就在这时"这类连接词。

写完一个场景的最后一句，直接空行，切下一个场景的第一句。

**要**：

```

我说撤。再不撤，咱们的人就全折在这里了。

拓跋禅的追击队伍停在五里外。

不是他不想追了。是前面那片地形不适合再追了。

```

### 18. 每章第一句必须入戏

不写"许若存从XX出发""天亮之后他沿着XX走"这类过渡型开头。第一章怎么写开头，第五十章也怎么写——直接扔进场景里。

**不要**：从落霞镇出来之后，许若存沿着铜钱指的方向往东走了一炷香的工夫。脚下的路面越来越低，两侧的土壁渐渐高过头顶……

**要**：鞋底磨穿的那一刻他听到了自己的脚底板踩在鹅卵石上的声音。

**能用的开头**：一个动作、一个声音、一个触感、一句对话、一个反常的画面。**不能用的开头**：地理课、时间表、行程汇报。

### 19. 节奏变速——高潮用短句，过渡用长句

全章一个语速 = 读者犯困。不同场景必须换挡：

| 场景类型 | 句长 | 段落 | 效果 |
|---------|------|------|------|
| 动作/逃跑/战斗 | 短句为主，偶尔一句成段 | 1-2句 | 呼吸急促，心跳加速 |
| 对话/互动 | 对话短，穿插动作 | 随对话切 | 节奏轻快 |
| 安静/独处/情绪 | 句子稍长，感官细节 | 2-3句 | 让读者喘口气 |
| 悬疑/紧张 | 短句→越来越短→一句一段 | 1句 | 屏住呼吸 |

**具体操作**：
- 许若存逃跑——"后窗推开。底下一条窄巷。月光白惨惨地照在青石板上。他翻出窗户，踩着屋檐落地。脚心被碎瓦硌了一下。没吭声。弓着腰贴着墙根。走。"
- 吃面/休息——"他吃第一口的时候烫了舌头，但他没停。面条筋道弹牙，汤是用骨头熬的，不浓，但有一股很干净的鲜味。荷包蛋咬开之后蛋黄微微流出来，混在面汤里把汤色染得更浓了些。"

**不要**：逃跑戏还用长句连成一整段，读者感受不到任何紧迫感。

### 20. 砍掉"内心总结"——用感官细节替代

每章结尾禁止许若存独自行走+反思"今天学到了什么"。禁止在中间段落插入"他把今天的事过了一遍"式的回顾蒙太奇。

这种段落有三个特征，全都要删：
1. **回顾句式**："从XX出来之后他做了A、B、C"
2. **分析他人动机**："她不止知道XX，她还知道自己要XX"
3. **主题升华**："他在网的正中间，XX替他在网边上掰开了一根线"

**替代方案**：

| 旧（内心总结） | 新（感官/动作） |
|--------------|--------------|
| 他在脑子里把今天的事过了一遍。从归寂院出来之后沿着干河床跑，老陈给他的地图册上的墨线遇光就消失了…… | 他蹲下来打开包袱翻干粮，摸到一个油纸包。打开一看，两块干饼，中间夹着咸菜。纸包底下还压着几枚铜钱。 |
| 她不止知道那个男人来干什么，她还知道自己要扮演什么角色。她什么都没问过他，但她替他挡了一次。 | 他把铜钱攥在手心里，攥了一会儿。月光把铜钱照得发亮。 |

**一句话标准**：读者看完这段之后，是自己品出来的感动——还是作者替他感动完了塞给他的？前一种留着，后一种删掉。

### 21. 对话要有信息差，不写寒暄

两个角色坐下来对话之前，先想清楚：**谁说出来的话对自己有利，谁说出来的话对自己有风险**。如果一段对话里双方都在说客观事实、没有任何一方在冒风险——那就是寒暄，删掉。

**删掉**：客店经营成本、今天走了多远、这道菜怎么做的、寺庙为什么不收你了。

**保留**："怕什么。鬼我不怕。人嘛，来住店的都给钱，不给钱的吃完面就走了。"——这句话有刺，有态度，有人物。

## 十、第10章起执行标准

> 第1-9章不动。从第10章开始，以下标准每条必过。

### 22. 玄幻感——每章至少三处能力/法宝/修为在实战中激活

不是设定里写了但正文里不用。每个能力必须在危机中发挥具体作用。

**第10章示范**：感气探追兵 + 未择之身选逃生路线 + 铜钱挡气感扫描。三个能力在同一场危机中联动。

**第11章示范**：气劲飞刀（修为） + 未择之身预判围堵 + 铜钱震波反弹丝线（法宝对撞）。三个不同的组合。

**判断标准**：遮住章节号，读一遍。如果整章没有任何修行元素在实战中激活——不合格。

### 23. 追兵/战斗场面——敌人必须有修行质感

不写"几个拿刀的灰衣人"。每个追兵都必须有修行者特征。

| 要素 | 写法 |
|------|------|
| 武器 | 刀身上有气劲灌注的共鸣纹，用的时间越长纹路越深 |
| 身体 | 骨骼肌肉被气反复淬炼过，骨重而步稳，踩楼梯板会往下沉 |
| 招式 | 拳头砸水不是溅水花，是气劲把水面压出碗口大凹陷 |
| 感知 | 散出气感扫描房间，像冰水从木板缝隙渗进来 |
| 配置 | 远程封路 + 近战收割 + 追踪锁定，专业猎杀组 |

**判断标准**：追兵能不能跟普通捕快互换？能——不合格。

### 24. 法宝必须有消耗和代价

铜钱不是无限使用的万能道具。每次使用都在消耗它攒的温度/能量。

- 第10章：挡气感扫描用掉大半热度
- 第11章：水底反弹丝线耗光所有热量 → 彻底凉透 → 短时间内回不来

**规则**：法宝每使用一次，必须写清楚消耗了什么。不能"铜钱又在发烫帮他"——那叫作弊。

### 25. 前后章追兵不重复

相邻两章不能使用同一类型的威胁。

| 第10章 | 第11章 |
|--------|--------|
| 镇魔司·制度压迫 | 雇佣猎杀组·野路子 |
| 封闭空间·心理恐怖 | 开阔地形·动作追杀 |
| 有柳三娘暗桩 | 完全solo |
| 铜钱尚温·能挡 | 铜钱凉透·耗竭 |

**规则**：如果连续两章的追兵/冲突类型相同——后一章重写。

### 26. 角色首次出场必须有贯穿全书的重量

不是写成"好心路人"。第10章柳三娘示范：

- **不靠秘密**：她不需要"知道许若存是谁"——秦望舒才是藏秘密那个人
- **靠本能**："见过太多从河床里爬出来的人。多他一个不多。但多看了他一眼。"
- **靠动作**：叠碎布的手法、搬铁锅用腰力、追兵来时不慌——每个动作都在说"这个人不简单"
- **铜钱归属**：她碰过铜钱但不问——不是不认识，是不需要问

**柳三娘 ≠ 秦望舒**：

| 柳三娘 | 秦望舒 |
|--------|--------|
| 不需要知道。直接接纳。 | 知道一切。不能说。 |
| 一碗面、荷包蛋、干净衣服 | 修旧纸、姜糖水、蓝棉布 |
| 身体层面——饿了累了冷了 | 记忆层面——知道什么忘了什么 |

### 27. 数据库同步

每次修改章节后必须执行：

```bash
cd server && node sync_chapter.js <章节号>
```

提交时包含 `server/writer.db`。不遗漏。

**要：**

- 要一段一景，每段只讲一件事

- 要用动词推进画面，少用形容词

- 要对话即剧情，每一句对话都推动情节或塑造人物

**不要：**

- 不要两个画面塞在同一段

- 不要连用两个形容词（"黑暗而寒冷的夜晚"→"冷。也黑。"）

- 不要让角色自己说"我明白了"——让读者自己明白

## 六、排版格式

### 18. 段间空行

段落之间留一个空行分隔。无论同场景还是跨场景，每段结束后空一行再接下一段。

**要**：

```

许若存蹲到自家摊位旁，压低声音。那个人又来了。

许老头正在修一只旧怀表，头也不抬，把发条往回退了一点，重新拧。别看。越看他越来劲。他不是承序府的人，是雇来的寻踪者。

```

**不要**：段落之间不留空行，连续换行接段。

### 19. 对话结尾用正确的标点

疑问句用问号，陈述句用句号。即使对话简短，也要按语气选对标点。

**要**：

```

"那他想要什么？"

"现在呢？"

"替谁？"

"跟你学的，你台上不也这么接。"

```

**不要**：把所有对话结尾统一用句号，忽略疑问语气。

## 七、主题深度

### 20. 不讲大道理，观点融入故事。官场/宗派切合社会现实

小说的玄幻设定是外壳，内核要扎进当代社会的真实问题。每一卷至少有两条情节线能让人联想到"这不就是现在发生的事吗"。

**核心铁律：不讲大道理。**
- 不借角色之口喊口号。不作价值判断。不让人物站成一排发表对社会的看法。所有观点藏在情节和选择里——读者自己品出来才有力量，作者直接说出来就是说教。
- 判断标准：遮住这一段，问自己"少了它剧情受影响吗？"如果答案是"不影响，但道理讲得很对"——删掉它。

**可影射的方向**：

- **秩序对个体的碾压**：承序府的"规矩"——表面上是为了稳定，实际上是把活人变成档案里的一行记录。被洗心的妇人连自己孩子的名字都叫不出来，但嘴角挂着笑，"整整齐齐的，像个瓷器"。
- **信息控制与记忆清洗**：涤心院的"洗心"——抽走记忆、抹掉过去、让人忘记自己是谁。一个不会做梦的人还是一个完整的人吗？
- **普通人面对体制的无力感**：老陈画了阵又被改了阵——"找了第一个人，那人说：你确定吗？不确定就别惹事。找了第二个人，那人说：根基太深了，你搬不动他。"叶知秋每次帮许若存都要回家面对老婆的药钱。
- **"为你好"式的控制**：许老头什么都瞒着——"总想着等哪天你长大了再说。结果等着等着，有些事就变得不好说了。"许衡把天道碎片塞进儿子身体——"他是爱他的，但爱里夹杂了债。"
- **被遗忘者的命运**：红姑怕的不是死，是死了之后没人记得她。老陈在归寂院里跟自己下棋等了十六年。孟婆婆喝了十六年湖里的水，走远了骨头会疼。
- **官场/宗派内幕**：承序府内部的派系倾轧、任务摊派、甩锅文化——基层累死累活，高层动动嘴皮子。谁的关系硬谁升得快，谁没背景谁背锅。陈静安画了一辈子阵图，功劳归上面，出事了自己扛。叶知秋一个狱卒，上司一句话他就得去送死。方主簿坐了二十年冷板凳，就是因为站错了队。这些不需要额外解释，放在剧情里自然就是讽刺。

**怎么做**：

- 不直接议论。不借角色之口喊口号。让读者自己在故事里看出"这不就是XX吗"——而不是作者跳出来说"这是在讽刺XX"。
- 影射要藏在情节里：一个被改了图纸的阵法、一个被洗掉记忆的弟子、一个"为了你好所以瞒着你"的老人。读者自己品出来才有力量。
- 每个正面角色都有缺陷，每个反面角色都有理由。承序府的人不是天生邪恶——他们只是选择了"秩序大于人"。纪无咎不是坏人——是选错了方法的好人。这让影射更有力，因为它不是黑白分明的寓言，是灰色地带里真实的人的选择。
- **官场/宗派写实的核心**：不要写"腐败"这个词，写具体的事——一笔说不清的账、一个不该出现的人、一句没头没尾的交代。读者自己看出"这不就是XX单位吗"，比你说一万句都管用。

**判断标准**：遮住玄幻设定（灵力、阵法、封印），故事里的人面对的问题——被控制、被遗忘、被代表、被"为你好"——是不是在今天的社会里同样存在。如果答案是"没有"，说明影射没到位。

## 八、写后自查

### 21. 每章写完后必须逐项检查

写完一章不是结束。以下四项必须过一遍，缺一不可。

**一、字数**

对照字数参考表，确认本章字数在目标范围内。情节推进章 4500-5500 字，大战/多线冲突章 5500-7500 字，过渡/铺垫章 4500-5000 字。差太多说明节奏出了问题——要么信息倒得太快，要么场景没展开。

**二、与前面剧情有无矛盾**

逐条核对：人物性格有没有前后不一致（许若存忽然沉默寡言、许老头忽然滔滔不绝）、已揭示的信息有没有被推翻（昨天说铜钱只有一枚，今天又冒出一枚没交代来源）、时间线有没有错位（同一天内发生的事对不上）、人物关系有没有跳级（昨天还是陌生人，今天忽然掏心掏肺）。

**三、对后续情节铺开有何影响**

本章打开的线索，后面必须收。本章关上的门，后面不能再开（除非有意的伏笔回收）。

**开写前**：搜索 `docs/伏笔与线索追踪.md`，找出所有「预期回收章节」= 本章号的条目。写的时候脑子里装着这些待还的债。

**写完后列出**：
- 本章新抛出的线索（一条一条记下来，后面哪一章收——必须给预计章号，不留空）
- 本章回收了前面哪条线索（标出来源章号，去线索文档里把状态从「未回收」改成「已揭示」或「已使用」）
- 本章出现的新人物/新地点/新设定（确认是否与设定文档一致）
- 本章给下一章留的钩子是什么
- 原定本章回收但没回收的线索——注明原因：延后到哪章 / 废弃 / 改方案

以上内容必须同步更新到 `docs/伏笔与线索追踪.md`。每章有独立的表格分区，按"类型/内容/状态/预期回收章节"四列记录。状态分四种：未回收、部分回收、已揭示（本章）、已使用（已发挥作用）。写完不更新线索文档等于白写——下周你自己就忘了埋过什么。

**四、规则自检**

对着规则表过一遍：有没有超过三句的段、有没有心理描写该改动作没改的、有没有破折号超标（>3处）、有没有作者跳出来解释动作/现象的句子、有没有讲大道理喊口号的段落、感情戏够不够直击内心、战斗场面够不够火爆（读者看了想不想拍桌子，不想说明没到位）、有没有对话标点写错的、有没有场景过渡写了"与此同时"的。一条一条过，不要凭感觉。

**新增检查项**——主角信息边界：本章出现的新势力/新人物/新术语，主角信息来源是否写清楚了？如果答不上来"他是怎么知道的"，删掉或补充来源。

**新增检查项**——开头入戏：本章第一句是否直接扔进场景里？有没有出现"从XX出发""沿着XX走了一炷香"的过渡型开头？

**新增检查项**——节奏变速：本章的动作/悬疑场景是否用了短句加速？安静/情绪场景是否缓下来了？全章是否一个语速到底？

**新增检查项**——"不是X是Y"句式：搜索"不是"二字。超过3处"不是……是……"结构说明这章在替读者阅读理解。删掉"不是X"，直接写Y。

**新增检查项**——过渡词禁用：搜索"然后""一会儿""忽然""接着""之后"。每超过2处说明这章在拉流水账。删掉，让动作直接发生。对话中的口语化"然后"可保留。

**新增检查项**——内心总结：本章结尾/中间是否有"把今天的事过了一遍"式的回顾蒙太奇？是否有主角分析他人动机的段落？有就删，替换为感官细节或动作。

**新增检查项**——对话信息差：本章对话中，每一段谁在冒风险、谁在说真话、谁在试探？如果一段对话没有任何一方冒风险——删掉。

**新增检查项**——曲折反转：本章如果涉及势力首次展开或人物立场揭示，有没有留一层"后面可以反转"的空间？主角有没有连续两次猜对？如果有，加一次误判。

**新增检查项**——时序逻辑：主角在本章中提到的每一处信息（地名、人名、事件），确认它在前文是否已经出现过——如果是本章首次出现，主角不应该在对话或内心独白中提前引用。特别注意内心独白段落，最容易出现"主角在脑子里提前知道还没人告诉他的事"。

**新增检查项**——他人知情权：本章是否有角色表现出知道主角有玉佩/铜钱等关键物品？如果有，确认该角色有明确的信息来源。特别警惕"连续相遇"（3章内多个路人皆知）和"信息跳级"（从A跳到B没交代过渡）。

**判断标准**：如果一章写完，说不清它推动了什么、改变了什么、为后面埋了什么——这一章就是废章，要么改，要么删。

## 九、叙事视角

### 22. 主角的信息边界——他不知道的就是不知道

**核心原则**：不能因为作者知道某个设定，主角就知道。

许若存从一个无名小城的旧货市场走出来，他对这个世界的认识应该是一层一层剥开的，不是一开始就打开的。每一个势力、每一个人物、每一个术语，他第一次接触到的时候，都必须有"发现"的过程。

**具体规则：**

1. **叙述者不代主角认人。** 不写"不周山庄的人"，写主角看到令牌上的字、听到路人的议论、通过观察和已有知识推测出来。

| 不要写 | 要写 |
|--------|------|
| 不周山庄的人。 | 那人腰上挂着一块铁牌，上面刻着一个"周"字。许若存在路上听人说过，这一带挂这个字的，是不周山庄的人。 |
| 幻音楼的人。 | 她袖口露出一截红线。许若存没见过这种装束，但地底下一个人蹲这么久的，他只能猜是幻音楼的观测者。 |
| 涤心院的地界。 | 翻过那座矮山就是涤心院的地界——昨晚歇脚的茶摊老板说的。 |

2. **主角可以猜错。** 他第一次接触某个势力时，可以片面理解、可以带有偏见、可以把A当成B。后续通过亲身经历修正认知。**猜错了比猜对了有戏。**

3. **知识必须有来源。** 主角知道的每一个势力名称、人物背景、规则细节，都必须有明确的获取渠道——"许老头说过"、"路上听人聊天提到的"、"上次在那本旧书里翻到的"。如果写不出来源，说明主角不该知道这个信息。

4. **信息分三档递进：**

| 档次 | 阶段 | 写法 |
|------|------|------|
| 听过名字 | 初次接触 | "承序府、涤心院这些名字他从小听到大，但从来只是名字" |
| 知道大概 | 接触后 | 通过市井闲聊、路边消息获得片面印象（可能有误） |
| 真正了解 | 亲历后 | 亲身接触过该势力的人/事，形成自己的判断 |

**什么时候可以例外？**

- **从主角的视角看出去时**，叙述语言可以带出他已知的信息——比如他到青石集之前已经听过不周山庄的名号，这时用"不周山庄"来指代是合理的，但必须在前文交代过他听过这个名号。
- **从其他人物的视角看出去时**，不受此限——苏缠的章节开头可以用"幻音楼"，因为她就是幻音楼的人。但切换到许若存视角时必须遵守。

**特别子规则——时序逻辑**：主角的内心独白和对话中提到的地名、人名、事件，必须在**前文已经实际出现过**，不能提前引用还没发生的信息。

**特别子规则——他人的知情权**：其他角色知道主角有玉佩、铜钱等关键物品时，必须有明确的信息来源——亲眼看到、有人告诉、通过组织网络得知。不能因为"这个角色应该知道"就让他知道。如果写不出他的信息渠道，就换一个人来问，或者让这个角色表现出"他知道但不全知道"的试探状态。

最容易出问题的两种情形：
- **连续相遇**：短时间内（3章内）连续出现多个"知道主角有东西"的角色，读者会感觉整个世界都在盯着他。解决办法：分散密度，或让后一个角色表现出"他只听说过但不确认"的不确定状态。
- **信息跳级**：角色从A信息（铜钱）直接跳到B信息（玉佩），中间没有交代信息链的传递路径。解决办法：要么让角色明确说"我听说了铜钱的事，那玉是不是也在你手上"（猜测+试探），要么让他承认"我知道的不多，但我该知道的都知道了"（有组织背景）。

**判断标准**：遮住本章正在和主角对话的角色名字，问自己——"他是怎么知道主角有这东西的？"如果答不上来，加一句交代或删掉他的知情。

最容易出问题的地方：
- **内心独白**：许若存在脑子里想"这个人说的是青石集"——但对方要过三十行之后才说这个词。脑子里不能提前知道。
- **对话回应**：对方还没说完的事，主角不能提前在下一句接上细节。
- **叙述者代述**：叙述者不能提前用"那地方"来指代一个还没被命名的地点。

**检查方法**：写完一章后，逐句看主角的内心独白和对话。对于每一处地名、人名、事件关键词，往前检索它在当前章节中是第几次出现。如果主角提到它的顺序早于它被实际引入的顺序——就是错误。

**判断标准**：遮住本章新出现的势力名称或人物身份，问自己——"许若存是怎么知道的？"如果答不上来，那他就是不应该知道。

### 23. 故事走向不能一眼看到底——曲折与反转

**核心原则**：不能因为作者知道结局，主角就知道——也不能因为作者知道结局，读者在开头就猜到了。

**① 许若存的能力不给他大局观**
- 未择之身只能看到"短期分岔"（一次战斗、一次选择的两种可能），看不到"长线结局"（一个月后、一年后、书末的事）。
- 他能看到的未来——最多几炷香。他连自己明天会不会死都看不到。
- 他可以用能力帮别人选，但他选完之后那个选择会引发什么连锁反应，他不知道。
- **具体操作**：每次使用未择之身，写"他看到了两个可能"但不要写"他知道哪个是对的"。有时候看起来好的那个选择，走下去才发现是陷阱。有时候他选对了，但代价比他想象的严重得多。
- 详见 `docs/功法与战斗体系.md` 中的"未择之身·核心限制"——五条铁律，一条不能破。

**② 主角的认知不可靠——他经常理解错**
- 许若存对世界的认识是片面的、有偏的、经常出错的。他听来的消息可能假、他观察到的现象可能误导、他自己判断出来的结论可能全是错的。
- **写作方法**：不让许若存"推理出正确答案"。他推理出一个答案，然后下一章让他发现这个答案不对——或者对了一半，另一半是个更大的坑。
- 判断标准：如果一章里主角连续两次猜对了，这一章就需要增加一次误判。

**③ 世界观的信息要挤牙膏式释放**
- 不在一章里倒完一个势力的全貌。第一次出场只给一个特征（"挂周字铁牌的人不好惹"），第二次出场给第二个特征（"不周山庄的人喜欢拆墙"），第三次出场才给核心理念（"完整是假象，缺口才是入口"）。
- 读者对每个势力的理解也跟着许若存一起，从片面到全面，中间经历多次修正。

**④ 重要转折必须反转两次以上**
- 每个核心角色的立场至少反转一次。最亲近的人可能有隐藏目的，最像反派的人可能有自己的正义。
- 每个重大事件至少有两层解读。表面看是A在追杀许若存，第三卷后才发现A追杀他的原因和许若存自己想的不一样。
- 判断标准：如果读者能在一卷之内猜到某个"重大秘密"，说明这个秘密埋得不够深。

**⑤ 结局的反转**
- 全书结局不能是"主角选择了拯救世界/毁灭世界"这种在一开始就能猜到的二选一。
- 许若存的最终选择必须出乎意料——但回过头看又合情合理。读者看到结局时的反应应该是"我没想到他会这么做"紧接着"但仔细一想他只能这么做"。
- 具体的结局设计方向见 `docs/全书新纲要.md` 中卷六和卷七的"反转候补"。

**写作铁律**：如果自己写完之后都觉得"这章没什么惊喜，就按大纲走的"——说明这一章不够好。改到连自己都想不到下一步会这样发展。

## 叙事结构铁律（第1-19章经验教训）

> 以下规则基于1-19章实际写作中暴露的问题——情节单一、过早暴露设定、冲突密度低。后续章节严格执行。

### 1. 人物不定性、不定型

不提前给角色贴标签。立场、阵营、动机通过选择和行动逐步展露，让读者猜不透。

**不要**：灰袍人上来就说"我是承序府的，来抓你"——读者瞬间知道他是谁、来干嘛。
**要**：一个人穿着灰袍在茶馆坐了很久，只喝茶不看书，手指在桌沿上敲了三下。另一桌上的人收了摊走了。读者不知道他是谁、什么立场、是敌是友。

**判断标准**：写一个角色的出场头三页，读者能判断他是好是坏吗？能——不合格，重写。

### 2. 大纲是活的不死的

如果写作中发现更精彩的方向，改大纲而不是硬塞。大纲是导航，不是铁轨。

**反面**：定了第X章要揭示Y信息，写到那章发现时机不合适，但还是硬塞进去——结果大白话把悬念毁了。
**正面**：拖延到更好的章节，或者改成借他人之口侧面透露。信任自己的直觉——写着写着觉得"现在说太早了"，那就是太早了。

### 3. 信息释放的"撒盐"原则

设定像盐——撒太少没味，撒太多齁人，一次性倒一整袋叫犯罪。

同一份信息（封印真相、洗心阵设计、母亲推门等）切碎成3-5块：

| 块 | 位置 | 谁说的 | 泄露多少 |
|----|------|--------|---------|
| 第1块 | 第X章 | 角色A | 只给一个异常现象，不给解释 |
| 第2块 | 隔3-5章 | 角色B | 给一半解释，留一半 |
| 第3块 | 再隔几章 | 事件/自证 | 再深入一层 |
| 第4块 | 高潮章 | 对比/反转 | 完整揭示但留一层反转余地 |

**参照**：不周山庄的理念在第16章谢渊口中露了第一层（"残缺不是脏"），下一层留到卷二谢渊再出场时才给。

**判断标准**：写一章时问——这一章交代的信息，有没有哪怕1/3可以延后到5章后再揭？如果有——延后。

### 4. 多线交织

不要全程跟主角。适时切到其他角色POV：
- **沈见微**：洗心弟子的挣扎，他的解密过程独立于许若存
- **叶知秋**：承序府内部运作，找出许衡档案的真相
- **纪无咎**：从自己的视角看"追捕"，让读者知道他为什么追
- **红姑**：说书场后台，隐藏的信息网络
- **风闻司/涤心院**：势力内部视角，在主角不知道的地方运转

**效果**：让读者看到主角看不到的事。主角以为A在追杀他，读者通过另一个POV知道A其实在保护他——这就是戏剧反讽。

**频率**：每5-8章至少安排一个非许若存POV场景。

### 5. 章末钩子铁律

每章最后一句话必须让读者想翻页。五种标准钩子：

| 类型 | 示例 |
|------|------|
| **突然出现的人** | 窗外有什么东西轻轻响了一下。他没睁眼。手指攥着那块玉，在黑暗里慢慢收紧了。 |
| **未说完的话** | "你爹当年走的时候说了一句话——"她停住了。"他说了什么？"她没回答。 |
| **反常的细节** | 桌上那只碗还在原位。但碗里的水，少了一半。 |
| **悬而未决的决定** | 他站在岔路口。左边是落霞镇，右边是洛京。铜钱在掌心里慢慢转了一个方向。 |
| **危机迫近** | 脚步声停了。停在门外。 |

**禁用结尾**：许若存睡着了、许若存继续赶路、许若存在脑子里总结今天的事、叙事者总结了本章含义。

### 6. 冲突前置

每一章至少有一个角色在冒风险、在做不可逆的选择。不写只有信息传递没有冲突的纯过渡章。

**冲突的最低标准**：两个角色之间至少有一个人说的话对自己有风险。如果一段对话里双方都在说客观事实、没有任何一方在冒风险——删掉。

**反面示例**：两人坐下聊"昨天发生了什么事"彼此交换已知信息——这等于作者借角色之口告诉读者。
**正面示例**：红姑知道铜钱的事但不说破，许若存知道她在瞒他——两人每一句话都是试探和抗拒。

### 7. 高潮密度

| 节奏 | 频率 | 内容 |
|------|------|------|
| **小高潮** | 每8-12章 | 一场单挑、一次突围、一个反转 |
| **重要战斗** | 跨2-3章写 | 中间穿插其他视角和线索，保持悬念 |

**不出现**：连续15章以上的"平路"（只有跑路和信息收集，没有冲突爆发）。

**判断标准**：读完最近5章后，能不能说出"这一章的爆点是什么"？说不出来——节奏出问题了。

### 8. 每弧是一个独立的"小故事"，气氛拉满

全书按弧组织，每弧10-15章。每一弧在读者感受中应该像一本独立的中篇小说——有自己的类型基调、情绪曲线和完整结局。

**每弧的类型基调要鲜明**：

| 弧 | 类型基调 | 用什么拉气氛 |
|----|---------|------------|
| 弧2·秩序之门 | **体制惊悚** | 城门像一张嘴，序塔像一只眼睛，每一个穿灰袍的人都在看你 |
| 弧3·湖底 | **黑暗奇幻** | 水下石门、母亲手印、虚甲兽的甲壳在黑暗中摩擦的声音 |
| 弧4·星散 | **群像悲情** | 每个人告别的方式——苏缠解绳时线断了的那一声、沈见微转身前最后那个眼神 |
| 弧5·推门 | **史诗牺牲** | 柳无眠燃尽时的光、沈见微化为云纹前问"今年有人记得我吗" |
| 弧6·裂隙 | **孤寂求生** | 域外没有声音、没有方向、没有温度——只有窃语者模仿母亲的呼唤 |
| 弧7·问宗 | **哲学恐怖** | 遗忘之主说"我连自己的名字都忘了，但我记得天道欠我一个回答" |
| 弧8·前世 | **身份惊悚** | 另一个自己坐在对面——"你不是来修复天道的，你是来替我回答的" |
| 弧9·归途 | **安静的告别** | 纸烧完了、面吃完了、铜钱还完了。剩下一个人，一锅粥，晨光 |

**气氛拉满的具体方法**：

① **弧开头三章定调**。每弧的前三章必须让读者立刻知道"这个故事和上一个不一样"。弧2开头是城门盘查的压迫感，弧3开头是湖水变黑的诡异感，弧4开头是各怀心事的离散感。不要写过渡章。

② **弧内每章递进一层紧张**。不原地打转。弧2第1章在城门口被查，第3章在序塔底下看到门缝，第6章发现万流坊也有眼线——每一章都让局势紧一分。可以有"透气章"（柳三娘的面、秦望舒的姜糖水），但透气章之后下一章必须比之前更紧。

③ **弧中段设一个"假希望"**。让读者以为问题解决了，但紧接着揭示问题更深。弧3的心湖之战击退虚甲兽→以为安全了→发现母亲手印凉了。弧4的苏缠解绳→以为自由了→月丝不断，她仍被追踪。

④ **弧末留一个"余味"画面**。不是总结，是一个有画面感的瞬间。弧3末段可以是许若存站在岸上，看湖面恢复到什么都照不出的黑色。弧4末段可以是四枚铜钱在桌上排成一条线，指向同一个方向。这个画面就是读者在等下一弧时脑子里反复回放的东西。

**判断标准**：读完一个弧的第5章后，读者应该能说出"这一弧的味儿不对，和前几章不一样"。如果5章了读者还觉得"和之前差不多"——类型基调没立住。
