# 《岁蚀》势力体系

> 六大势力详情、关系矩阵、核心矛盾

---

## 六大势力总览

| 势力 | 定位 | 制度/结构 | 核心矛盾 | 代表人物 |
|------|------|----------|---------|---------|
| **昭朝**（世俗皇权） | 统治凡人世界，定都洛京。下设**风闻司**（特务监察）暗中监控承序府与宗门动向 | 户籍黄册→赋役包税→驿传关禁（路引）→科举选官→风闻司（特务监察）。律法双轨：凡人归刑部，修行者归承序府 | **外部矛盾**：昭帝是凡人（曾自废修为，不愿变"不死"），无法压制宗门，想在死前让昭朝脱离宗门依赖。**内部矛盾**：皇权体系本身暗流涌动——风闻司与文官体系的博弈、昭帝与主战派的路线分歧 | 昭帝（深居内宫）、韩指挥使（风闻司）、九公主、蓝衣人（风闻司外勤，第11章银链标记首次出现） |
| **承序府**（修行事务衙门） | 名义隶属昭朝、实际独立运作的中间机构。下设武装执行机构**镇魔司** | 廷正→档案楼→仲裁台→仙吏司→收缴司→**镇魔司**（追捕押解）。双重供养（朝廷工资+宗门供奉）= 双重忠诚，两边都可听完不办 | 对凡人行使管辖权远超应有边界（路引追踪、灵力残留备注、转交权、"不开放层"档案） | 叶知秋（基层仙吏）、纪无咎（收缴司）、钟离燕（档案员）、**沈见微**（镇魔司头号人物/涤心院出身）、**陈静安**（前任阵师，设计涤心院洗心阵后被关归寂院十六年） |
| **万流坊**（地下交易网） | 无中心交易体系，遍布灰色地带 | 三级节点制（公开→半公开→隐蔽），因果券交易。没有账本，没有主人，但有"看门人"维护规则 | 昭朝禁而不绝（官吏自己也在买），涤心院是记忆珠最大买家 | 老鸹（看门人）、花姐（青石集）、老鼹鼠（隙径情报） |
| **涤心院**（净化之道） | 全书深层反派。**表面**：记忆净化，制造"完美的人"。**真相**：洗心=控制——每半年一次不是因为你"需要净化"，是因为组织需要你忘记你开始怀疑的东西。**档案涂改系统**：入院弟子的原始档案被系统性篡改——抹掉家庭背景、涂改父母信息、伪造"孤儿"身份——切断弟子与外界的一切联系 | 洗心制度（每半年一次），心网监控，档案涂改（老廖守档四十年，最后两年一直在改旧档，死后院长下令封存）。**洗心阵来源**：承序府阵师陈静安设计三道防线，第三道被改（归档→抽走）。**核心讽刺**：涤心院以为洗心能对抗岁蚀，实际岁蚀最先瓦解的就是洗心构建的虚假秩序 | 沈见微（昔年最强洗心者·三十年才看清洗心=控制·死后化为石门云纹）、裴先生（导师·真心相信洗心在救人）、老廖（档案管理员·涂改者·第13章丧钟七声·已故）、**陈静安**（洗心阵设计者）、还有成百上千个没有沈见微那些偶然的弟子——他们还在被洗 |
| **幻音楼**（梦境观察者） | 非势力中立的观测者组织 | 不入世，只记录。梦境通道感知世界状态 | 观测行为本身会改变被观测对象——越深入观测，越影响世界，越偏离中立 | 苏缠（红绳感应者） |
| **不周山庄**（残缺哲学） | 独立宗门，主张"完整是假象" | 不对外扩张，对天道碎裂的态度：不是坏事。"缺口才是入口" | 不周山庄的哲学和承序府的秩序至上完全相反——冲突迟早爆发 | 谢渊（主动接触许若存） |

### 势力关系矩阵

| 甲 vs 乙 | 关系 |
|----------|------|
| **昭朝 ↔ 承序府** | 名义上下级，实质互相利用。昭朝靠承序府制衡宗门，承序府借皇权扩张凡人管辖权 |
| **昭朝 ↔ 涤心院/幻音楼** | 无法指挥，但可通过控制凡人资源（粮食、户籍、路引）施压 |
| **昭朝 ↔ 万流坊** | 禁而不绝——官吏自己也在万流坊买东西 |
| **承序府 ↔ 涤心院** | 涤心院是记忆净化者，承序府需要涤心院产出"安定"的修行者。**陈静安关联**：承序府阵师为涤心院设计洗心阵，第三道防线被改后陈静安被关归寂院——两家之间有一笔旧账 |
| **承序府 ↔ 万流坊** | 监控利用——档案室有节点名录，但不彻底铲除 |
| **涤心院 ↔ 万流坊** | 涤心院是记忆珠的最大买家（洗掉的记忆被万流坊回收交易） |
| **不周山庄 ↔ 承序府** | 哲学对立——不周山庄认为"完整是假象"，承序府追求绝对秩序 |
| **不周山庄 ↔ 万流坊** | 合作关系——万流坊的盗机巷设在不周山庄地界上 |
| **镇魔司 ↔ 涤心院** | 人员互通——镇魔司从涤心院招募洗心者协助追踪（沈见微即此渠道）。涤心院借镇魔司的执法权扩大活动范围 |
| **镇魔司 ↔ 承序府** | 直属上下级——承序府提供编制、授权和档案情报，镇魔司负责执行。但镇魔司有跨势力人员，不完全受承序府控制 |

### 对凡人的控制手段（承序府专表）

| 手段 | 表面功能 | 实际功能 |
|------|---------|---------|
| **路引系统** | 关隘检查，身份核验 | 追踪所有人（包括凡人）流动轨迹——"XX日离籍，XX日未归"长期标注 |
| **户籍黄册扩展** | 人口登记 | 加"灵力残留备注"——接触过修行者的凡人被悄悄标注"可疑"，永不删除 |
| **转交权** | 涉修行案件移交 | 转交后的凡人不再受昭朝律法保护，关多久、审不问全由承序府说了算 |
| **档案楼"不开放层"** | 宗门秘密存档 | 大量凡人"疑似修行接触记录"——没修过行的人因孩提时见过修士就被记录在案，永不销毁 |

### 镇魔司（承序府武装执行机构）

> 承序府下属的追捕押解机构，专门处理涉及修行者的高危案件。跨势力招募人员，独立行动权极大。

| 项目 | 内容 |
|------|------|
| **隶属** | 承序府（名义），实际对承序府和昭朝双重汇报 |
| **职能** | 追捕修行者逃犯、押解高危目标回洛京、处理跨区域修行者案件 |
| **标准配置** | 三人小组——主搜（明心境，气感扫描）+ 堵口（感气高阶，守后路）+ 望风（覆盖视野）。第10章镇魔司搜捕柳三娘客店即用此配置 |
| **追踪手段** | **"压"**——大面积气感扫描，像铁板按过去，感知范围内所有人的气脉分布；**封印残留追踪**——激活过的封印会留下"痕迹"，可被远程感知定位 |
| **标志** | 玄色袍服、黑铁令牌（压镇魔司纹印） |
| **代表人物** | **沈见微**（头号人物，涤心院出身，昔年最强洗心者，5岁起洗心三十次） |
| **核心矛盾** | 跨势力人员构成导致忠诚分裂——沈见微的身体在抓捕许若存时自行失控，说明洗心也洗不掉的东西会通过镇魔司的执法行动被重新激活 |
| **首次出场** | 第10章（客店搜捕），第12章正面展开（沈见微包子摊对峙） |

### 昭朝权力结构（皇权体系）

> 名义上统治所有人，实际上对宗门和承序府的控制力有限。皇权内部同样暗流涌动。

| 层级 | 机构/角色 | 职能 | 现状 |
|------|----------|------|------|
| **皇帝** | 昭帝 | 凡人之身统治昭朝，曾自废修为（不愿变"不死"） | 深居内宫，想在死前让昭朝脱离宗门依赖 |
| **特务监察** | **风闻司** | 昭帝的耳目，监控承序府越权行为和宗门动向。外勤人员以银链为标记 | 与承序府暗中角力。第11章蓝衣人（风闻司外勤）出现在落霞镇，给许若存递纸条——昭朝在追踪许若存事件中的立场尚未明朗 |
| **文官体系** | 六部+地方官府 | 科举选官，管理凡人赋税、路引、关隘 | 基层执行力被承序府架空——关隘名义属昭朝，实际承序府仙吏坐凉棚说了算（第11章石桥关隘） |
| **军事力量** | 昭朝兵丁 | 凡人军队，维持关隘和地方治安 | 对修行者无战斗力，只能管凡人 |
| **皇室内部** | 九公主等 | 昭帝子女，各有立场 | 卷四九公主出场，与许若存联手交换情报 |

**核心矛盾**：
- **对外**：名义上"所有人归皇权管理"，实际上承序府管修行者却不断越权渗透凡人体系。昭朝通过路引/关隘控制凡人流动，但承序府在关隘设凉棚翻册子，把跟修行者说过话的凡人也标注"灵力残留观察"——昭朝的关隘变成了承序府的档案采集点。
- **对内**：昭帝是自废修为的凡人，风闻司与文官体系路线分歧——是继续容忍承序府越权换取稳定，还是撕破脸夺回管辖权？

### 各势力追踪许若存的手段（非开天眼）

> 不是"全世界都知道他有东西"。是他激活封印时泄漏了一次信号，信号在灰色市场里被转卖。不同势力拿到信息的精度和时间不同——所以追兵分批次到达，互相之间不认识。

| 势力 | 怎么知道他的 | 信息来源 | 精度 |
|------|------------|---------|------|
| **镇魔司（承序府）** | 封印激活→信号被镇魔司法器捕捉+路引关隘发现他没办路引+柳三娘客店被搜出痕迹 | 官方监测网络 | 高（坐标级），有专人追踪 |
| **雇佣猎杀队（水塘三人组）** | 信号被万流坊信息贩子捕捉→转卖给中间人→中间人雇佣猎杀队 | 灰色市场转卖 | 中（方向+大致区域），需要自己布网搜 |
| **风闻司（昭朝）** | 蓝衣人在落霞镇监控各方动向→发现多路人马在围同一个目标→反向推断出目标的重要性 | 人力情报+反向推理 | 低，靠观察和分析 |
| **涤心院** | 沈见微的报告→洗心三十次的身体记忆被许若存左肩封印触发→组织层面开始关注 | 内部报告 | 中，沈见微个人感受先于组织反应 |
| **万流坊（信息贩子）** | 许老头是三级节点→许若存穿着万流坊标记服上路→节点之间的信息流通 | 组织内网络 | 低，只有当许若存主动经过节点时才被注意 |
| **何伯安** | 灵脉偏移→用罗盘追踪偏移源头→追了几个月→在青石集锁定许若存 | 技术追踪（非宿命） | 中，追到青石集才精确 |
| **红姑** | 多年前一个姓许的教书先生留了一枚铜钱→她自己选择留着→讲老陈的故事等那个"会来的人" | 个人选择（非安排） | 极低，不知道来的是谁 |

### 各派进场原则

> **核心铁律**：人物进场时不围绕主角转。每个势力有自己的内部逻辑和议程——他们在做自己的事，恰好与许若存的轨迹交叉。用独立POV场景展示势力的内部运作，让读者看到世界在主角之外也在运转。

| 原则 | 说明 |
|------|------|
| **有自己的事** | 每个势力角色出场时，带着自己那摊事——谢渊下山是因为师尊有任务，不是来偶遇许若存的；叶知秋在茶亭等人是因为有人托他传话，不是专门来等许若存的 |
| **内部视角** | 关键势力（涤心院、承序府、风闻司）有独立POV场景——院长翻报告、纪无咎写案卷、韩指挥使下指令——这些场景里许若存不在场 |
| **交叉而非围绕** | 人物的轨迹和许若存的轨迹交叉，不是所有人围着他转。何伯安追踪灵脉偏移追到青石集，许若存刚好在那里——是何伯安的追踪把他带到了许若存面前，不是他"等了三个月" |
| **第一次出场不一定是和主角互动** | 蓝衣人第一次出场在第11章茶馆（许若存视角），但他的身份和任务在第18章末风闻司内部场景才揭示——读者比许若存先知道一些事 |
