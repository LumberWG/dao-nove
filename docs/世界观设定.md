# 《岁蚀》世界观设定

> 整合第1-82章已揭示的世界规则与设定，按主题分类，方便写作时快速查阅。
> 本文档中的每条规则都在正文中已实际出现或被明确暗示。

## 一、天道与岁蚀

### 天道碎裂

很久以前，天道是完整的。完整不是"完美"，而是"所有可能性都还在流动"。后来有人试图把天道"固定"下来——修门、筑塔、写律、设阵。天道没有反抗，它只是"裂开了一道缝"。

那道缝，就是第一扇门。门不是通道，是伤口。门后面不是另一个世界，是天道被切掉、被遗忘、被压扁的那部分。

### 岁蚀四阶段

| 阶段 | 现象 | 卷一对应 |
|------|------|---------|
| **前兆期** | 局部记忆错位、梦境渗透、旧伤发热 | 许若存封印跳动、玉佩纹路生长 |
| **渗透期** | 小范围空间失声、人伦关系断裂 | 金泉镇无声区域（14章）、王猎户忘记妻子 |
| **降临期** | 大范围"被允许遗忘"，秩序信息死亡 | 洛京城墙从内开裂（70-80章） |
| **终末期** | 门彻底打开，未被选择的可能性涌入现实 | 未发生（卷七） |

## 二、隙径

### 本质

隙径不是物理通道，是天道碎裂在空间上留下的裂缝，属于宏观尺度的量子效应。

### 核心规则

| 规则 | 说明 | 正文范例 |
|------|------|---------|
| **不要回头** | 回头=自我观测=坍缩到不可控状态 | 第8章灵犀警告 |
| **拴住最重要的一样东西** | 否则会在退相干中丢失核心记忆 | 第8章许若存拴铜钱 |
| **走得越多，散得越多** | 每次穿越都会丢一点东西 | 第8章丢了对马婶的难过 |
| **记忆泄漏** | 退相干中信息可被他人捡取 | 第8章捡了"对馒头的愧疚" |
| **镜中人** | 完全退相干者留下的残影，重复同一句话 | 第8章破庙女人声音 |
| **隙径入口可改造** | 许衡改造过隙径出口 | 第34章老隙径师指认 |

### 隙径中的关键存在

| 存在 | 说明 | 出现章 |
|------|------|--------|
| **镜中人** | 走隙径时散掉的人留下的残影 | 8 |
| **灵犀（隙径向导）** | 天生部分状态与隙径重叠，走隙径不会退相干 | 8, 37+ |
| **灵犀的妹妹** | 隙径原生居民，被留在隙径深处 | 63 |
| **记忆碎片** | 穿越者泄漏的记忆可被他人捡取 | 8 |

### 隙径与万流坊

万流坊控制着隙径网络的部分入口和交易。隙径网络在洛京的入口逐个失效（第37章），暗示隙径有人在"从另一边关上"。

## 三、门

门是连接凡间与域外的伤口。全书不止一扇门，但所有门都是同一道伤口在不同位置、不同尺度上的显形。

### 门一览

| 门 | 位置 | 状态 | 相关人物 |
|----|------|------|---------|
| **心湖之门** | 无名小城心湖底 | 母亲右手按了十六年 | 母亲、许衡 |
| **序塔之门** | 洛京序塔底部 | 铁门，被守塔人看守 | 曲姓守塔人、许衡 |
| **井底门（"门边上那道缝"）** | 城西土地庙/黑石城 | 许衡改造的隙径出口 | 许衡、老隙径师 |
| **净心堂后门** | 涤心院 | 从未打开过 | 裴先生、沈见微母亲 |
| **归寂院之门（隐喻）** | 归寂院 | 让人"被世界忘记"的微观门 | 老陈 |
| **原型门** | 隙径深处 | 所有门的原型 | 萧轨、秦鹿 |

### 门的规则

- 门关得越紧，凡间越稳定；门开得越大，域外漏进来的越多
- 门不可能永远关死——天道本来就是要流动的
- 门环被许衡掰断后，门关不紧也打不开
- 母亲用身体住门缝十六年

## 四、封印体系

### 许若存的左肩封印

| 维度 | 内容 |
|------|------|
| **来源** | 许衡在天道裂缝闭合最后一瞬，把自己的部分本源撕下来塞进儿子身体 |
| **成分** | 许衡的修为、感知、记忆、预视能力 |
| **功能** | 封印住"未择之身"，同时封存钥匙 |
| **脉动触发** | 靠近门、隙径、铜钱共鸣 |
| **信号泄漏** | 主动内视封印=在公共频道广播位置 |
| **钥匙属性** | 第40章揭示：钥匙在封印里 |

### 灵犀的左肩封印

与许若存的封印同源——两人在"被门选择"这一点上有共同的印记。灵犀的封印在出豁口后不再跳动。

### 纪无咎的缺指封印

缺指处的皮肤与许若存左肩封印同为暗青色——他也是"钥匙"的一部分（第61章）。

### 沈见微的掌纹分叉

不是封印，是洗心洗不掉的遗留——母亲的身体记忆通过血脉留在了他身体里。

## 五、水的记忆

地下水系连接所有水域，记忆可在水中传递。心湖是连接门的关键节点。

| 现象 | 出现章 | 说明 |
|------|--------|------|
| 心湖映出母亲影像 | 6 | 水面上浮现十六年前的画面 |
| 孟婆婆说"水认识你父母" | 6 | 水认识的人会记住 |
| "看完别碰水" | 6 | 碰了会被拉进门里 |
| 镜井中看到许老头 | 51 | 映出有人想让你看见的东西 |
| 倒流河银白色骨沙 | 85(卷二) | 门后之人的记忆残留 |

## 六、名字与遗忘

| 规则 | 说明 | 出现章 |
|------|------|--------|
| 有名字的东西能被记住 | 名字是存在的凭证 | 核心设定 |
| 无名的会自动隐形 | 归寂院第一步就是划掉名字 | 9 |
| 归寂院"被允许遗忘" | 不是死了，是没有人再记得 | 9 |
| "净尽"让人想起被洗掉的东西 | 涤心院的解药 | 47/50/56 |
| 被门留下的人 | 记得门的存在，忘记了自己是谁 | 81 |

## 七、灵气与修行

### 境界体系

| 境界 | 核心能力 | 突破方式 | 卷一状态 |
|------|---------|---------|---------|
| **感气** | 感知天地之气 | 自然觉醒 | ✅ 第1章起 |
| **明心** | 气凝聚、外放、扫描 | 看见一个真相 | ✅ 第7章封印内视 |
| **通法** | 气与术结合，御器、布阵 | 做一个不可逆的选择 | 📝 卷一末未突破 |
| **破妄** | 识破规则层面的幻象 | 承认最不想承认的事 | 🔮 待后续 |
| **问道** | 与天道裂缝对话 | 回答"我是谁" | 🔮 待卷六 |

| 谢渊 | 通法 | 不周山庄弟子，剑术 |
