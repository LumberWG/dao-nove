# 《岁蚀》封面设计方案

## 风格定位

文学玄幻 · 暗调 · 质感优先——不是传统仙侠的金碧辉煌，而是伪戒式的灰调质感。像一本旧书店里偶然发现的古籍，封面被岁月磨出了毛边。

## 色彩

| 色 | 用途 | 色值 |
|:---:|------|:----:|
| ⚫ 墨黑 | 底色，占70% | #1a1a1a |
| 🟤 旧纸 | 文字和点缀 | #f5f2ed |
| 🟡 铜色 | 核心高光 | #c4943c |
| 🟢 玉色 | 玉佩点缀 | #6b8e6b |
| ⚪ 雾灰 | 过渡 | #8a8a8a |

## 构图方案

### 方案A（推荐）：裂纹·铜钱·门

竖排三分构图：

```
┌─────────────────────────┐
│                         │
│     ·  ·    ·            │  上1/3：虚空
│          ·               │  散落光点 + 隐约裂纹
│    ╱╲   ╱╲              │  右上角微型裂纹纹理
│   ╱  ╲ ╱  ╲             │
│  ──  ◆  ──              │  中1/3核心区
│     ◇ ◇ ◇               │  一枚碎玉（裂成两半）
│                         │  三枚铜钱呈三角排列
│    ░░░░░░░              │  玉上裂纹延伸至上/下
│       ▓                 │
│       ▓                 │  下1/3
│   ┌── ▓ ──┐            │  一扇简笔石门的轮廓
│   │    ▓    │           │  门缝里透出一线昏黄光
│   └────────┘            │
│                         │
│     岁  蚀              │  标题：竖排或横排，铜色
│   SUI EROSION           │  副标题，极淡小字
│                         │
│    — 全七卷 —           │  底部信息
└─────────────────────────┘
```

### 方案B：极简·两个字

整张封面只有两个字「岁蚀」，用粗犷的毛笔字体，但笔画边缘有破碎/消散的效果（像墨迹被风蚀了一部分）。底色深灰到黑的渐变。

### 方案C：归寂院·墙

一道长长的灰墙从左到右贯穿封面，占画面下2/3。墙上有一扇虚掩的黑漆木门，门缝里一丝光。墙的上方是灰白色的天空，标题用极细的字体放在天空里。

## AI绘图提示词（方案A）

> **中文描述：**
> 小说封面，暗调东方奇幻风格。深色背景，中央一枚纵向裂纹的青白玉佩悬浮，玉佩断口处延伸出细密的金色裂纹，像发光血管一样向四周扩散。三枚古铜钱呈三角形环绕玉佩。画面下方一扇简笔石门轮廓，门缝透出极细的暖光。细微光点像灰尘般飘散。整体质感厚重，像被岁月侵蚀过的旧绢本。配色以墨黑、旧铜色、暗玉绿为主。标题"岁蚀"用行楷竖排。气氛孤寂、神秘、有一丝衰败的美感。

> **English prompt (Midjourney):**
> Chinese fantasy novel cover, dark literary xuanhuan style -- A vertically cracked celadon jade pendant floating center, golden crack lines spreading from the break like luminous veins -- Three ancient bronze coins arranged in triangle around the jade -- Subtle stone door silhouette at bottom, thin warm light seeping through door crack -- Tiny floating dust-like particles -- Heavy texture like aged silk scroll -- Color palette: ink black, aged copper, faded jade green, mist grey -- Title "岁蚀" in vertical calligraphy -- Atmosphere of solitude, mystery, eroded beauty -- No character figures, pure symbolic composition -- Aspect ratio 2:3

> **适用于：** Midjourney v6 / DALL-E 3 / Stable Diffusion with LoRA
