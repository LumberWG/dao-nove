# 《岁蚀》核心设定

> 适配300万字·7卷·伪戒风格版。

---

## 一、故事核心

### 一句话概括

一个旧货摊长大的少年，拿到一块碎玉和三枚铜钱后，发现自己能看到"两种未来"——代价是每次使用都让左肩的封印松一道，而封印下面锁着门后的虚无。

### 核心命题

不是"拯救世界"，是**"找回的记忆让你变成一个你憎恨的人，你还想找回它吗"**。许若存每接触一块碎片就恢复一部分记忆，但那些记忆里的自己是个冷血、聪明、为达目的可以牺牲任何人的怪物。而且他发现自己正在越来越像那个人。这不是寻回自我——是前世的自己在利用碎片覆盖现在的他。

### 终极指向：天道·世道·人道的一致性

本书的底层世界观不是玄幻物理——是**同一套规则在不同尺度上的显形**。

- **天道** = 自然法则（量子涨落、熵增、信息守恒、对称性破缺）
- **世道** = 社会秩序（权力分配、信息控制、体制对人的塑造）
- **人道** = 人性选择（记住/遗忘、信任/背叛、付出/索取）

这三者不是三层，是同一个东西在不同尺度上的投影。微观粒子会量子纠缠，人也一样——人与人的羁绊、记忆的纠缠、因果的传递，底层逻辑和隙径的量子效应是同一条规则在变大。天道碎裂在空间上留下隙径，在人心上留下岁蚀，在社会结构上留下权力裂缝——同一个创伤的三个剖面。

**科学概念的写作指导：**

不需要在故事里讲科学。是把科学的内核转化成玄幻语言，让读者在读故事时隐约感受到"这个规则好像在哪见过"。

| 科学概念 | 玄幻化表达 | 小说中的体现 |
|---------|-----------|-------------|
| **量子叠加** | "同时存在两种可能" | 未择之身、隙径穿越 |
| **量子纠缠** | "隔着很远也能互相感应" | 铜钱认主、红绳感应、母子左肩共鸣 |
| **退相干** | "接触外界后确定性塌缩" | 隙径中"不要回头看"、走完隙径的"定下来" |
| **熵增** | "一切都在走向遗忘" | 岁蚀、归寂院的信息死亡 |
| **信息守恒** | "信息不会消失，只会转移" | 隙径中的记忆互换、"别人的感觉塞进你脑子里" |
| **对称性破缺** | "完美中裂开一道缝" | 天道碎裂、不周山庄"缺口才是入口" |
| **局域逆熵（生命）** | "在局部范围内对抗遗忘" | 柳三娘一碗面的暖意、秦望舒在岁蚀之夜继续做纸 |

**最核心的一条对应：**

> **岁蚀不是自然灾害——是天道忘记了一段记忆。一个人忘记一件事，和世界忘记一个概念，是同一个过程在不同尺度上的发生。**

这就是天道·世道·人道一致性的终极表达：微观的量子效应放大到宏观就是世界规则，世界规则落在人类社会就是权力结构，权力结构落实到个人就是记忆与遗忘的选择。整个故事从许若存个人丢失一块感觉（对马婶的难过），到全城记忆互渗（岁蚀渗透期），到天道遗忘一整段历史——是一条逻辑链条上的三个尺度。

**写作上怎么实践：**

不用角色之口解释"这就像量子力学"。用动作、现象、规则去展示。读者自己品出"这不就是熵增吗"才是成功。如果角色跳出来说"这跟热力学第二定律一样"，反而是败笔。

**判断标准**：遮住魔法术语（灵力、封印、阵法），看故事内核——那些规则是不是和现实世界的物理/社会/心理规律一致。如果答案是"是"，科学视角就融进去了。

### 三条主线

| 线 | 内容 | 贯穿卷 |
|----|------|--------|
| **记忆线** | 许若存寻找身世真相，接触碎片→恢复记忆→发现前世自己是另一个人 | 一至六 |
| **门线** | 石门内外——母亲推了十六年，父亲在门后等了十六年，许若存最终走进去 | 一至七 |
| **选择线** | 未择之身的每一次使用都在松动封印，能力越强代价越大——选还是不选 | 一至六 |

## 二、世界观

### 世界规则

1. **名字的力量**：有名字的东西能被记住，无名的会自动隐形

2. **水的记忆**：地下水系连接所有水域，记忆可在水中传递

3. **力的传递**：灵力、体力、温度可在不同人之间传导分担

4. **因果链**：签下的契书具有强制约束力，违约自动抽取代价

5. **被遗忘的代价**：归寂院的人不是死了——是被世界确认不会再有人记得他们

6. **隙径的量子底层**：隙径不是物理通道，是宏观尺度的量子效应。两个遥远坐标在深层纠缠——走入一端即从另一端出。穿越时人体处于宏观叠加态（同时存在于所有可能位置），观测行为导致坍缩，因此"不要回头"是硬规则。记忆/感觉泄漏的本质是**量子退相干**——信息从穿越者身上退到环境中，可被他人捡取，也可完全散去（"散掉"）。

7. **隙径中的"镜中人"**：在隙径中迷失者的意识残影（完全退相干者留下的残余纠缠痕迹），会跟着路过的行者重复同一句话

### 岁蚀的本质

天道碎裂后产生的"遗忘泄露"。不是自然灾害——是天道忘记了某段记忆，这段记忆在现实中的存在就会变薄消失。四阶段：前兆期→渗透期→降临期→终末期。

### 隙径

天道碎裂在空间上留下的裂缝，连接不相邻的坐标。使用需感知入口→注入灵力激活→穿越模糊期。修为越低风险越大，有人会"散掉"变成镜中人。玉佩上的纹路是一幅隙径地图。

### 境界体系（5境）

感气→明心→通法→破妄→问道。突破不靠打坐闭关，靠"看见"真相。每个境界都有对应的实战能力。

## 三、势力格局

**核心矛盾——三轴冲突**：

- **秩序vs自由**：昭朝+承序府 vs 不周山庄+万流坊

- **记忆vs遗忘**：照影台+幻音楼 vs 涤心院

- **现世vs域外**：所有地面势力 vs 岁蚀

### 七大势力

| 势力 | 本质 | 核心利益 | 代表人物 |
|------|------|---------|---------|
| **昭朝** | 凡人皇权 | 在死前脱离对宗门的依赖 | 昭帝、韩指挥使 |
| **承序府** | 双重隶属的灰色衙门 | 在昭朝和宗门之间活下来 | 叶知秋、纪无咎、钟离燕 |
| **万流坊** | 无中心地下交易网 | 在所有势力之间保持缝隙 | 老鸹、花姐、老鼹鼠 |
| **涤心院** | 记忆清洗 | 证明洗心是对的 | 沈见微、裴先生 |
| **幻音楼** | 被动观测者 | 记录一切，不站队 | 苏缠 |
| **不周山庄** | 解构主义者 | 推翻所有"秩序" | 谢渊 |
| **照影台** | 情绪映照 | 维持镜子不碎 | 白无瑕 |

## 五、风格定位

- **叙事**：对话驱动，每章末尾必须有钩子，无大段背景描写

- **语言**：少用破折号，主角内心可用现代口语，每章2-3处

- **人物**：许若存开朗嘴贫有脾气，不是沉默观察者

- **战斗**：每8-12章一个小高潮，重要战斗跨多章

- **感情**：不说破为主——柳三娘懂他不点破，秦望舒知道一切但等他来懂（表面"等他不追问"，底层是"她替他守着秘密直到他能听懂"），苏缠分而未分让读者自己判断
