# 《岁蚀》功法与战斗体系

> 基于第1-82章已揭示的修行规则。境界突破不靠打坐闭关，靠"看见"真相。

---

## 一、境界体系（五境）

| 境界 | 核心能力 | 突破方式 | 卷进度 |
|:----:|---------|---------|:-----:|
| **感气** | 感知天地之气，分辨气息类型 | 自然觉醒或被外力触发 | ✅ 卷一初 |
| **明心** | 气凝聚、外放、扫描，可内视己身 | 看见一个真相 | ✅ 卷一中 |
| **通法** | 气与术结合，御器、布阵 | 做一个不可逆的选择 | 📝 卷二 |
| **破妄** | 识破规则层面的幻象 | 承认自己最不想承认的事 | 🔮 卷四/五 |
| **问道** | 与天道裂缝对话 | 回答"我是谁" | 🔮 卷六 |

### 各境界实战能力

| 境界 | 可做到 | 做不到 |
|:----:|--------|--------|
| **感气** | 感知灵力波动、分辨气息来源、微弱的防护 | 无法外放攻击、无法御器 |
| **明心** | 气劲外放、短距扫描、内视自身、基础防护罩 | 无法长时间御器、无法布阵 |
| **通法** | 御器飞行、布设阵法、气劲离体攻击、灵力传音 | 无法改变规则、无法与门对话 |
| **破妄** | 识破幻象、影响他人感知、短距瞬移 | 无法修复天道裂缝 |
| **问道** | 与门对话、在隙径中自由行动 | — |

## 二、未择之身

### 定位

许若存的核心能力。不是"天赋"，是许衡在天道裂缝闭合最后一瞬撕下本源塞进他身体造的。

### 能力表现

- 同时看见一个选择的两种短期未来（最多几炷香）
- 铜钱可作为未择之身的延伸——铜钱发热/转动的方向是预判的一种表现

### 代价

| 代价 | 说明 | 首次出现 |
|------|------|---------|
| 流鼻血 | 轻度使用的代价 | 第5章 |
| 消耗许衡的残余寿命 | 每次使用都在消耗父亲留在封印里的本源 | 第7章 |
| 松动左肩封印 | 使用越多，封印越松 | 第7章 |
| **被前世覆盖（长期）** | 能力和记忆同时从封印里泄漏——前世的自己正在借助碎片覆盖现在的他 | 🔮 卷四/六 |

### 核心限制（不可打破）

1. **只能看短期分岔**（几炷香以内），看不到长线结局
2. **看见不等于知道哪个对**——有时候看起来好的选择是陷阱
3. **不能用在自己身上**——无法预判"自己下一秒会选什么"
4. **使用需要代价**——没有无代价的使用
5. **不能连续使用**——封印需要时间恢复

## 四、法宝体系

### 铜钱

| 属性 | 内容 |
|------|------|
| **铸造者** | 萧轨与秦鹿（八百年前） |
| **总数** | 九枚 |
| **卷一已出现** | 七枚 |
| **核心功能** | 路标、信物、钥匙组件 |
| **消耗规则** | 使用消耗温度/能量，短期内无法恢复（第11章） |
| **认主** | 纪无咎的血被铜钱吸收（第35章）——铜钱在"记债" |

### 玉佩

| 属性 | 内容 |
|------|------|
| **来源** | 许衡留给许若存 |
| **功能** | 隙径地图（纹路随月光生长，靠近隙径入口发热） |
| **断口** | 被人从反面掰断，断口与门环断裂位置一致 |
| **关键特性** | 两扇门合成一扇完整门的投影（第4章拓片） |
| **状态** | 卷一全程持有，弧2后使用频率降低 |

### 红绳/月丝

| 属性 | 内容 |
|------|------|
| **来源** | 苏缠（幻音楼） |
| **功能** | 感知共鸣、方向指引（第54章）、封印共鸣 |
| **材质** | 月丝——幻音楼特制 |
| **状态** | 第49章留在井底门环上，第82章取回 |

### 镜子碎片（守镜人）

| 属性 | 内容 |
|------|------|
| **来源** | 守镜人 |
| **功能** | "门的眼睛"——能看到门后的碎片影像、看到过去 |
| **代价** | "收了他就能跟着你"（第70章） |
| **状态** | 许若存持有，已学会使用 |

### 藏心纸

| 属性 | 内容 |
|------|------|
| **来源** | 秦望舒（洛京纸坊） |
| **特性** | 能"留住被洗掉的记忆"，能扛住岁蚀侵蚀 |
| **标识** | 左上角有棉花印记 |
| **用途** | 档案保存、关键信息传递 |
