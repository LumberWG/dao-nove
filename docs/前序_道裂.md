# 前序

据说在很久以前，久到"很久"这个词还没被造出来，世界不是现在这样的。

那时候天和地的距离比现在近。山会走，水会倒流，有些石头偶尔会睁开眼睛看一眼路过的人，然后又闭上。所有东西都还没有完全定型，像一块还没干透的泥，可以捏成任何形状。

火不用燃料就能烧。一把剑铸成了，只要不折断，就永远锋利。祖母的脸不会在岁月里模糊成一道轮廓。遗忘这件事本身——还没有被发明出来。

没人觉得这有什么不对。鱼不会觉得水有什么不对。

后来，有了两股力量。

没有名字——那时候还没有任何东西有名字——但如果非要说，一股是"想要固定下来的"，另一股是"不想被固定的"。

一个想让山永远待在该待的地方，河流走固定的河道，日月按同一个时辰升起落下。它想要一张网，把所有东西都兜住，各归其位。

另一个说：不。

它没有理由，它只是不愿意。它生来就是为了挣脱的。

这两个东西撞在了一起。

那是天地之间第一次有"冲突"这种东西。在此之前，万物各是各的，互不妨碍。但一个要定，一个要走，就像骨头和筋长不到一块儿。

没人知道它们打了多久。那时候还没有白天和黑夜的分别，所以没法用"天"来算。只知道天被撕开过很多次，又被拼回去很多次。大地翻涌起来，像一锅烧开的水。山被连根拔起，甩到空中，碎成粉末落在千里之外。江河改了无数回河道，一会儿往东流，一会儿往西流，最后干脆不流了，悬在半空中，像一条凝固的绸带。

星辰的位置换了一轮又一轮。

有些地方被打出了裂缝。不是地上的裂缝——是更深处的东西裂了。像一面看不见的墙，被一锤砸下去，裂纹从撞击点向四面八方爬开。没有声音。或者声音太大了，大到没有任何东西能听见。风停在半空不动了。雨悬着，落不下来。所有正在发生的事情都停了一瞬。

然后——有什么东西碎了。

碎片落了下来。

有的沉进山腹，在地底的挤压中慢慢嵌入岩层，被后来的石头包裹起来，再也看不出来。有的落入江河，被鱼吞进去，又被更大的鱼吞进去，一代一代往下传。有的被风卷着飘了很久很久，最后落在一片荒地上，被雨水埋进土里。没人知道那些碎片是什么，也没人知道它们是从哪里来的。它们散落在各个角落，无声无息。

那两股力量也在那一击中耗尽了。一个散了，另一个也散了。像是两团撞碎了的浪花，各自化作无数细小的水珠，融进了世界各处。没有人再见过它们，或者说，它们一直都在，只是碎成了谁也认不出来的样子。

从那以后，世界变了。

火开始需要燃料。刀剑不再永远锋利。有人发现，同样的功法教给三个人，练出来会是三种不同的结果。祖母的脸真的会模糊。遗忘——这件事被发明了。

但也不是全变了。

有些地方还残留着一点从前的气息。沉在山腹里的碎片，偶尔会让那一整座山的石头比别处更硬。漂在水里的碎片，会让喝了那水的树多活几百年。落在荒野里的碎片，如果恰好有人踩过，可能会在夜里梦见什么说不清的东西。没有人知道那是为什么，也没有人把这些事联系起来。它们只是偶尔发生，然后被记下来，或者被忘掉。

又过了很久，久到那些事变成了传说，传说又变成了传说背后的影子。

山不再走了。水也不再倒流。石头闭上了眼睛，再也没睁开过。

世界变成了一个稳固的、可以预测的地方——绝大部分时候是这样。

而那些碎片，还在。

散落在没人知道的地方，等着被什么人捡起来。
