# 伏笔与线索追踪

> 用途：每章完结后记录新埋的伏笔，避免后续写作遗漏或矛盾。

> **更新规则**：
> 1. 每写/改完一章，在本文件末尾新增该章的线索表格。
> 2. 新增线索必须填写「预期回收章节」——不给预计位置就是给自己埋雷。
> 3. **开写新章前**：搜索本文件，找出所有「预期回收章节」= 本章号的条目，写完后逐条核销——已回收的改状态为「已揭示/已使用」，没回收的标注原因（延后/废弃/改方案）。
> 4. 状态分四种：**未回收**（埋了还没碰）、**部分回收**（露了一点但没全揭）、**已揭示**（本章正式揭晓）、**已使用**（已发挥作用可归档）。
