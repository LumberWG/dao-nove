### 情绪周期复盘（对照新补充规则§28）

**第1-9章（无名小城篇）**：
```
压抑(1-4) → 蓄势(5-6) → 爆发(7:封印真相) → 透气(8:隙径/灵犀) → 蓄势(9:归寂院)
```
第7章爆发后，第8章透气，第9章蓄势——节奏基本合理。但第1-4章压抑偏长（4章），按新规则应缩短。

**第10-20章（落霞镇→青石集篇）**：
```
微爽(10:柳三娘) → 压抑(11:追兵) → 蓄势(12:沈见微) → 透气(13:不会做梦的人)
→ 压抑(14:岁蚀区域/裂痕) → 微爽(15:叶知秋信息) → 蓄势(16-17:何伯安/第四枚铜钱)
→ 中爽(18:红姑说书/老陈真相) → 蓄势(19:纪无咎) → 压抑+钩子(20:十日倒计时)
```

**问题**：
1. 压抑最长连续 **3章**（11-12-13前半）——未超过新规则的4章上限，但偏紧
2. 中爽密度偏低——18章一次中爽，前后各有3-4章的无爽区间
3. 第20章以压抑结尾，直接连到弧2——好在弧2第26章（沈见微POV）自带透气功能
