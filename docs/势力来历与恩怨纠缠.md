# 势力来历与恩怨纠缠

> 七大势力的起源、发展、彼此之间的旧账。这些不是"设定"——是分散在七卷中、通过人物对话和情节逐步揭示的"故事"。

---

## 一、时间线全景

```
远古     道裂——两股意志碰撞，天道碎裂，碎片落入人间
  ↓
?年前    问宗建立——不认同秩序的人聚在一起追问天道本质
  ↓
~60年前  涤心院建立——创始人从归寂院出来，"经历被遗忘后建立了遗忘的技术"
  ↓
~40年前  问宗覆灭——残余转入地下，成为万流坊规则最早维护者
  ↓
~30年前  许衡+纪无咎发现天道裂缝（承序府修行学院同期）
  ↓
~20年前  许衡偷碎片救妻（照影台弟子），老陈设计洗心阵，许衡掰断门环
  ↓
~16年前  许若存出生，许衡走入石门，许老头接手抚养
  ↓
现在     故事开始（卷一），许若存16岁
```

## 二、七大势力来历

### 1. 昭朝——凡人皇权

| 维度 | 内容 |
|------|------|
| **起源** | 道裂之后，修行者崛起，凡人退守。昭朝建立于中央平原，以洛京/天阙京为中心，用户籍、路引、关隘三件工具维持对凡人的控制。存在时间比承序府更久——在承序府出现之前，昭朝是唯一的管理体系 |
| **核心矛盾** | 昭帝是凡人（曾自废修为，不愿变"不死"），无法压制宗门，想在有生之年让昭朝脱离宗门依赖 |
| **下属机构** | 风闻司——特务监察，暗中监控承序府和宗门动向。蓝衣人（第11章）即风闻司外勤 |

**揭示方式**：第11章石桥关隘（昭朝兵丁+承序府仙吏并排坐，管辖权冲突的微型模型）。第18章风闻司内部场景（韩指挥使写"不动"）。卷四九公主出场后展开昭朝内部派系。

---

### 2. 承序府——夹缝中的修行衙门

| 维度 | 内容 |
|------|------|
| **起源** | 昭朝无法管理修行者，又不敢放任宗门坐大。于是设立承序府——名义隶属昭朝，实际由各宗门出人出力，成为"双重隶属"的灰色机构。从一开始就不是"执法者"，是缓冲区 |
| **膨胀过程** | 最初只管修行者之间的纠纷。后来一步步越权：给凡人户籍加"灵力残留备注"→设关隘凉棚翻册子→"转交权"让凡人不再受昭朝律法保护→档案楼"不开放层"永久记录疑似接触者 |
| **内部结构** | 廷正→档案楼→仲裁台→仙吏司→收缴司→镇魔司 |
| **核心矛盾** | 双重供养（朝廷工资+宗门供奉）= 双重忠诚。两边的话都得听，两边的事都可以拖着不办。这种灰色生存方式本身就是它的本质 |

**揭示方式**：卷一通过叶知秋（基层仙吏）、纪无咎（收缴司）、沈见微（镇魔司）三个不同层级的人物，从侧面展示承序府的运作。卷二进入洛京后正面展开——档案楼、序塔、仲裁台逐一呈现。

---

### 3. 涤心院——遗忘的制造者

| 维度 | 内容 |
|------|------|
| **起源** | 创始人是一个从归寂院出来的女人。她经历了"被遗忘"——名字从户册划掉、认识的人不再想起她、自己也开始忘记自己是谁。她在归寂院里待了不知多少年，出来之后创建了涤心院。她的初衷或许是"让被遗忘的人有一个去处"，但技术一旦建立，控制就跟着来了 |
| **洗心阵的诞生** | 创始人找到承序府，要求提供阵师技术支持。承序府派了最好的阵师陈静安（老陈）。老陈设计了三道防线：归档→分类→封存。涤心院在交付后改了第三道：归档→分类→**抽走**。被抽走的记忆成为阵法燃料。老陈发现时已经晚了——洗心阵已经在运转，第一批弟子已经洗完了 |
| **核心讽刺** | 创立它是为了对抗遗忘，但它本身成了遗忘的制造者。更深的讽刺：岁蚀来的时候，最先瓦解的不是正常人的记忆，而是洗心构建的虚假秩序——被洗掉的东西在岁蚀里变成了回声 |

**揭示方式**：卷一第1章外乡人闲聊（名词出现）→第5章红姑讲老陈故事（洗心阵被改）→第12-13章沈见微个人线（洗心的代价）→第20章裴先生出场（教义内部视角）→卷二全面展开洗心阵运作。**创始人的故事**是最深的一层，留在卷四或卷五由归寂院里的某个残留信息揭示——让那个从归寂院出来的女人的故事，在归寂院里被重新讲出来。

---

### 4. 万流坊——缝隙里的网络

| 维度 | 内容 |
|------|------|
| **起源** | 道裂之前就有雏形——那时候没有路引关隘灵石，交易靠信物完成。道裂之后，问宗的残余转入地下，把原有的信物交易体系改造成了三级节点网络。"半扇门"符号（铜钱背面符纹）是那个时代的标记——意思是"你知道这扇门通往哪里，但你得自己走" |
| **为什么没有主人** | 问宗的教训——中心化的组织会被连根拔起。所以万流坊从一开始就不设中心。没有掌门，没有总部，没有账本。三级节点之间靠"看门人"维护规则。规则只有一条：欠了要还。不还——看门人会来收 |
| **与各方的关系** | 昭朝禁而不绝（官吏自己在买）。涤心院是记忆珠最大买家（回收被洗掉的记忆）。不周山庄提供地盘（盗机巷）。承序府监控但不铲除（档案室有节点名录） |

**揭示方式**：卷一第1章寻踪者提及→第25章花姐展开（铜钱符纹的万流坊含义）→卷二末~卷三全面展开（老鸹出场，许老头三级节点身份，因果券体系）。**问宗覆灭→万流坊转型**这段历史在卷三由老鸹或花姐讲述。

---

### 5. 不周山庄——缺口里的学派

| 维度 | 内容 |
|------|------|
| **起源** | 道裂之后最早出现的学派之一。不认同"天道碎裂是灾难"的主流叙事——他们认为碎裂本身就是天道的新形态。不修城墙，不修档案，不修人。"完整是假象，缺口才是入口" |
| **与其他势力的旧账** | 涤心院院长曾亲自上山庄，想在庄内建洗心堂——被师尊拒绝。院长走之前说"你们不周山庄什么都好，就是太相信人了"。承序府视其为异端，但因为山庄不扩张、不传教、不挑战现存秩序，一直没找到借口动手 |
| **问字令** | 山庄嫡传弟子持有的令牌。持令者可在山庄地界向任何人问一个问题，被问者不得撒谎。每次下山只带一枚，用完了就回来 |

**揭示方式**：第16章谢渊初登场（碎玉比喻、涤心院旧事、问字令）。卷二洛京重逢时展开山庄哲学体系。卷三谢渊带许若存上山庄——正面展示。**山庄与承序府的哲学对抗**在卷二~卷三逐步升级为实际冲突。

---

### 6. 幻音楼——沉默的观测者

| 维度 | 内容 |
|------|------|
| **起源** | 道裂之后，有人选择不站队，只记录。他们在梦境通道中观测世界状态——不介入，不评判，不留下痕迹。最初的观测者可能是问宗时期被排斥的"中立派" |
| **核心悖论** | 观测行为本身会改变被观测对象。越深入观测，越影响世界，越偏离中立。幻音楼内部一直有争论：看到了不说，算不算帮凶？ |
| **红绳** | 幻音楼观测者的体质标记。红绳感应体制——能感知特定人物的灵力频率和状态变化。苏缠的红绳连接着许若存体内的两道频率（封印+未择之气） |

**揭示方式**：卷一第1章名词出现→第30章苏缠初登场（红绳感应）→第50章展开（红绳体质代价）。**幻音楼内部争论**在卷四由苏缠的离开触发——她选择"说了"，就是选择了不再中立。

---

### 7. 照影台——镜子的守护者

| 维度 | 内容 |
|------|------|
| **起源** | 道裂之后，有一批修行者发现：天道碎裂不仅裂了空间（隙径），还裂了人心。情绪被放大，记忆互渗，越来越多的人分不清自己的感受和别人的感受。照影台创立于水域，用镜面阵法帮助修行者"照见自己的情绪"——镜子里映出的不是脸，是内心 |
| **代价** | 心镜越亮，照见的东西越多，照镜子的人就越分不清镜子里外。大成之后被心镜吞噬一半意识——这就是许若存母亲遭遇的 |
| **与许衡的关联** | 许衡的妻子是照影台弟子。心镜大成后意识被吞噬一半。许衡偷天道碎片打碎心镜救她——成功了，但她出来后境界崩塌，因为体内的碎片残留与儿子的未择之气互相吸引 |

**揭示方式**：第24章纪无咎首次提及（母亲是照影台弟子，心镜吞噬）。卷二末白无瑕下山——照影台正面展开。**心镜的本质**在卷五~卷六由许衡或母亲本人揭示。

## 三、核心恩怨纠缠线

### 线1：承序府 ↔ 涤心院（老陈·洗心阵事件）

| 时间 | 事件 |
|------|------|
| ~20年前 | 涤心院找承序府要阵师→派陈静安→设计三道防线→第三道被改（归档→抽走）→老陈发现→找人说理被堵→放走被陷害的徒弟→自废修为→入归寂院 |
| 现在 | 老陈在归寂院等了十六年。第9章许若存拿走棋谱。原件还在老陈手里——卷二隙径返回去取 |

**揭示方式**：分三段讲——第5章红姑讲前半段（阵被改了），第18章补充（他在刨坑找门），卷四或卷五揭示涤心院内部视角（当年谁改的、为什么改）。

### 线2：许衡 ↔ 纪无咎（师兄弟决裂）

| 时间 | 事件 |
|------|------|
| ~30年前 | 同门师兄弟，纪无咎视许衡为兄长。许衡替纪无咎挡过一刀（砍在左肩——许若存封印的位置） |
| ~20年前 | 共同发现天道裂缝→约定封印→许衡背叛→纪无咎追到石门前→门环断裂→纪无咎断指 |
| 现在 | 纪无咎找到了许衡的儿子。他要的不是仇——是"把未择之气抽回来补天"。但更深处，是一个被兄长抛弃的师弟，还在等一个解释 |

**揭示方式**：第19章纪无咎首次讲述（简短版），第24章补充完整版（照影台/心镜/母亲推门真相）。第28章（铜钱老人回归）何伯安补充许衡替纪无咎挡刀的往事——让纪无咎的执念多一层"欠了没还"的重量。卷五许衡隔门对话——纪无咎终于面对面问出那句话。

### 线3：涤心院 ↔ 沈见微（三十年洗心）

| 时间 | 事件 |
|------|------|
| 5岁 | 被母亲送进涤心院（大雨）——"母未绝"被涂改为"孤儿，父母双亡" |
| 5-35岁 | 每半年洗心一次，被称为"最强洗心者" |
| 第12章 | 手在包子摊前自己抖了——身体比大脑先醒来 |
| 第13章 | 第一个梦+厨娘来访+丧钟七声（老廖） |
| 第52-65章 | 岁蚀剥落洗心层→觉醒→倒戈→站上石门化为云纹 |

**揭示方式**：沈见微个人线是全书最完整的人物弧之一，不需要额外"讲述"——他的每一个动作、每一次手抖、每一个梦，都在揭示洗心阵对他做了什么。

### 线4：不周山庄 ↔ 涤心院（洗心堂拒绝事件）

| 时间 | 事件 |
|------|------|
| 多年前 | 涤心院院长亲访不周山庄，想在山庄建洗心堂——"合作共赢" |
| 当场 | 师尊拒绝。"洗心的前提是人本来不干净。我们不这么看。" |
| 院长走时 | "你们不周山庄什么都好，就是太相信人了。" |
| 含义 | 她不是在夸他们。她是在说——你们迟早被人坑 |

**揭示方式**：第16章谢渊走路时随口提起（已写入）。这个旧账不需要单独展开——它在卷二不周山庄和涤心院发生实际冲突时自然会被重新激活。

### 线5：万流坊 ↔ 问宗（前身→转型）

| 时间 | 事件 |
|------|------|
| 约40年前 | 问宗覆灭——原因不明（可能是被承序府+涤心院联手剿灭，也可能是内部分裂） |
| 之后 | 残余转入地下，不建宗门，改建成无中心网络——万流坊 |
| 铜钱符号 | "半扇门"——问宗时代的信物标记，万流坊沿用。三枚铜钱分别是问宗三位创始人的遗物 |

**揭示方式**：第25章花姐讲述铜钱符纹的万流坊含义。卷三老鸹讲述问宗覆灭→万流坊转型。**问宗覆灭的真正原因**是最深的一层——留在卷五或卷六由域外的遗忘之主（问宗第一任宗主）揭示。

### 线6：照影台 ↔ 许衡家族（心镜→母亲→封印）

| 时间 | 事件 |
|------|------|
| ~20年前 | 许衡妻子（照影台弟子）心镜大成→意识被吞噬→许衡偷碎片打碎心镜→成功→但碎片残留→怀胎时胎儿被改写→未择之气诞生 |
| 生完后 | 母亲境界崩塌→体内碎片被儿子未择之气吸引→她必须远离儿子→但她没走→她在石门前按了十六年——不是推门，是用身体挡门对儿子的吸力 |
| 现在 | 母亲还在门后。手还在按着 |

**揭示方式**：第24章纪无咎讲述（外部视角，强调"许衡的罪"）。卷五许衡隔门对话（内部视角，强调"不是推门是挡门"）。卷七——如果母亲能回来，或者永远回不来——那是最后的收束。

## 五、写作原则

1. **每次只漏一点**。一个人物不会把完整的恩怨史讲给许若存听——他们只说自己关心的那一截。
2. **不同人讲同一件事，版本不同**。纪无咎讲的许衡和老陈讲的许衡不是同一个人。读者自己拼。
3. **不替读者总结**。恩怨的"真相"不写成设定百科——让读者在不同人物的叙述中自己判断谁在说谎、谁在隐瞒、谁在自欺。
4. **新恩怨在旧恩怨上叠加**。各势力之间的旧账不是"背景板"——它们会在现在的冲突中重新激活。涤心院想在山庄建洗心堂被拒→卷二中段两人打起来的时候，谢渊说的那句"残缺不是脏"就有了历史重量。
