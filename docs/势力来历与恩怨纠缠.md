# 《岁蚀》势力来历与恩怨纠缠

> 各势力起源、关键时间线、主要恩怨关系。

---

## 一、势力时间线

### 八百年前：天道第一次碎裂

萧轨与秦鹿铸造了第一套九枚铜钱，关上了第一扇门。他们是第一批守门人。黑石城是他们最后居住的地方。萧轨遗言："门能关，但关不住记忆。记忆会自己找路回来。"（该信息在卷一仅通过铜钱暗示，卷二揭示）

### 二十三年前：许衡发现门

许衡第一次找到那扇门的位置。同一时期，钟离燕的师父从门缝里出来，三天后疯了投井。钟离燕因师父的死而走上追查"缝"的路。

### 十六年前：许衡掰断门环

许衡在许若存出生当天掰断门环，母亲将自己推进门里挡住裂缝。许衡把本源撕下来塞进婴儿许若存体内。同一天，他找到何伯安、纪无咎、老陈，分别托付了铜钱、图纸和棋谱。

**同一年冬天**：许衡在城西土地庙选了两个女人走入镜井——柳青辞（许若存母亲，被选者）和陈静姝（沈见微母亲，误入者）。

## 二、核心恩怨线

### 许衡—纪无咎

曾是同门。许衡选择"门"，纪无咎选择"秩序"。分别前纪无咎伸左手，许衡斫下小指——"你选了承序府，我和你的线断了。"

十六年间纪无咎追许衡的踪迹，不是要抓他，是想看他的路能不能走通。第19章他说"你爹走的时候我没送他——我怕我送了他，就忍不住跟他一起走了。"

第35章纪无咎断令牌，用行动表明立场。第61章揭示小指的秘密。

### 柳青辞—陈静姝

两人在同一日、同一地点（城西土地庙）被许衡带入镜井。一人被选（柳青辞），一人误入（陈静姝）。陈静姝被洗心后送入涤心院，柳青辞推门十六年。

两人的儿子（许若存和沈见微）在命运的安排下走到一起，各自继承了母亲留下的碎片——许若存有左肩封印，沈见微有掌纹分叉。

第60章两人的名字在底册上同时浮现：陈静姝、柳青辞。

### 沈见微—裴先生

裴先生是沈见微在涤心院的导师。他亲手洗了沈见微三十次，知道洗心有问题，但他选择"洗一次"代替"深层净化"。第47章他把"净尽"给了沈见微，第78章他的暗格被沈见微打开——他留下了裴字铜钱和纸条。

裴先生不是好人也不是坏人——他是"体制内改良派"的典型。

### 钟离燕—纪无咎

承序府内部的对立。钟离燕代表"规则至上"，纪无咎代表"人情和秩序需要平衡"。钟离燕的师父从门缝里出来疯了，投井——这让钟离燕认为"门"必须被封死。

第35章她公开纪无咎小指的秘密，第53章她放了纪无咎。不是因为她心软——是需要他去处理城西的事。

## 三、势力之间的核心矛盾

| 矛盾对 | 冲突内容 | 代表的价值观对抗 |
|--------|---------|----------------|
| 昭朝 vs 承序府 | 凡人皇权 vs 修行者管理 | 谁有权力定义"人" |
| 承序府 vs 涤心院 | 档案管控 vs 记忆清洗 | 控制人的行为 vs 控制人的记忆 |
| 涤心院 vs 不周山庄 | 秩序固化 vs 解构秩序 | 人应该被"修"还是被允许"碎" |
| 幻音楼 vs 涤心院 | 记录 vs 清洗 | 记忆应该被保存还是被消除 |
| 万流坊 vs 昭朝 | 灰色交易 vs 官方管制 | 信息应该自由流动还是被监管 |
| 照影台 vs 涤心院 | 情绪映照 vs 情绪清洗 | 真相让人承受不住怎么办 |

---

## 四、各势力对"门"的立场

| 势力 | 对门的态度 | 原因 |
|------|-----------|------|
| **昭朝** | 不知道门的存在 | 在修行者的博弈中被隔离在信息外 |
| **承序府** | 想控制门 | 纪无咎想守住，钟离燕想封死 |
| **涤心院** | 想洗掉门 | 门是"不应该被记得的记忆" |
| **万流坊** | 想利用门 | 门的碎片/信息/记忆都是可交易的 |
| **不周山庄** | 接受门 | "缺口才是入口" |
| **幻音楼** | 记录门 | 不干预 |
| **照影台** | 映照门 | 让看见的人自己决定 |
