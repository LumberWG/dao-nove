# 《岁蚀》人物出场记录

> 核心人物总览 + 按七卷结构标注出场。

---

## 核心人物总览

| 人物 | 身份 | 首次出场 | 核心功能 |
|------|------|---------|---------|
| **许若存** | 主角，未择之身，玉佩持有者 | 卷一·第1章 | 为他人选择未来→直面天道→回归日常 |
| **许老头** | 旧货摊摊主，许若存养祖父 | 卷一·第1章 | 暗中守护十六年，关键时坦白真相 |
| **柳三娘** | 官道"三娘客店"老板娘，煮面人，被问"许"时答"住下了" | 卷一·第10章 | 一碗面·荷包蛋——"瘦的人跑不远"，不问来路不问归期 |
| **秦望舒** | 洛京郊外纸坊老板娘，手工造纸、修复旧卷，全书暗线。**深层身份**：从一开始就知道隙径、岁蚀和许若存要去做什么——但从未说破 | 卷二·第24章 | 全书最长暗线——从修旧纸到烧纸封岁蚀到遗忘。不是"等他不追问"，是"知道一切但等他自己来懂" |
| **沈见微** | 涤心院洗心者，镇魔司头号人物 | 卷一·第12章 | 从追捕者到倒戈者→站上石门牺牲 |
| **红姑** | 说书人，万流坊信息节点 | 卷一·第4章 | 以说书暗藏线索，传递老陈故事 |
| **叶知秋** | 承序府仙吏司档案科 | 卷一·第15章 | 规则内的帮助者，用档案权限拖延和掩护 |
| **小六子** | 旧货市场跑腿少年 | 卷一·第8章 | 市井情感锚点，从"若存哥"叫到"许哥" |
| **阿芷** | 无名小城小女孩 | 卷一·第3章 | 讨铜钱→终章还铜钱，前后呼应 |
| **苏缠** | 红绳感应的孤身修行者 | 卷一·第50章 | 灵魂纠缠→主动解开红绳→废界开布料铺 |
| **老陈** | 归寂院年迈修士 | 卷一·第17章 | 预言式指引，说出"戴玉的人分三种" |
| **纪无咎** | 承序府收缴司司长 | 卷一·第40章 | 中期主要对手，称许衡为"叛徒" |
| **许衡** | 许若存之父，守门人 | 卷一·第46章（对话） | 制造门→掰断门环→与门共生 |
| **母亲** | 石门后推了十六年的人 | 卷一·第43章 | 沉默的驱动力，手印与推门 |

## 按卷出场

### 卷一·旧雨不来（许若存 16 岁）

日常→逃亡→发现母亲。无名小城至落霞镇，逐步接近洛京。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 第1章首秀 | 旧货摊帮吵架，获玉佩铜钱，嘴贫记仇性格确立 |
| **许老头** | 第1章起贯穿 | 摆摊养大，地窖暗道，坦白非亲祖父 |
| **小六子** | 第8章起 | 跑腿报信，"穿灰袍的在打听许老头" |
| **柳三娘** | 第10章首秀 | 一碗面·荷包蛋——"瘦的人跑不远"，提供安全空间 |
| **老陈** | 第17章首秀 | 落霞镇归寂院自我对弈，预言戴玉者三种 |
| **沈见微** | 第12章首秀 | 落霞镇包子摊截住许若存，抓捕瞬间手抖放人 |
| **叶知秋** | 第15章首秀 | 黄土路茶亭传话——"名单上快有你的名字了" |
| **红姑** | 第4章首秀 | 旧货市场巷子夜访许若存，花生空壳暗语，说书场开讲老陈故事 |
| **纪无咎** | 第40章首秀 | 断指出场，称许衡"叛徒" |
| **母亲** | 第43章 | 地下湖石板下的手印——按了十六年 |
| **许衡** | 第46章 | 隔石门对话——"你来告诉我我当初的选择是错的" |
| **苏缠** | 第50章首秀 | 红绳感应到许若存体内两道频率 |
| **柳无眠** | 第58章首秀 | 第一次燃命挡虚甲兽 |
| **霍斩** | 第70章首秀 | 青石集废界武修，打半场架成朋友 |
| **季真** | 第73章首秀 | 女剑修实战教学——收剑和出剑一样重要 |

### 卷二·序隙之裂（许若存 17 岁）

承序府追捕→隙径初现→各方入场→沈见微与柳无眠牺牲。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 贯穿 | 入洛京，走隙径，接触各大势力 |
| **叶知秋** | 卷初 | 用内部权限放许若存入城，欠人情 |
| **钟离燕** | 卷初首秀 | 承序府档案员，读灵力残留，发现许衡档案被删改 |
| **谢渊** | 卷中首秀 | 不周山庄代表，聊天式切入——"碎裂的天道需要人去问" |
| **花姐** | 卷中首秀 | 万流坊接头人，青石集茶馆老板娘 |
| **灵犀** | 卷一·第8章首秀 | 隙径向导，许若存行走隙径的**专属搭档**（寻路/穿越/战斗/逃生四大职能）。花姐线人。无八岁前记忆。从卷一到卷六贯穿出场约12-15次，每次出场推动隙径规则新发现或灵犀身世线索。卷二起个人线逐步展开——隙径崩塌、镜子反转、与天道代理人连接 |
| **老鼹鼠** | 卷中首秀 | 万流坊信息贩子，专营隙径路线情报 |
| **秦望舒** | 第42章（纸坊线首次） | 纸坊第一次推门——"这种纸是西真陆的构树皮做的" |
| **白无瑕** | 卷末首秀 | 照影台圣女，心镜碎裂下山，包子不知要付钱 |
| **沈见微** | 卷末收束 | 推开许若存站上石门，身体化为云纹浮雕——牺牲 |
| **柳无眠** | 卷末收束 | 最后一次燃命，花生藤根系接入地下水脉——牺牲 |

### 卷三·万流暗涌（许若存 18 岁）

万流坊争夺→隙径旅行→碎片线索→秦望舒暗线浮出。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 贯穿 | 深入万流坊，集齐三枚铜钱，定位门的位置 |
| **许老头** | 卷初 | 万流坊三级节点身份浮出水面 |
| **老鸹** | 卷初首秀 | 万流坊实际操盘手，传递许衡失联前线索 |
| **秦望舒** | 第24/31/38/44章 | 四次"路过"：姜糖水→洗碗→巷口看晒纸→冬天蓝棉布 |
| **谢渊** | 卷末 | 正式邀许若存访不周山庄，完整是假象漏洞才是入口 |
| **道痕猎手** | 卷中首秀 | 天道自愈机制人格化，隙径中追击许若存 |
| **铜钱老人** | 卷中回归 | 三枚铜钱聚齐可定位石门确切位置 |

### 卷四·记忆吞噬（许若存 19 岁）

岁蚀降临→记忆战争→身边人各自离开→许若存推门进入域外。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 贯穿 | 左肩封印裂缝扩大，决定推门进入域外 |
| **许老头** | 卷末 | 最后一次煮粥——水放多了，一个人喝完一锅 |
| **秦望舒** | 第38章 | 岁蚀之夜，构树皮褪成灰色，她继续做纸 |
| **九公主** | 卷中首秀 | 昭朝内部视角，与许若存联手交换情报 |
| **霍斩** | 卷中 | 回到废界，刀挂村口老榆树 |
| **季真** | 卷中 | 离开队伍，去无名坟前拔草 |
| **苏缠** | 卷中 | 主动解开红绳，叠好放木盒，去废界开布料铺 |
| **沈见微** | 卷中（记忆） | 许若存看到他留在门上的记忆碎片——"今年有人记得我吗" |
| **母亲** | 卷末 | 门缝里漏出域外气息，手还在推 |

### 卷五·虚空不空（许若存 20 岁）

进入域外→寻找许衡→遗忘之主真相→许若存前身。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 贯穿 | 沿隙径纹路方向走，玉佩纹路是域外地图 |
| **窃语者** | 卷初首秀 | 低阶域外生物，模仿母亲声音 |
| **许衡** | 卷中重逢 | 右手与门板融为一体，说出门的真相和"未择之身"来源 |
| **遗忘之主** | 卷中首秀 | 问宗第一任宗主，被天道遗忘变为遗忘本身——被问倒后消失 |
| **秦望舒** | 穿插 | 纸坊门被推开——不是他。她把蓝布抚平："冬天凉。" |

### 卷六·前世来生（许若存 21-23 岁）

面对天道→终极选择→前世许若存→不修复也不毁灭。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 贯穿 | 面对天道代理人，做出最终选择——带走天道忘记的记忆 |
| **天道代理人** | 卷初首秀 | 被困在答案位置上太久——"问我一个答不上的问题" |
| **前世许若存** | 卷中首秀 | "未择之身"是被重置的结果，他前世选择了重新开始 |
| **秦望舒** | 卷末穿插 | 构树皮换了一季又一季，竹帘被手指磨亮，蓝布还在窗台 |

---

### 卷七·后来不来（许若存约 25 岁）

回无名小城→还九文碗钱→所有人归位→秦望舒纸坊收束，柳三娘最后一碗面。

| 人物 | 章节段 | 本章功能 |
|------|-------|---------|
| **许若存** | 贯穿 | 买小屋，每日喝豆浆，不再选——不再需要选 |
| **许老头** | 卷中 | 住在洛京附近小屋，每天早上煮粥，水放多了 |
| **小六子** | 卷中回归 | 长大了，不再叫"若存哥"，叫"许哥" |
| **阿芷** | 卷中回归 | 还当年讨回的铜钱——"我没欠你的了" |
| **秦望舒** | 第42-53章 | 锁门→货郎担里的银杏叶纸→掌印→"收了好"——没有结局的结局 |
| **柳三娘** | 第71-80章 | 最后一碗面，荷包蛋还在——"不来就不来，面给你留着" |
| **苏缠** | 卷末 | 废界布料铺，布上纹路"没什么意思，好看" |
| **白无瑕** | 卷末 | 洛京南门外镜子铺，背面刻"你看到的是真实的你" |
| **霍斩** | 卷末 | 废界帮村里修屋顶，刀挂村口老榆树 |
