# 《岁蚀》全书新纲要（弧线结构版）

> **总规模**：约9个故事弧 × 每弧12-20章 × 每章约5000字 = 约600章 ≈ 300万字
> **核心转变**：从"一卷一条线"改为"每10-15章一个完整的弧"，每个弧有自己的冲突、高潮和结局

---

## 写作原则（基于第1-19章经验）

详见 `docs/笔法核心规则.md` 末尾"叙事结构铁律"章节。

核心四句话：
- 人物不定性、不定型——读者前三页猜不出角色的立场
- 大纲是活的不死的——发现更好的方向就改大纲
- 信息像撒盐——切碎隔章释放，不一次性倒完
- 每弧独立可读——哪怕不看前后，单读一个弧也是完整故事

## 弧1·旧雨（第1-19章）已发表

**篇幅**：19章 / **状态**：已发表，不可修改
**主题**：谎言的重量——十六年的日常是假的，但那些感情是真的。假的真相和真的感情哪个更重？

一个少年在旧货摊发现玉佩和铜钱后，发现自己十六年的生活是谎言。被迫逃离无名小城，在落霞镇到青石集的路上逐步接触各大势力。

| 核心冲突 | 弧内悬念 | 弧末高潮 | 弧末钩子 |
|---------|---------|---------|---------|
| 生活建立在谎言之上 | 玉佩和铜钱是做什么的？ | 纪无咎茶馆对峙，"门后面不是你想的那样" | 铜钱指向洛京 |

## 弧2·秩序之门（第20-32章）待重组

**篇幅**：13章 / **状态**：已有草稿，需按弧重组
**主题**：规则的囚笼——每一股势力都说"为了你好"，但许若存每走一步都觉得这座城在收紧。

许若存进入洛京后，发现这座"秩序之城"的每一股势力都想利用他。门在序塔底下——但靠近门需要先穿过承序府、万流坊和涤心院的三层网。

**本弧的核心问题**：门到底在哪儿？谁在阻止他靠近？

**弧内对手**：承序府的体制（表面秩序）、纪无咎的追捕（表面对手）、涤心院的暗手（隐藏对手）

**类型基调**：体制惊悚
**气氛设计**：城门入城时的盘查像一张嘴在合拢；序塔的每一层檐角下都挂着一面镜子，像无数只眼睛；穿灰袍的人走在街上，所有人低头。读者应该感到"喘不过气"——不是因为追兵，是因为规则。

**章节分配**：
| 章 | 内容 | 功能 |
|----|------|------|
| 20 | 沈见微通牒·洛京 | 双线：沈见微的十日倒计时 + 许若存接近洛京 |
| 21 | 界碑·问宗残迹 | 气氛章：问宗的过去暗示"门"不是第一扇 |
| 22-24 | 进塔·门首次出现 | **弧内高潮**：只看到门缝里透出一线光，一只手伸出来又缩回去。不许进 |
| 25-26 | 万流坊·灰市 | 缩减为两章，信息密度加倍 |
| 27-28 | 隙径·旧庙 | 灵犀再出场，规则展示 |
| 29-30 | 苏缠·封印纹理 | 首次揭示封印的双频本质 |
| 31-32 | 岁蚀前兆+因果铺 | **弧末高潮**：签因果契，背负父亲债务 |

**弧末变化**：因果契签订，许若存正式背负父亲的债务。
**弧末钩子**：三件事——第一件事，从序塔底层取回因果眼。

## 弧3·湖底（第33-45章）待重组

**篇幅**：13章 / **状态**：已有草稿，需按弧重组
**主题**：债与偿——父亲的债凭什么要儿子还？但如果不还，母亲十六年的撑就白费了。

许若存必须在岁蚀吞噬心湖前完成父亲留下的因果债。心湖下的门是唯一的入口——但门后有虚甲兽守着，门后还有母亲在撑着。

**本弧的核心问题**：父亲许衡到底是叛徒还是牺牲者？

**弧内对手**：心湖水底的虚甲兽（表面）、纪无咎（立场转变中）、门后的未知存在

**类型基调**：黑暗奇幻
**气氛设计**：心湖的水从透明变成黑色，湖面不反光，像一面被涂黑的镜子。母亲的手印从温热变凉的那一夜，湖边的风停了。虚甲兽的甲壳在黑暗中摩擦的声音像指甲刮黑板。

**章节分配**：
| 章 | 内容 | 功能 |
|----|------|------|
| 33-34 | 柳无眠+许老头坦白（后移） | 坦白放在此刻比第7章有力——读者已经知道门，现在知道代价 |
| 35-37 | 心湖之战 | **弧内高潮1**：虚甲兽击退，母亲手印点亮，门裂一道缝 |
| 38-39 | 因果铺 | 三选一展示但不选——时机未到 |
| 40-42 | 入幕后·母亲消散 | **弧内高潮2**：门后见到母亲消散，墙上刻满十六年的记录 |
| 43-45 | 季真/霍斩登场 | 建立人物关系后迅速回归主线 |

**弧末变化**：母亲消散，许若存知道父亲的真相不完整。
**弧末钩子**：老陈说"你认识的那个人手里有一份情报"。

## 弧4·星散（第46-58章）待重组

**篇幅**：13章 / **状态**：已有草稿，需按弧重组
**主题**：告别——每个人都用自己的方式说"我不能陪你走到底了"。

岁蚀降临，每个人都有自己必须面对的过去。沈见微记忆恢复、苏缠脱离幻音楼、四枚铜钱聚齐——但聚齐意味着有人要代替许若存走进门里。

**本弧的核心问题**：这些人为什么愿意为许若存牺牲？

**弧内对手**：涤心院（沈见微的对抗对象）、幻音楼（苏缠的对抗对象）

**类型基调**：群像悲情
**气氛设计**：每个人都走在自己的告别路上。苏缠解绳时红绳断了的那一声，沈见微倒戈前在巷子里站了多久，阿芷走入门里之前回头看了许若存一眼。每一场告别的气氛都不一样。

**章节分配**：
| 章 | 内容 | 功能 |
|----|------|------|
| 46-47 | 方主簿信+心湖告别 | 一封信揭示许衡支线布局 |
| 48-50 | 柳三娘碗+苏缠解绳 | **弧内高潮2**：苏缠主动解绳——不是魔法消失，是选择 |
| 51-54 | 沈见微倒戈 | **弧内高潮1**：恢复记忆→挣扎→叛逃→牺牲伏笔（4章密集叙事） |
| 55-57 | 阿芷铜钱四合一 | 精简霍斩/季真支线，四铜钱聚齐 |
| 58 | 许老头煮粥告别 | 情感锚点——孤身一人的许若存 |

**弧末变化**：四枚铜钱聚齐，团队成员各奔东西。
**弧末钩子**：门在序塔底层等着——但岁蚀已经进城。

**主题**：选择的代价——推门的结果是未知的，但不推的结果是已知的。

## 弧6·裂隙（第71-85章）新写

**篇幅**：15章 / **状态**：新写
**主题**：重逢不等于找回——你找了一辈子的人，找到时他已经不是你想的那个了。

域外的物理法则与人间截然不同——没有光，只有温度感知，温度越高越危险。许若存必须在迷失自我前找到父亲。

**弧内对手**：窃语者（低阶域外生物，模仿母亲声音引诱他）
**弧内高潮**：许衡重逢——右手与门板融为一体，十六年后的父子对话
**弧末钩子**：遗忘之主在域外更深处等着

**类型基调**：孤寂求生
**气氛设计**：域外没有光、没有声音、没有方向。温度感知是唯一的感官——温度越高越危险。窃语者模仿母亲的声音从黑暗中传来，每一声都比上一声更像。

## 弧7·问宗（第86-100章）新写

**篇幅**：15章 / **状态**：新写
**主题**：被遗忘比死更可怕——一个被世界忘记的宗门，一个连自己名字都不记得的宗主。

问宗的覆灭真相、万流坊的起源、铜钱背后的秘密。遗忘之主——问宗第一任宗主，被天道遗忘变成了遗忘本身。

**弧内对手**：遗忘之主
**弧内高潮**："你记得你自己吗？"——遗忘之主被问倒后选择消失
**弧末钩子**：天道代理人——最后一道门

**类型基调**：哲学恐怖
**气氛设计**：遗忘之主没有固定形态——它是空的轮廓，你盯着它看越久，自己的轮廓越模糊。它说"我连自己的名字都忘了，但我记得天道欠我一个回答"。

## 弧8·前世（第101-115章）新写

**篇幅**：15章 / **状态**：新写
**主题**：如果过去的自己是个怪物——找回记忆等于变成你憎恨的人，你还要找吗？

许若存发现未择之身不是父亲给的——是前世的自己留下的。前世的自己选择了被重置，变成了许老头养大的那个16岁少年。

**弧内对手**：天道代理人
**弧内高潮**：与前世自己的对话——"你不是来修复天道的。你是来替我回答的。"
**弧末变化**：不修复天道也不毁灭天道——带走天道忘记的记忆

**类型基调**：身份惊悚
**气氛设计**：另一个自己坐在对面，穿着一样的衣服、用一样的语气说话。他说"你每一脚踩下去都是我自己踩过的坑"——这不是比喻，是他的记忆在覆盖许若存的记忆。

## 弧9·归途（第116-135章）新写

**篇幅**：20章 / **状态**：新写
**主题**：走了那么远，想要的原来在身边——一碗面、一张纸、一锅粥，不是妥协，是看懂了。

回到人间，所有人都在等他——但不是所有人都还记得他。

**弧内对手**：无（收束篇）
**弧内高潮1**：秦望舒烧纸——"你是谁？"
**弧内高潮2**：柳三娘最后一碗面——"不来就不来，面给你留着"
**弧末**：许若存在无名小城买小屋，不再选。晨光照在木盒盖上，他为自己煮了一锅粥。水放得正好。

**类型基调**：安静的告别
**气氛设计**：全弧没有一场战斗、没有一个反派。只有一碗面、一张纸、一枚还回来的铜钱、一间没有锁的纸坊。最重的那场戏是秦望舒烧纸——她烧掉的不是纸，是一个人七年的记忆。火灭了之后她问"你是谁"，语气和问"今天天气不错"一样平静。
