# 《岁蚀》全书新纲要（300万字·40弧版）

> **总规模**：7卷 × 每卷约85章 = 约600章 × 每章5000字 = 约300万字
> **每卷结构**：4-6个故事弧，每弧15-20章，独立起承转合
> **核心原则**：每层真相只揭一半，不让读者猜到结局

## 第一卷·旧雨不来（约85章）

**卷主题**：从逃到选——许若存从被动逃命到主动选择的转变
**许若存年龄**：16岁

### 弧1·旧雨（第1-19章）
**篇幅**：19章 / **状态**：已发表

**弧主题**：谎言的重量——十六年的日常是假的，但那些感情是真的。假的真相和真的感情哪个更重？

**弧中心悬念**：玉佩和铜钱是什么？为什么所有人都在盯着许若存？

| 弧内对手 | 弧末高潮 | 下弧钩子 |
|---------|---------|---------|
| 多方势力（未知） | 纪无咎茶馆对峙，"门后面不是你想的那样" | 铜钱指向洛京 |

### 弧2·追猎（第20-35章）待重构
**篇幅**：约16章

**弧主题**：逃亡中的人性——追他的人各怀目的，帮他的人也各怀目的。没有人纯粹是"好人"或"坏人"。

**弧中心悬念**：五拨追兵到底谁是谁的人？那个蓝衣人给纸条是帮他还是钓他？

**关键节点**：
| 节点 | 内容 |
|------|------|
| 第20-21章 | 沈见微十日通牒+界碑汇合 |
| 第22-24章 | 入洛京·序塔·第一次见门（只给光，不让进） |
| 第25-28章 | 万流坊·灰市·隙径规则展示 |
| 第29-31章 | 灵犀再出场·苏缠初遇·封印双频揭示 |
| 第32-35章 | 岁蚀前兆·因果铺签约·纪无咎追至 |

**弧末变化**：许若存签了因果契，背负父亲的债务。

**下弧钩子**：契约入肉那一刻，左肩封印裂了一道。苏缠说"你体内有两道频率"——但有一道她看不清楚。

---

### 弧3·湖下（第36-52章）待重构
**篇幅**：约17章

**弧主题**：债与偿——父亲的债凭什么要儿子还？但如果不还，母亲十六年的撑就白费了。

**弧中心悬念**：母亲在门后推了十六年——她是在推门进来，还是在挡门不让东西出去？

**关键节点**：
| 节点 | 内容 |
|------|------|
| 第36-38章 | 心湖之战·虚甲兽·门裂开了缝（伸出的手不是母亲的） |
| 第39-40章 | 因果铺三件事·第一件事取因果眼 |
| 第41-43章 | 入幕后见母亲·她推许若存肩膀不让进·消散 |
| 第44-46章 | 季真·霍斩登场·纪无咎坦白 |
| 第47-49章 | 沈见微倒戈·苏缠解绳·各自抉择 |
| 第50-52章 | 四铜钱聚齐·许老头告别·巷口人影 |

**弧末变化**：母亲消散。四枚铜钱聚齐。团队各自告别。

**下弧钩子**：母亲消散前说"别让他们进来"——不是"别进来"，是"别让**他们**进来"。门后还有别的东西。

### 弧4·围城（第53-68章）待编写
**篇幅**：约16章

**弧主题**：选择的代价——你选什么，就要承担什么。不选，也是选择。

**弧中心悬念**：岁蚀来了，门在自动打开——许若存根本没得选？

**弧末变化**：柳无眠燃尽，沈见微化为云纹，门开了。

**下弧钩子**：他走进去，门后不是虚空——是一个有光的地方。

### 弧5·门后（第69-85章）待编写
**篇幅**：约17章

**弧主题**：你以为你知道了真相——但真相不止一层。

**弧中心悬念**：门后是什么地方？许衡在哪里？

**弧末变化**：卷一结束。许若存在门后看到了"天道忘记的那段记忆"的碎片。

**下弧钩子**：许衡的笔迹出现在墙上——但不是他一个人写的。还有一个人。

## 防剧透反转结构（贯穿全书）

| 第7章让读者以为 | 实际反转 | 揭示弧 |
|----------------|---------|--------|
| 母亲在推门进来 | 她在挡门不让东西出去 | 弧3 |
| 父亲撕本源是为了给儿子能力 | 碎片里混着前世许若存留下的东西，父亲不知道 | 弧5 |
| 未择之身是天赋 | 是债务——前世的自己留下的欠条 | 弧6 |
| 门后是域外 | 门后是"天道忘记的那段记忆" | 弧4 |
| 纪无咎是反派 | 他是唯一一个知道"门不能开"的人 | 弧3 |
| 沈见微牺牲是因为正义 | 他牺牲是因为终于能自己选一次了 | 弧4 |
| 灵犀是偶然遇到的向导 | 她的失忆和许若存的前世有关 | 弧6 |
