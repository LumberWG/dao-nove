# 《岁蚀》全书新纲要（大纲版）

> 总规模：7卷 × ~85章 = 约600章 × 约5000字 = 约300万字
> 每卷4-6弧，每弧15-20章
> 核心目标：每章预留扩展空间，时空跨度逐层拉大

---

## 空间扩展层级（全书由小到大的尺度）

小说开篇从一个小城的旧货摊开始，终点是天道记忆的虚空。中间必须逐层拉大空间感，不能突然跳跃。

| 层级 | 尺度 | 典型场景 | 对应章节段 |
|------|------|---------|-----------|
| **L1 街巷级** | 百米 | 旧货市场、老槐树巷、心湖 | 弧1前期 |
| **L2 城际级** | 百里 | 无名小城→落霞镇→青石集→洛京 | 弧1-弧2 |
| **L3 域内级** | 千里 | 洛京→天阙京→涤心院→不周山庄 | 弧2-弧3，全部卷二 |
| **L4 隙径级** | 任意 | 隙径穿越将千里压缩为一刻钟 | 贯穿，从弧1末开始 |
| **L5 跨界级** | 世界间 | 门内→域外→遗忘回廊→道网 | 弧4开始，卷五展开 |
| **L6 天道级** | 概念边界 | 面对天道代理人，面对前世自己 | 卷六 |

**关键约束**：不能从L2直接跳到L5。每一级必须在前一级中被**提及或暗示**至少一次，才能在后文正式出现。

---

## 时空跨度的三根锚桩

要支撑全书300万字，每章必须至少锚定以下三根桩中的一根：

### 桩一：空间锚
不能只靠走路。每个弧引入一种新的移动/感知方式：

| 弧 | 引入的移动/感知方式 | 为什么需要它 |
|----|-------------------|-------------|
| 弧1 | 步行+骑马 | 建立"世界很大"的体感 |
| **弧2** | **隙径初体验** | 第一次压缩空间，暗示"世界不止表面这么大" |
| 弧3 | 隙径频繁使用+隙径地图 | 建立隙径网络的存在感，为跨界级做铺垫 |
| 弧4 | 隙径中的"镜中人"和时空异常 | 暗示隙径之外还有更深的空间层 |
| 弧5 | 门内初探 | 第一次触碰"世界之外" |
| 卷2 | 隙径网络全面展开 | "地下世界"和"地上世界"并行 |
| 卷3 | 万流坊灰市·时间流速异常 | 空间之外，时间也可以不同 |
| 卷4 | 岁蚀侵蚀空间本身 | 概念被吃掉，空间开始崩塌 |
| 卷5 | 域外行走 | 彻底离开"人间"的地理概念 |
| 卷6 | 天道之网 | 空间不再存在——只有问题和答案 |

### 桩二：时间锚
时间不能均匀流过。需要"快进"和"慢放"制造节奏变化：

| 手法 | 用法 | 示例 |
|------|------|------|
| 隙径时间差 | 隙径内外时间流速不同 | 弧2第一次走隙径：里面一刻钟，外面三天 |
| 灰市时间流 | 灰市时间比外界慢/快 | 弧3灰市：里面三个时辰，外面九天 |
| 岁蚀症状 | 被岁蚀侵蚀的区域时间感错乱 | 弧4岁蚀降临：一条街上的人过了不同天数 |
| 域外无时间 | 域外没有统一时间流逝 | 卷五：许若存不知道自己在里面待了多久 |
| 回忆/闪前 | 通过封印/铜钱看到过去或未来的碎片 | 贯穿全书 |

### 桩三：信息差锚

**读者知道的事情不能和主角一样多，也不能比他少太多。**

| 信息差类型 | 用法 |
|-----------|------|
| **读者>主角**（戏剧反讽） | 读者通过沈见微POV知道涤心院在设局，许若存不知道 |
| **主角>读者**（悬念） | 许若存看到门缝里的东西——但没告诉读者是什么 |
| **同一起跑线**（同步探索） | 许若存和读者第一次看到隙径，体验相同的新奇和恐惧 |

**每章至少使用其中一种信息差**。如果一章里读者和主角知道的事完全一样——这章的信息密度不合格。

## 第一卷·旧雨不来（约85章·5弧）

**卷主题**：从逃到选
**空间跨度**：L1街巷级 → L4隙径级（首次触碰L5）
**时间跨度**：约两个月
**信息差锚**：沈见微POV提供戏剧反讽

### 弧1·旧雨（第1-19章）已发表
**篇幅**：19章 / **空间级**：L1-L2
**弧主题**：谎言的重量
**预留扩展**：铜钱、玉佩、隙径（第8章首次出现）、问宗（第9章界碑）——这四个元素在后续都会展开为完整的子系统

### 弧2·追猎（第20-35章）待重构
**篇幅**：约16章 / **空间级**：L2-L3（首次引入L4隙径）
**弧主题**：逃亡中的人性
**时间跨度**：约10天

**必须完成的任务**：
1. **隙径规则建立**——第一次走隙径，展示"一刻钟从千里外回到原点"的震撼感，建立"世界比步行大得多"的认知
2. **洛京的规模建立**——洛京不能只是一个"有城墙的地方"。它是承序府总部、昭朝权力中心、七大势力暗线的交汇点。弧2离开洛京后，要留下"这座城还有我没去的地方"的感觉
3. **苏缠和封印的双频**——为弧3"第三道频率"埋线索
4. **时间差**——灰市的时间流速异常（三个时辰=九天），让读者意识到时间不是均匀的

**预留扩展空间**：
- 隙径网络：弧2只展示"一条隙径"——暗示"地下有一个完整的网络"但不展开
- 万流坊：弧2只展示"三级节点"——暗示"还有二级、一级、灰市"但不展开
- 问宗：弧2只展示"界碑被磨掉了"——暗示"这个宗门的历史很深"但不展开

### 弧3·湖下（第36-52章）待重构
**篇幅**：约17章 / **空间级**：L2-L4
**弧主题**：债与偿
**时间跨度**：约7天

**必须完成的任务**：
1. **心湖的空间层次**——心湖不止是一个湖。水面是L2，湖底石门是L4（世界边界），门缝里透出的气息是L5（域外）
2. **隙径地图展开**——玉佩的纹路是一张隙径全图，弧3展示"这个世界的地下交通网络有多庞大"
3. **母亲消散的空间感**——她消散时"墙上的字"是许衡十六年刻的，但有些字笔迹不对——暗示有第三个人在门后待过
4. **岁蚀的空间影响**——岁蚀不仅影响"人"，也影响"地方"——被侵蚀的区域路会断、方向会乱、空间会扭曲

**预留扩展空间**：
- 灰市：弧3只展示灰市的"存在"——暗示"每一座城下面都有这样一个地方"
- 隙径镜中人：弧3只展示"遇到一个镜中人"——暗示"隙径里散掉的人不止这一个"

### 弧4·围城（第53-68章）待编写
**篇幅**：约16章 / **空间级**：L2-L5（首次触碰L5）
**弧主题**：选择的代价
**时间跨度**：约5天（岁蚀加速了一切）

**必须完成的任务**：
1. **岁蚀的空间崩塌**——墙消失、路断掉、天上出现裂缝。洛京不再是"安全的城"，变成了"困住所有人的牢笼"
2. **门内的气息**——第64-65章推门时，门缝里透出的不是"黑暗"——是"另一个世界的颜色"。读者第一次看到L5的样子
3. **灵犀的隙径体质**——灵犀在岁蚀中展现出"与隙径重叠"的特殊能力——为卷二的灵犀个人线留空间

### 弧5·门后（第69-85章）待编写
**篇幅**：约17章 / **空间级**：L4-L5
**弧主题**：真相不止一层
**时间跨度**：隙径内时间混乱，无法精确计算

**必须完成的任务**：
1. **门后世界的第一印象**——不是虚空，不是黑暗——是"一片没有天的空地，远处有光"
2. **许衡的踪迹**——墙上刻着许衡的留言，但不是他一个人写的——还有另一个人的字迹
3. **卷末钩子**——许若存看到"天道忘记的那段记忆"的碎片——**记忆里有他自己的脸**

## 全卷时空扩展路径图

```
弧1  ─── 无名小城                       L1-L2
         ↓ （发现铜钱→出走）
弧2  ─── 落霞镇→青石集→洛京             L2-L3
         ↓ （隙径第一次使用）
弧3  ─── 心湖·石门·万流坊灰市           L2-L4
         ↓ （隙径网络展开）
弧4  ─── 洛京围城·岁蚀崩塌·门开         L2-L5
         ↓ （进门）
弧5  ─── 门后·域外边沿·天道记忆碎片     L4-L5
         ↓ 
卷二  ─── 隙径网络全面展开+各方势力       L3-L4
         ↓
卷三  ─── 万流坊地下世界+灰市深处        L3-L4
         ↓
卷四  ─── 岁蚀全面降临+城际崩塌          L2-L5
         ↓
卷五  ─── 域外·遗忘回廊·天道代理人      L5-L6
         ↓
卷六  ─── 天道之网·前世自己             L6
         ↓
卷七  ─── 回到L1——无名小城             L1
```

每一级的空间感在正式出现之前，必须在前面至少**被暗示一次**：
- L3（洛京之外的势力分布）：在弧1结尾纪无咎提到"承序府"时就应该暗示"这个组织遍布整个大陆"
- L4（隙径网络）：在弧1第8章第一次走隙径时就暗示"这只是冰山一角"
- L5（域外）：在弧2苏缠说"第二道频率不属于这个世界"时暗示
- L6（天道）：在弧3纪无咎说"你爹在补天道"时暗示
